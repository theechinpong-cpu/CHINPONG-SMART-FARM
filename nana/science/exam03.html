คลังข้อสอบและเฉลยวิชาวิทยาศาสตร์

ข้อสอบคัดเลือกเข้าศึกษาต่อชั้น ม.1 โรงเรียนรัฐบาล (ชุดที่ 3)
คำสั่ง: จงเลือกคำตอบที่ถูกต้องที่สุดเพียงคำตอบเดียวพร้อมทบทวนคำอธิบายเฉลย
1. เหตุที่หลุมอุกกาบาตบนผิวของดวงจันทร์มีมากมายกว่าบนผิวโลก เพราะเหตุใด
1. ไม่มีบรรยากาศเหมือนโลก จึงไม่มีการเผาไหม้ของอุกกาบาตก่อนตก 2. มีแรงดึงดูดมากกว่าโลก จึงทำให้อุกกาบาตตกมาก
3. มีแรงดึงดูดน้อยกว่าโลก จึงทำให้อุกกาบาตตกได้ดีกว่าผิวโลก 4. อุกกาบาตที่ตกบนโลกมีมากกว่าบนดวงจันทร์ แต่ไปตกในทะเลหมด
เฉลยตอบ ข้อ 1 | เนื่องจากดวงจันทร์ไม่มีชั้นบรรยากาศห่อหุ้ม คอยสร้างแรงเสียดสีและเผาไหม้วัตถุจากอวกาศ อุกกาบาตจึงพุ่งเข้าชนพื้นผิวดวงจันทร์โดยตรงจนเกิดเป็นหลุมบ่อขนาดใหญ่จำนวนมาก
2. ดาวเทียมสื่อสารมีประโยชน์อย่างไร
1. ช่วยพยากรณ์อากาศ 2. ช่วยในการหาทิศ
3. ช่วยสะท้อนสัญญาณวิทยุ 4. ช่วยในการคมนาคมทางอากาศ
เฉลยตอบ ข้อ 3 | ดาวเทียมสื่อสารทำหน้าที่เป็นสถานีรับและทวนสัญญาณ หรือสะท้อนคลื่นวิทยุ/ไมโครเวฟกลับลงมายังสถานีรับภาคพื้นดินเพื่อเชื่อมต่อเครือข่ายโทรคมนาคมไร้สาย
3. การต่ออุปกรณ์ไฟฟ้าภายในบ้าน เป็นการต่อตามภาพในข้อใด
[ แผนภาพจำลองวงจรไฟฟ้า ]
ข้อ 1. วงจรต่อแบบอนุกรม (เรียงต่อกันหลอดต่อหลอด)
ข้อ 2. วงจรต่อแบบขนาน (ต่อคร่อมขนานขั้วร่วมกันอย่างอิสระ)
ข้อ 3 และ ข้อ 4. วงจรผสมรูปแบบต่างๆ
1. รูปที่ 1 (แบบอนุกรม) 2. รูปที่ 2 (แบบขนาน)
3. รูปที่ 3 4. รูปที่ 4
เฉลยตอบ ข้อ 2 | การต่อเครื่องใช้ไฟฟ้าในบ้านต้องต่อแบบขนาน เพื่อให้ทุกอุปกรณ์ได้รับแรงดันไฟฟ้าเท่ากัน (220V) และเมื่อชิ้นใดชิ้นหนึ่งชำรุดเสียหาย ชิ้นอื่นยังคงทำงานได้ตามปกติ
4. ในการผสมคอนกรีต เราใช้ทราย ปูนซีเมนต์ และน้ำผสมกัน ลักษณะการเกิดคอนกรีตจึงเปรียบได้กับการเกิดหินอะไรในธรรมชาติ
1. หินอัคนี 2. หินแปร
3. หินตะกอน 4. หินอ่อน
เฉลยตอบ ข้อ 3 | คอนกรีตเกิดจากการรวมตัวของวัสดุเม็ดหยาบ (กรวด ทราย) ประสานเข้าด้วยกันด้วยปูนซีเมนต์และน้ำ ซึ่งเลียนแบบกระบวนการเกิดหินตะกอน (หินชั้น) ที่มีสารละลายตามธรรมชาติมาประสานตะกอนให้จับตัวแน่น
5. หินที่มักพบซากดึกดำบรรพ์ (ฟอสซิล) คือหินอะไร
1. หินบะซอลต์ 2. หินออบซิเดียน
3. หินดินดาน 4. หินศิลาแลง
เฉลยตอบ ข้อ 3 | ซากดึกดำบรรพ์มักสะสมและถูกอนุรักษ์ไว้ได้ดีในหินตะกอนเนื้อละเอียดที่มีกระบวนการทับถมไม่รุนแรง เช่น หินดินดาน หรือหินปูน ส่วนหินบะซอลต์และหินออบซิเดียนเป็นหินอัคนีที่เกิดจากลาวาร้อนจัดซึ่งจะทำลายซากสิ่งมีชีวิตหมดสิ้น
6. แร่ "ลิกไนต์" ใช้ทำประโยชน์ด้านใดในประเทศไทย
1. ใช้ผสมทำซีเมนต์ 2. ใช้ทำไส้หลอดไฟฟ้า
3. ใช้ผสมโลหะให้แข็งขึ้น 4. ใช้เป็นเชื้อเพลิงในโรงไฟฟ้า
เฉลยตอบ ข้อ 4 | ถ่านหินลิกไนต์เป็นถ่านหินคุณภาพต่ำถึงปานกลาง ประโยชน์หลักในประเทศไทยคือการใช้เป็นเชื้อเพลิงในการต้มน้ำเพื่อหมุนกังหันผลิตกระแสไฟฟ้า เช่นที่โรงไฟฟ้าแม่เมาะ จังหวัดลำปาง
7. ดินขาว ใช้ประโยชน์ในด้านใด
1. ทำปูนขาว 2. ทำเครื่องแก้ว
3. ใช้ผสมทำสี 4. ปั้นถ้วยชาม
เฉลยตอบ ข้อ 4 | ดินขาว (Kaolin) เป็นวัตถุดิบสำคัญและเป็นหลักในอุตสาหกรรมเครื่องเซรามิก (Ceramics) ผลิตภัณฑ์เครื่องปั้นดินเผา และถ้วยชามกระเบื้องเคลือบต่างๆ
8. เครื่องมือที่ใช้ดูวัตถุไกลๆ ให้เห็นชัดเจนขึ้น คือข้อใด
1. แว่นขยาย 2. กล้องจุลทรรศน์
3. กล้องโทรทรรศน์ 4. กล้องสลับลาย
เฉลยตอบ ข้อ 3 | กล้องโทรทรรศน์ (Telescope) ขยายภาพวัตถุที่อยู่ไกลมากๆ ในอวกาศ เช่น ดวงดาว ส่วนกล้องจุลทรรศน์ใช้ส่องดูวัตถุขนาดเล็กจิ๋วที่อยู่ใกล้ตัว
9. ปรากฏการณ์ธรรมชาติข้อใดที่เกิดขึ้นเนื่องจากการหักเหของแสงในบรรยากาศ
1. รุ้งกินน้ำ 2. สุริยุปราคา
3. จันทรุปราคา 4. ข้างขึ้น-ข้างแรม
เฉลยตอบ ข้อ 1 | รุ้งกินน้ำเกิดจากแสงอาทิตย์เคลื่อนที่ผ่านละอองน้ำในอากาศ แล้วเกิดกระบวนการหักเห แยกแสงออกเป็นสเปกตรัม 7 สี พร้อมสะท้อนกลับหมดภายในหยดน้ำแล้วหักเหออกมาเข้าสู่ตาเรา
10. กฎการสะท้อนของแสงมีใจความว่าดังนี้
    ก. มุมตกกระทบ เท่ากับมุมสะท้อน
    ข. รังสีตกกระทบ รังสีสะท้อน และเส้นแนวฉากอยู่บนระนาบเดียวกัน
ข้อใดถูกต้อง
1. ข้อ ก 2. ข้อ ข
3. ถูกทั้งข้อ ก และ ข 4. ผิดทั้งข้อ ก และ ข
เฉลยตอบ ข้อ 3 | กฎการสะท้อนของแสงที่สมบูรณ์มี 2 ข้อหลัก ซึ่งครอบคลุมทั้งพฤติกรรมความเท่ากันของมุม (ข้อ ก) และการอยู่บนระนาบทางเรขาคณิตเดียวกัน (ข้อ ข)
11. การทำให้บริสุทธิ์ที่สุด ทำได้โดยวิธีใด
1. การกรอง 2. การต้ม
3. การกลั่น 4. การใส่สารเคมีบางชนิด
เฉลยตอบ ข้อ 3 | การกลั่นแยกสารผสมโดยการทำให้ของเหลวกลายเป็นไอแยกตัวออกจากสิ่งเจือปน จากนั้นจึงควบแน่นกลับมาเป็นของเหลวบริสุทธิ์ ทำให้สารที่ได้มีความบริสุทธิ์สูงกว่าการกรองหรือต้มธรรมดา
12. ถ้าจัดทำการทดลองดังรูป จะมีแก๊สชนิดหนึ่งเกิดขึ้น ซึ่งมีผลทำให้เปลวเทียนดับ แก๊สชนิดนี้ควรเป็นแก๊สใดในข้อต่อไปนี้
[ แผนภาพการทดลอง ] หลอดทดลองใส่น้ำกรดเกลือ หยดลงในขวดรูปชมพู่ที่มีหินปูน ต่อท่อนำแก๊สไปจ่อที่เปลวเทียน
1. เป็นแก๊สเสียที่ถูกขับออกจากปอด 2. เป็นผลผลิตที่พืชได้จากการสังเคราะห์ด้วยแสง
3. เป็นแก๊สที่บรรจุในลูกโป่งสวรรค์ 4. เป็นแก๊สชนิดเดียวกับแก๊สที่ใช้ในการหุงต้ม
เฉลยตอบ ข้อ 1 | กรดเกลือ (HCl) ทำปฏิกิริยากับหินปูน (CaCO3) จะได้ แก๊สคาร์บอนไดออกไซด์ (CO2) ซึ่งมีคุณสมบัติไม่ช่วยให้ไฟติดและมีความหนาแน่นสูงจึงเข้าคลุมและดับเปลวเทียน ซึ่งแก๊สนี้เป็นแก๊สเสียที่มนุษย์หายใจขับออกจากปอด
13. พิจารณาแผนภาพการเปลี่ยนสถานะของสารดังนี้
← คายความร้อน ←
ของแข็ง → (ก) → ของเหลว → (ค) → แก๊ส
ของแข็ง ← (น) ← ของเหลว ← (ข) ← แก๊ส
→ ดูดความร้อน →
"การเกิดฝนตก" ควรเป็นการเปลี่ยนแปลงที่เกิดขึ้นในขั้นตอนใด
1. ก 2. ข
3. ค 4. ง
เฉลยตอบ ข้อ 2 | การเกิดฝนตกคือการที่ไอน้ำ (สถานะแก๊ส) ในอากาศได้รับความเย็นแล้วเกิดการควบแน่นเปลี่ยนสถานะกลับเป็นหยดน้ำ (ของเหลว) ซึ่งเป็นกระบวนการคายความร้อน ตรงกับตำแหน่งลูกศร ข
14. สารละลายข้อใดมีฤทธิ์เป็นเบสทั้งหมด
1. น้ำฝน น้ำอัดลม น้ำขี้เถ้า 2. น้ำมะนาว น้ำประปา น้ำปูนใส
3. น้ำกลั่น น้ำผงซักฟอก น้ำสบู่ 4. น้ำปูนใส น้ำสบู่ น้ำขี้เถ้า
เฉลยตอบ ข้อ 4 | สารที่มีฤทธิ์เป็นเบสจะเปลี่ยนกระดาษลิตมัสจากสีแดงเป็นสีน้ำเงิน รสชาติฝาด สารในกลุ่มข้อ 4 ล้วนเป็นเบสทั้งหมด ได้แก่ น้ำปูนใส น้ำสบู่ และน้ำขี้เถ้าละลายน้ำ
15. เราจำแนกหินออกเป็น 3 ชนิดใหญ่ๆ ตามข้อใด
1. หินอัคนี หินตะกอน หินดินดาน 2. หินอัคนี หินตะกอน หินแปร
3. หินชั้น หินตะกอน หินแปร 4. หินอ่อน หินทราย หินปูน
เฉลยตอบ ข้อ 2 | ตามหลักธรณีวิทยาจำแนกหินตามลักษณะทางกายภาพและการกำเนิดออกเป็น 3 ประเภทหลัก ได้แก่ หินอัคนี (เกิดจากแมกมา/ลาวาเย็นตัว) หินตะกอนหรือหินชั้น (เกิดจากการทับถมของตะกอน) และหินแปร (แปรสภาพจากความร้อนและความดัน)
16. สัตว์ในข้อใดที่มีอุณหภูมิร่างกายเปลี่ยนไปตามสิ่งแวดล้อม
1. กบ ปลาดุก 2. หนูนา เป็ด
3. เต่า โลมา 4. นกกระจอกเทศ จระเข้
เฉลยตอบ ข้อ 1 | สัตว์ที่อุณหภูมิร่างกายเปลี่ยนไปตามสภาพแวดล้อมจัดเป็น "สัตว์เลือดเย็น" ได้แก่ กลุ่มสัตว์ครึ่งบกครึ่งน้ำ (กบ) และกลุ่มปลา (ปลาดุก) ส่วนหนู เป็ด นก โลมา เป็นสัตว์เลือดอุ่น
17. ข้อใดไม่จำเป็นต้องเป็นลักษณะของสัตว์เลี้ยงลูกด้วยน้ำนม
1. มีกระดูกสันหลังและเลือดอุ่น 2. มีต่อมสร้างน้ำนม
3. มีขนลักษณะเป็นเส้น 4. ต้องอยู่บนบกเสมอ
เฉลยตอบ ข้อ 4 | สัตว์เลี้ยงลูกด้วยน้ำนมบางชนิดอาศัยและดำรงชีวิตอยู่ในน้ำตลอดชีวิต เช่น วาฬ โลมา พะยูน ดังนั้น ข้อ 4 จึงไม่จำเป็นต้องเป็นลักษณะของสัตว์กลุ่มนี้เสมอไป
18. "เป็นสัตว์เลือดอุ่น ออกลูกเป็นไข่ หายใจด้วยปอด" คือลักษณะทั่วไปของสัตว์ประเภทใด
1. สัตว์ปีก 2. สัตว์เลื้อยคลาน
3. สัตว์ครึ่งบกครึ่งน้ำ 4. สัตว์เลี้ยงลูกด้วยน้ำนม
เฉลยตอบ ข้อ 1 | สัตว์ปีกทุชนิดเป็นสัตว์เลือดอุ่น ออกลูกเป็นไข่ที่มีเปลือกแข็งหรือเปลือกเหนียวหุ้ม และใช้ปอดในการแลกเปลี่ยนแก๊สเพื่อหายใจ
19. สัตว์ในข้อใดไม่มีกระดูกสันหลังทั้งหมด
1. แมลงวัน ผึ้ง ผีเสื้อ นก 2. แมงมุม ผีเสื้อ ยุง ตะขาบ
3. ผีเสื้อ ไส้เดือน กิ้งกือ งู 4. แมงกะพรุน หอย ม้าน้ำ ปลิง
เฉลยตอบ ข้อ 2 | สัตว์ในข้อ 2 (แมงมุม ผีเสื้อ ยุง ตะขาบ) เป็นสัตว์ขาข้อหรือสัตว์ไม่มีกระดูกสันหลังทั้งหมด 100% ส่วนข้ออื่นมีสัตว์มีกระดูกสันหลังปนอยู่ เช่น นก (ข้อ 1), งู (ข้อ 3), ม้าน้ำ ซึ่งจัดเป็นปลา (ข้อ 4)
20. การรับประทานผักผลไม้เป็นประจำทุกวัน จะช่วยรักษาอวัยวะส่วนใดมากที่สุด
