สรุปเนื้อหาคณิตศาสตร์ ม.5: ฟังก์ชัน

1. สมาชิกของ (Element of)

สัญลักษณ์ ∈ ใช้บอกว่าสิ่งนั้นเป็นส่วนหนึ่งของกลุ่ม

ตัวอย่าง: ถ้า A = {1, 2, 3} จะได้ว่า 1 ∈ A

2. เกมพิชิตพื้นฐานฟังก์ชัน

3. นิยามและการตรวจสอบฟังก์ชัน

ฟังก์ชัน คือ ความสัมพันธ์ที่สมาชิกตัวหน้า (x) จับคู่กับสมาชิกตัวหลัง (y) ได้เพียงค่าเดียวเท่านั้น

4. โดเมนและเรนจ์

  • ตัวส่วนห้ามเป็นศูนย์
  • ค่าในรากคู่ (√) ต้อง ≥ 0

5. ฟังก์ชันประกอบและฟังก์ชันผกผัน

ฟังก์ชันประกอบ: (f ∘ g)(x) = f(g(x)) ทำจากในออกนอก

ฟังก์ชันผกผัน: สลับ x และ y (ต้องเป็นฟังก์ชัน 1-1)
