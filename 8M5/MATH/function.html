สรุปเนื้อหาคณิตศาสตร์ ม.5: ฟังก์ชัน (Function)

สมาชิกของ (Element of)

สัญลักษณ์ ∈ (สมาชิกของ) ใช้บอกว่าสิ่งนั้นเป็นส่วนหนึ่งของกลุ่มนั้นๆ

ตัวอย่าง: ถ้าเซต A = {1, 2, 3} จะได้ว่า 1 ∈ A

🎮 เกมพิชิตพื้นฐานฟังก์ชัน

1. นิยามและการตรวจสอบฟังก์ชัน

ฟังก์ชัน คือ ความสัมพันธ์ที่สมาชิกตัวหน้า (x) จับคู่กับสมาชิกตัวหลัง (y) ได้เพียงค่าเดียวเท่านั้น

2. โดเมนและเรนจ์

  • ตัวส่วนห้ามเป็นศูนย์
  • ค่าในรากคู่ (√) ต้อง ≥ 0

3. ฟังก์ชันประกอบและฟังก์ชันผกผัน

ฟังก์ชันประกอบ: (f ∘ g)(x) = f(g(x)) ทำจากข้างในออกข้างนอก
