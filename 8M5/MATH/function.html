สรุปเนื้อหาคณิตศาสตร์ ม.5: ฟังก์ชัน (Function)

สมาชิกของ (Element of)

สัญลักษณ์ ∈ (สมาชิกของ) ใช้บอกว่าสิ่งนั้นเป็นส่วนหนึ่งของกลุ่มหรือก๊วนนั้นๆ

ตัวอย่าง: ถ้าเซต A = {1, 2, 3} จะได้ว่า 1 ∈ A

🎮 Function Master: 4 ฐานพิชิตฟังก์ชัน

ฐานที่ 1: เรื่องเซต (Sets) - การเขียนเซต

1. นิยามและการตรวจสอบฟังก์ชัน

ฟังก์ชัน คือ ความสัมพันธ์ที่สมาชิกตัวหน้า (x) จับคู่กับสมาชิกตัวหลัง (y) ได้เพียงค่าเดียวเท่านั้น

ภาพประกอบ: เทคนิคการตรวจสอบว่าเป็นฟังก์ชันหรือไม่

2. โดเมนและเรนจ์ (Domain & Range)

  • ตัวส่วนห้ามเป็นศูนย์
  • ค่าในรากคู่ (√) ต้องมากกว่าหรือเท่ากับ 0

3. ฟังก์ชันประกอบและฟังก์ชันผกผัน

  • ฟังก์ชันประกอบ: (f ∘ g)(x) = f(g(x)) ทำจากข้างในออกข้างนอก
  • ฟังก์ชันผกผัน: หาได้จากการสลับที่ x และ y (ฟังก์ชันต้องเป็นแบบ 1-1 เท่านั้น)

4. กฎเหล็กเรื่องฟังก์ชัน

  • กฎการเป็นฟังก์ชัน: ตัวหน้าห้ามซ้ำ!
  • กฎฟังก์ชัน 1-1: สมาชิกเรนจ์ห้ามแย่งสมาชิกโดเมน
  • กฎฟังก์ชันประกอบ: สลับที่ไม่ได้ (f ∘ g ≠ g ∘ f)
