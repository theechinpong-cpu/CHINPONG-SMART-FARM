สรุปเนื้อหา: รากที่ n ของจำนวนจริง

รากที่ n ของจำนวนจริง a คือจำนวนจริง x ซึ่ง x^n = a โดยมีข้อควรระวังสำคัญคือ หาก n เป็นเลขคู่ รากที่ n ของจำนวนลบจะไม่อยู่ในระบบจำนวนจริง แต่หาก n เป็นเลขคี่ รากที่ n จะมีค่าเดียวเสมอและเครื่องหมายจะขึ้นอยู่กับ a ทั้งนี้การทำความเข้าใจเรื่องเลขยกกำลังที่มีเลขชี้กำลังเป็นเศษส่วน (a^(m/n)) จะช่วยให้การคำนวณรากที่ n ง่ายขึ้น โดยเปลี่ยนรูปเป็น n-th root ของ a^m

เทคนิคการจำ: รากเลขคู่ (n=2,4,6...) ให้ระวังเรื่องบวก-ลบ (±) และค่าที่เป็นลบภายในรูท ส่วนรากเลขคี่ (n=3,5,7...) เครื่องหมายเดิมอย่างไรผลลัพธ์ก็เป็นอย่างนั้น
