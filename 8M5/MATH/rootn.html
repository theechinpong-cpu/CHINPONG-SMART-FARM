สรุปเนื้อหา: รากที่ n ของจำนวนจริง

รากที่ n ของจำนวนจริง a คือจำนวนจริง x ที่ยกกำลัง n แล้วได้ a (x^n = a)... [เนื้อหา 400 คำขึ้นไป...]

เทคนิคการจำ:

1. ถ้า n เป็นเลขคี่: รากที่ n มีเพียงค่าเดียวเสมอ (บวกหรือลบตาม a)
2. ถ้า n เป็นเลขคู่: ถ้า a > 0 จะมี 2 ราก (บวกและลบ), ถ้า a = 0 มี 1 ราก, ถ้า a < 0 ไม่มีรากที่เป็นจำนวนจริง

ข้อ 1: จงหาค่าของรากที่ 3 ของ -64

คำใบ้: จำนวนใดคูณกัน 3 ครั้งได้ -64

เขียนโดย พระจอมเกล้าติวเตอร์
