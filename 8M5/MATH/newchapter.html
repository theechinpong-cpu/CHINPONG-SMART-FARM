แบบทดสอบ: เลขยกกำลังเลขชี้กำลังเป็นจำนวนตรรกยะ

📖 สรุปเนื้อหา: เลขยกกำลังที่มีเลขชี้กำลังเป็นจำนวนตรรกยะ

เลขยกกำลังในรูปแบบนี้คือการขยายขอบเขตจากเลขชี้กำลังที่เป็นจำนวนเต็ม ไปสู่เลขชี้กำลังที่เป็นเศษส่วน ซึ่งมีความสัมพันธ์โดยตรงกับราก (Roots) ของจำนวนจริง

1. นิยามและสมบัติที่สำคัญ

นิยามรากที่ n: สำหรับ a ที่เป็นจำนวนจริง และ n เป็นจำนวนเต็มบวกที่มากกว่า 1

a1/n = n√a

นิยามเลขชี้กำลังที่เป็นเศษส่วน: สำหรับ a > 0 และ r = p/q (เศษส่วนอย่างต่ำ)

ap/q = q√ap = (q√a)p

2. ข้อควรระวังและเงื่อนไข (สำคัญมาก!)

  • ฐานติดลบ: ถ้า a < 0 จะหา a1/n ได้ก็ต่อเมื่อ n เป็นเลขคี่เท่านั้น
  • รูปเศษส่วนอย่างต่ำ: ต้องทอนเลขชี้กำลังให้เป็นเศษส่วนอย่างต่ำเสมอก่อนคำนวณ
  • กรณี a = 0: 0r = 0 สำหรับเลขชี้กำลังที่เป็นบวก แต่ถ้าเลขชี้กำลังเป็นลบหรือศูนย์จะไม่นิยาม

3. ทฤษฎีบทเลขยกกำลัง

สมบัติ อธิบาย
am × an = am+nฐานเหมือนกัน คูณกัน เอาเลขชี้กำลังมาบวก
(am)n = am×nเลขชี้กำลังซ้อนกัน เอามาคูณกัน
(a × b)n = an × bnกระจายเลขชี้กำลังเข้าไปในวงเล็บคูณ
(a/b)n = an / bnกระจายเลขชี้กำลังเข้าไปในวงเล็บหาร
am / an = am-nฐานเหมือนกัน หารกัน เอาเลขชี้กำลังมาลบ

💡 Memory Tricks & กระบวนการคิด

  • "บนเป็นกำลัง ล่างเป็นราก": เลขชี้กำลัง p/q ตัวบนคือกำลัง ตัวล่างคือราก เช่น x2/3 = 3√x2
  • จัดการฐานก่อน: แปลงฐานให้เป็นเลขฐานเฉพาะที่เล็กที่สุดก่อน (เช่น 8 → 23) เพื่อให้ตัดเลขได้ง่าย
  • เช็คเงื่อนไขเสมอ: ถ้าฐานติดลบ ให้ดูที่ตัวส่วนของเศษส่วนอย่างต่ำ ถ้าเป็นเลขคู่ = ไม่เป็นจำนวนจริง
