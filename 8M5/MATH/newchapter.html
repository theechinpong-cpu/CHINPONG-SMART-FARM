แบบทดสอบ: เลขยกกำลังที่มีเลขชี้กำลังเป็นจำนวนตรรกยะ

📖 สรุปเนื้อหา: เลขยกกำลังที่มีเลขชี้กำลังเป็นจำนวนตรรกยะ

เลขยกกำลังในรูปแบบนี้คือการขยายขอบเขตจากเลขชี้กำลังที่เป็นจำนวนเต็ม ไปสู่เลขชี้กำลังที่เป็นเศษส่วน ซึ่งมีความสัมพันธ์โดยตรงกับราก (Roots) ของจำนวนจริง

1. นิยามและสมบัติที่สำคัญ

นิยามรากที่ n: a1/n = n√a

นิยามเลขชี้กำลังที่เป็นเศษส่วน: ap/q = q√ap = (q√a)p

2. ข้อควรระวัง

  • ฐานติดลบ: กรณี a < 0 จะหา a1/n ได้ก็ต่อเมื่อ n เป็นเลขคี่เท่านั้น
  • การคำนวณ: ต้องทอนเลขชี้กำลังให้เป็นเศษส่วนอย่างต่ำเสมอก่อนคำนวณ
  • กรณี a = 0: 0r = 0 (เมื่อ r > 0) แต่ถ้า r ≤ 0 จะไม่นิยาม

3. ทฤษฎีบทเลขยกกำลัง

สมบัติอธิบาย
am × an = am+nฐานเหมือนกัน คูณกัน เอาเลขชี้กำลังมาบวก
(am)n = am×nเลขชี้กำลังซ้อนกัน เอามาคูณกัน
am ÷ an = am-nฐานเหมือนกัน หารกัน เอาเลขชี้กำลังมาลบ

💡 Memory Tricks: "บนเป็นกำลัง ล่างเป็นราก"

จัดการฐานให้เป็นเลขฐานเฉพาะเล็กสุดเสมอ และระวังรากเลขคู่ของเลขติดลบ!
