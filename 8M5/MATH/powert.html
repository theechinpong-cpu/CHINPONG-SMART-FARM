แบบทดสอบ + คู่มือครูเฉลยละเอียด: รากที่ n ในระบบจำนวนจริง

ระดับมัธยมศึกษาปีที่ 5 (ความยากปานกลาง) | อ้างอิงโรงเรียนกัลยาณีศรีธรรมราช

สรุปเนื้อหาบทเรียน: รากที่ n ในระบบจำนวนจริง (Radicals)

บทเรียนเรื่องรากที่ n ในระบบจำนวนจริง เป็นพื้นฐานทางคณิตศาสตร์ที่สำคัญสำหรับนักเรียนชั้นมัธยมศึกษาปีที่ 5 เพื่อนำไปต่อยอดในการแก้สมการอตรรกยะ ฟังก์ชันกำลัง และแคลคูลัสในระดับสูง การเข้าใจนิยามและสมบัติของกรณฑ์อย่างถูกต้องจะช่วยให้สามารถแก้ปัญหาได้อย่างแม่นยำ

1. นิยามของรากที่ n (Definition of nth Root)

ให้ x และ y เป็นจำนวนจริง และ n เป็นจำนวนเต็มบวกที่มากกว่า 1 ถ้า yⁿ = x แล้ว จะเรียก y ว่าเป็น "รากที่ n ของ x"

  • กรณี n เป็นจำนวนคู่: จำนวนจริงบวก x จะมีรากที่ n อยู่ 2 ค่าเสมอ คือ ค่าบวก และ ค่าลบ ตัวอย่างเช่น รากที่ 2 ของ 16 คือ 4 และ -4 เนื่องจาก (4)² = 16 และ (-4)² = 16 สำหรับจำนวนจริงลบ จะไม่มีรากที่ n เป็นจำนวนจริงเมื่อ n เป็นจำนวนคู่
  • กรณี n เป็นจำนวนคี่: จำนวนจริง x ใดๆ (ไม่ว่าบวก ลบ หรือศูนย์) จะมีรากที่ n เพียงค่าเดียวเท่านั้น และจะมีเครื่องหมายเดียวกับ x ตัวอย่างเช่น รากที่ 3 ของ 27 คือ 3 และ รากที่ 3 ของ -8 คือ -2

💡 เทคนิคการจำ / กระบวนการคิด: ราก vs ค่าหลัก

รากที่ n (มีคำว่า "รากที่"): ถ้ายกกำลังคู่ ต้องตอบทั้งค่าบวกและค่าลบ เช่น รากที่ 4 ของ 81 ตอบ 3 และ -3

ค่าหลักของรากที่ n (เครื่องหมาย ⁿ√): สัญลักษณ์ ⁿ√x หมายถึง "ค่าหลัก" เพียงค่าเดียวเท่านั้น! ถ้า n เป็นคู่ ผลลัพธ์จากการถอดสัญลักษณ์กรณฑ์จะต้องเป็นจำนวนจริงบวกหรือศูนย์เสมอ (ห้ามติดลบ)

2. ค่าหลักของรากที่ n (Principal nth Root)

เพื่อป้องกันความสับสนในสัญลักษณ์ นักคณิตศาสตร์ได้กำหนดให้สัญลักษณ์ ⁿ√x แทน "ค่าหลักของรากที่ n ของ x"

  • ถ้า n เป็นจำนวนคู่ และ x ≥ 0 ค่าหลัก ⁿ√x ต้อง ≥ 0 เสมอ เช่น √16 = 4 (ไม่ตอบ -4)
  • ถ้า n เป็นจำนวนคี่ ค่าหลัก ⁿ√x จะมีเครื่องหมายตาม x เช่น ³√27 = 3 และ ³√(-27) = -3

3. สมบัติสำคัญของกรณฑ์ที่ต้องระวัง

ให้ a และ b เป็นจำนวนจริงที่มีรากที่ n และ n เป็นจำนวนเต็มบวกที่มากกว่า 1

  • การคูณและการหาร: ⁿ√(ab) = ⁿ√a · ⁿ√b และ ⁿ√(a/b) = ⁿ√a / ⁿ√b (โดย b ≠ 0)
  • การถอดรากกำลังคู่: ⁿ√(aⁿ) = |a| เมื่อ n เป็นจำนวนคู่ (สำคัญมาก: ต้องใส่ Absolute เสมอหากไม่ทราบค่าของตัวแปร เพื่อประกันว่าผลลัพธ์ไม่ติดลบ)
  • การถอดรากกำลังคี่: ⁿ√(aⁿ) = a เมื่อ n เป็นจำนวนคี่ (เครื่องหมายคงเดิมตาม a)
  • ยกกำลังนอกกรณฑ์: (ⁿ√a)ⁿ = a (โดยที่ a ต้องอยู่ในโดเมนที่ทำให้ ⁿ√a เป็นจำนวนจริง)

⚠️ ข้อควรระวังและการทำตัวส่วนไม่ให้ติดกรณฑ์ (Rationalization)

1. ข้อห้ามยุบรวม: √(a + b) ≠ √a + √b และ √(a - b) ≠ √a - √b ห้ามกระจายหรือยุบรวมรากกับการบวกลบเด็ดขาด!

2. การจัดรูปไม่ให้ตัวส่วนติดกรณฑ์:
- กรณีพจน์เดียว: คูณทั้งเศษและส่วนด้วยตัวส่วนนั้น เช่น a/√b = (a√b)/b
- กรณีสองพจน์บวกลบ: ใช้สังยุค (Conjugate) และสูตรผลต่างกำลังสอง (A-B)(A+B) = A² - B² เช่น c/(√a - √b) ให้คูณด้วย (√a + √b)/(√a + √b)
