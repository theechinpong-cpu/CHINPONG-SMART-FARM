วิทยาศาสตร์ ป.5: การเปลี่ยนแปลงของสาร

แบบทดสอบวัดผลสัมฤทธิ์ บทที่ 3 (ระดับ Advance)

คัมภีร์ลับวิทย์ ป.5 บทที่ 3

รวมเทคนิคจาก Tutor ชื่อดัง: การเปลี่ยนแปลงของสาร (Advanced Version)

1. จุดตาย: การเปลี่ยนสถานะ (Physical Change)

ติวเตอร์ทุกคนเน้นย้ำเรื่อง "ทิศทางพลังงาน" จำง่ายๆ ด้วยสูตร "ดูด-พอง-ร้อน / คาย-ยุบ-เย็น"

🔴 กระบวนการ "ดูดความร้อน"

สูตรจำ: "ดูดเพื่อสลาย" (สลายแรงยึดเหนี่ยวเพื่อแยกกันอยู่)

  • หลอมเหลว: แข็ง → เหลว (น้ำแข็งละลาย)
  • กลายเป็นไอ: เหลว → แก๊ส (ต้มน้ำเดือด)
  • ระเหิด: แข็ง → แก๊ส (ลูกเหม็น/พิมเสน)

🔵 กระบวนการ "คายความร้อน"

สูตรจำ: "คายเพื่อสร้าง" (สร้างแรงยึดเหนี่ยวเพื่อกลับมาเกาะกัน)

  • แข็งตัว: เหลว → แข็ง (ทำน้ำแข็ง)
  • ควบแน่น: แก๊ส → เหลว (ฝน/หยดน้ำข้างแก้ว)
  • ระเหิดกลับ: แก๊ส → แข็ง (หิมะ/แม่คะนิ้ง)

2. การละลาย (Solubility) - สูตรคำนวณที่ต้องเป๊ะ

สูตรเด็ดจาก Tutor: "ตัวทำคือเจ้าบ้าน ตัวละลายคือแขก"

"สารละลาย = ตัวทำละลาย + ตัวละลาย"

📊 สูตรคำนวณความเข้มข้นร้อยละ (ป.5 เริ่มได้ใช้):

ร้อยละโดยมวล = (มวลของตัวละลาย / มวลของสารละลายทั้งหมด) x 100

⚠️ จุดห้ามพลาด: มวลของสารละลาย "ต้อง" เอามวลน้ำ + มวลเกลือ/น้ำตาล เข้าด้วยกันก่อนหาร!

3. การเปลี่ยนแปลงทางเคมี (Chemical Change)

ติวเตอร์สรุปไว้ 5 สัญญาณอันตราย ถ้าเจอสิ่งนี้คือ "ทางเคมี" แน่นอน:

สัญญาณ ตัวอย่างในข้อสอบ
1. สีเปลี่ยน แอปเปิ้ลปอกทิ้งไว้แล้วคล้ำ, บ่มมะม่วงแล้วเหลือง
2. กลิ่น/รสเปลี่ยน นมบูด, อาหารเน่าเสีย
3. ฟองแก๊ส เบกกิ้งโซดาผสมน้ำส้มสายชู, โลหะแช่กรด
4. ตะกอน น้ำปูนใสพ่นลมหายใจเข้า (เกิดแคลเซียมคาร์บอเนต)
5. ระเบิด/แสง/ร้อน จุดพลุ, เผาฟืน, การเกิดสนิม

⚡ เทคนิคจำขั้นเทพ (Mnemonic) ⚡

  • ✅ "ระเหิด" คือของแข็งกลายเป็นไอ (เหมือนลูกเหม็นหายไปในอากาศ)
  • ✅ "ควบแน่น" คือแก๊สอยากกลับเป็นน้ำ (เหมือนฝนที่ตกลงมา)
  • ✅ "เคมี" คือเกิดสารใหม่ ย้อนกลับยาก (เหมือนทอดไข่ กลับไปดิบไม่ได้)
  • ✅ "กายภาพ" คือแค่เปลี่ยนร่าง แต่ใจเดิม (เหมือนน้ำแข็ง กลายเป็นน้ำ ก็ยังคือน้ำ)

🎬 ห้องทดลองเสมือน (ฉบับแก้ไข: รันได้ 100%)

🧪 การเปลี่ยนสถานะของสาร (สรุปชัดเจน)
