ข้อสอบวิชาเคมี สอวน. ค่าย 1 (2567)

วิเคราะห์ผลและเฉลยละเอียดด้วยระบบอัตโนมัติ

สรุปเนื้อหาและเทคนิค (400+ คำ)

การสอบ สอวน. เคมี ค่าย 1 เน้นทักษะพื้นฐานที่แม่นยำ บทที่ 1 ความปลอดภัย ต้องจำสัญลักษณ์ GHS ให้แม่น โดยเฉพาะความแตกต่างระหว่าง 'สารไวไฟ' (เปลวไฟ) และ 'สารออกซิไดซ์' (เปลวไฟเหนือวงกลม) บทที่ 2 ทักษะคำนวณ การปัดเศษทศนิยมตามหลักเลขนัยสำคัญเป็นเรื่องที่นักเรียนพลาดบ่อยที่สุด กฎคือการคูณหารให้ยึดจำนวนตัวเลขที่มีนัยสำคัญน้อยที่สุด ส่วนการบวกลบให้ยึดตำแหน่งทศนิยมที่น้อยที่สุด

เทคนิคการจำ: แนวโน้มตารางธาตุ "ขวาบนสูง" (IE, EN, EA สูง) "ซ้ายล่างใหญ่" (ขนาดอะตอมใหญ่) ในเรื่องพันธะเคมี มุมพันธะจะแคบลงเมื่อมีอิเล็กตรอนคู่โดดเดี่ยวเพิ่มขึ้น และอย่าลืมว่าแรงยึดเหนี่ยวระหว่างโมเลกุลมีผลโดยตรงต่อจุดเดือดและจุดหลอมเหลว

การคิดทศนิยม: ในข้อสอบชุดนี้ หากต้องคำนวณ เช่น 40.125 หากต้องการทศนิยม 2 ตำแหน่ง ให้ดูตำแหน่งที่ 3 คือ '5' กฎสากลคือปัดขึ้นเสมอเป็น 40.13 แต่ถ้าโจทย์ระบุเลขนัยสำคัญ 3 ตัว ต้องตอบ 40.1 เพื่อความแม่นยำตามหลักการทางวิทยาศาสตร์

ข้อที่ 1: สารเคมีชนิดหนึ่งระคายเคืองผิวหนัง เป็นมะเร็งเมื่อได้รับระยะยาว และจุดวาบไฟ 20 °C สัญลักษณ์ GHS ควรเป็นอย่างไร?

เฉลย: ก.
วิธีคิด: จุดวาบไฟ < 60°C คือไวไฟ (เปลวไฟ), ระคายเคืองคือ (!), ก่อมะเร็งคืออันตรายต่อสุขภาพ

ข้อที่ 2: ข้อใดปฏิบัติได้ถูกต้องที่สุดในห้องปฏิบัติการ?

เฉลย: ข.
วิธีคิด: การสำรวจต้นตอเพื่อระงับเหตุเป็นสิ่งที่ควรทำ ส่วนการดมโดยตรงหรือใช้ยาสีฟันทาแผลเป็นวิธีที่ผิด

ข้อที่ 4: หน่วยปาสคาล (Pa) เทียบเท่ากับหน่วยเอสไอพื้นฐานใด?

เฉลย: ข.
วิธีคิด: P = F/A = (ma)/A = (kg·m/s²) / m² = kg·m⁻¹·s⁻²

ข้อที่ 9: ธาตุ X มี 3 ไอโซโทป (2a+4), (2a+6), (2a+8) อัตราส่วน 5:2:1 มวลอะตอมเฉลี่ยคือ?

เฉลย: ก.
วิธีคิด: [(2a+4)*5 + (2a+6)*2 + (2a+8)*1] / 8 = (16a + 40) / 8 = 2a + 5
