สรุปเนื้อหาเคมี สอวน. ค่าย 1 (บทนำและโครงสร้างอะตอม)

สรุปสาระสำคัญ (โดย พระจอมเกล้าติวเตอร์):

ในการสอบ สอวน. เคมี ช่วงข้อต้นๆ มักเน้นเรื่องความปลอดภัยในห้องปฏิบัติการ (GHS Symbols), การวัดและหน่วย (SI Units), และโครงสร้างอะตอม สัญลักษณ์นิวเคลียร์คือหัวใจสำคัญ โดยเลขล่างคือเลขอะตอม (จำนวนโปรตอน) และเลขบนคือเลขมวล (โปรตอน + นิวตรอน) ไอโซโทปคือธาตุชนิดเดียวกันที่มีนิวตรอนต่างกัน [cite: 5, 46]

เรื่องพันธะเคมี สารประกอบไอออนิกเกิดจากโลหะ+อโลหะ มีจุดเดือดสูงและนำไฟฟ้าเมื่อหลอมเหลว ส่วนสารโคเวเลนต์เกิดจากอโลหะ+อโลหะ มักไม่นำไฟฟ้า การคำนวณปริมาณสารสัมพันธ์ต้องแม่นเรื่อง "โมล" (Mole) ซึ่งเชื่อมโยงระหว่างมวล (g), จำนวนอนุภาค (6.02x10^23), และปริมาตรแก๊สที่ STP (22.4 L) [cite: 9, 93]

💡 ทริคการจำ: "บนลบข้างล่างได้นิวตรอน" / "ไอโซโทป - ป.ปลาคือโปรตอนเท่า" / "GHS รูปหัวใจระเบิด = ก่อมะเร็ง/อันตรายเรื้อรัง"

1. สารเคมีที่มีจุดวาบไฟ 20°C ระคายเคืองผิวหนัง และก่อมะเร็งหากรับต่อเนื่องหลายปี ควรมีสัญลักษณ์ GHS ใดบ้าง? [cite: 10, 23]

ก. เปลวไฟ, เครื่องหมายตกใจ, ร่างกายมีรัศมี
ข. เปลวไฟ, หัวกะโหลก, ร่างกายมีรัศมี
ค. วัตถุระเบิด, เครื่องหมายตกใจ, ร่างกายมีรัศมี
ง. เปลวไฟ, การกัดกร่อน, หัวกะโหลก
เฉลย: ก.
คำอธิบาย: จุดวาบไฟต่ำ (20°C) = ไวไฟ (เปลวไฟ), ระคายเคือง = เครื่องหมายตกใจ (!), ก่อมะเร็ง = อันตรายต่อสุขภาพระยะยาว (สัญลักษณ์คนมีรัศมีที่หน้าอก) [cite: 23, 246]

2. หน่วย "ปาสคาล" (Pa) ซึ่งสอดคล้องกับ P = F/A เทียบได้กับหน่วยเอสไอพื้นฐานในข้อใด? [cite: 26, 263]

ก. kg·m·s⁻²
ข. g·cm·s⁻²
ค. kg·m⁻¹·s⁻²
ง. N·m⁻²
เฉลย: ค.
คำอธิบาย: จาก P = F/A; F = ma (kg·m/s²) และ A = m² ดังนั้น P = (kg·m/s²) / m² = kg·m⁻¹·s⁻² ซึ่งมีค่าเท่ากับ N/m² (แต่โจทย์ถามหน่วยพื้นฐาน) [cite: 204, 263]

3. ธาตุ X มี 3 ไอโซโทป คือ 2a+4, 2a+6 และ 2a+8 โดยมีอัตราส่วนโดยมวลเป็น 5:2:1 มวลอะตอมเฉลี่ยคือเท่าใด? [cite: 44]

ก. 2a + 5
ข. 2a + 5.5
ค. 2a + 6
ง. 2a + 6.5
เฉลย: ก.
คำอธิบาย: มวลเฉลี่ย = [(2a+4)*5 + (2a+6)*2 + (2a+8)*1] / (5+2+1) = [10a+20 + 4a+12 + 2a+8] / 8 = (16a+40)/8 = 2a + 5 [cite: 44]
