Science Foundations: ม.1 Intensive Exam

สรุปเนื้อหาสำคัญ (400+ คำ)

1. กระบวนการทางวิทยาศาสตร์: การคิดแบบนักวิทยาศาสตร์เริ่มต้นที่การตั้งคำถาม (Questioning) บนพื้นฐานของการสังเกต หากต้องการทราบผลของปัจจัยหนึ่ง ให้ใช้หลัก "Fair Test" คือควบคุมตัวแปรอื่นให้คงที่ (Controlled Variable) เปลี่ยนเฉพาะตัวแปรต้น (Independent Variable) เท่านั้น
Memory Trick: ให้ท่องว่า "ต้นต้องเปลี่ยน (I), ตามต้องวัด (D), ควบคุมให้คงที่ (C)"

2. สารรอบตัว: สสารประกอบด้วยอนุภาคที่มีความหนาแน่นต่างกัน $d = m/v$ สารบริสุทธิ์ (ธาตุ/สารประกอบ) มีจุดเดือด/จุดหลอมเหลวคงที่
Thought Process: ก่อนแยกสาร ให้ถามว่า "สารนั้นมีสมบัติอะไรต่างกัน?" เช่น ขนาดอนุภาค (การกรอง), จุดเดือด (การกลั่น), หรือความหนาแน่น

3. เซลล์และหน่วยชีวิต: พืชและสัตว์มีโครงสร้างต่างกัน พืชมีผนังเซลล์ (Cell Wall) เพิ่มความแข็งแรงและคลอโรพลาสต์ (Chloroplast) สำหรับผลิตอาหารผ่านแสง
Memory Trick: "X-Water" (ไซเล็ม = ลำเลียงน้ำ), "Phloem-Food" (โฟลเอ็ม = ลำเลียงอาหาร)

4. พลังงานความร้อน: ความร้อนถ่ายโอนผ่านอนุภาค (การนำ - ของแข็ง), ของไหล (การพา - ของเหลว/ก๊าซ), และคลื่น (การแผ่รังสี - สุญญากาศ)
Thought Process: ถ้าโจทย์ไม่พูดถึงตัวกลาง ให้เดาไว้ก่อนว่าเป็นการแผ่รังสี
