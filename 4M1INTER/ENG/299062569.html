หลักสูตร ม.1 (EP / MEP / Gifted) อ้างอิงหนังสือเรียน Gold Experience 1 (Unit 1 & 2)

1. Unit 1: Possessive Adjectives vs. Possessive Pronouns

การแสดงความเป็นเจ้าของในภาษาอังกฤษแบ่งออกเป็น 2 ประเภทหลักตามโครงสร้างไวยากรณ์ ซึ่งห้ามจำสับสนเด็ดขาด:

  • Possessive Adjectives (my, your, his, her, its, our, their): ทำหน้าที่ขยายคำนาม ต้องมีคำนามตามหลังเสมอ เช่น "This is my room."
  • Possessive Pronouns (mine, yours, his, hers, ours, theirs): ทำหน้าที่แทนคำนามที่กล่าวไปแล้ว ห้ามมีคำนามตามหลังเด็ดขาด เช่น "This room is mine."
🧠 Thought Process / Memory Trick:

ท่องให้ขึ้นใจ: "Adjective มีนามท้าย / Pronoun บินเดี่ยวห้ามมีนาม" (สังเกตว่า Pronoun ส่วนใหญ่จะเติม s ยกเว้น mine และ his เพราะสรีระคำเอื้ออยู่แล้ว)

2. Unit 1: There is / There are & Quantifiers

โครงสร้างประโยคระบุการมีอยู่ของสิ่งของประกอบกับการเลือกใช้คำบอกปริมาณ (Quantifiers) ที่เหมาะสม:

  • There is + นามเอกพจน์/นามนับไม่ได้: ใช้กับสิ่งเดียวหรือของที่นับไม่ได้ เช่น There is a pen. / There is some water.
  • There are + นามพหูพจน์: ใช้กับสิ่งที่มีตั้งแต่สองสิ่งขึ้นไป เช่น There are some books.
  • a / an: ใช้ในประโยคบอกเล่า/ปฏิเสธ/คำถาม กับนามนับได้เอกพจน์เท่านั้น (an นำหน้าเสียงสระ)
  • some: ใช้ในประโยค บอกเล่า (ใช้ได้ทั้งนามพหูพจน์และนับไม่ได้) หรือใช้ในการยื่นข้อเสนอ/ขอร้อง
  • any: ใช้ในประโยค ปฏิเสธและคำถาม (ใช้ได้ทั้งนามพหูพจน์และนับไม่ได้)
🧠 Thought Process / Memory Trick:

กฎสามเหลี่ยมปริมาณ: "Some บอกเล่า / Any ปฏิเสธ-คำถาม / Single เติม a-an" หากเห็นประโยคติดลบ (not) หรือลงท้ายด้วยเครื่องหมายคำถาม ให้พุ่งเป้าไปที่ any ทันที!

3. Unit 2: Present Simple Tense & Subject-Verb Agreement

ใช้พูดถึงเหตุการณ์ที่เป็นจริงตามธรรมชาติ หรือกิจวัตรประจำวัน (Routine) โครงสร้างที่ออกข้อสอบบ่อยที่สุดคือกฎการเติม s/es หรือ "The S-Rule":

  • ประธานเอกพจน์ (He, She, It, Tom): กริยาแท้ต้องเติม s หรือ es เสมอ
  • ประธานพหูพจน์ (I, You, We, They, Tom and Ken): กริยาแท้คงรูปเดิม (Infinitive ไม่เติม s)
  • ในการปฏิเสธและคำถาม: ใช้ Does / Doesn't กับประธานเอกพจน์ และใช้ Do / Don't กับประธานพหูพจน์ (เมื่อมี do/does เข้ามาแล้ว กริยาแท้ต้องกลับไปเป็นรูปเดิมที่ไม่เติม s เสมอ)
🧠 Thought Process / Memory Trick:

กฎแห่งความสมดุล (The S-Rule): "ประธานเหงา (เอกพจน์) กริยาเติม S / ประธานพวกเยอะ (พหูพจน์) กริยาไร้ S" จำไว้ว่า S จะอยู่ที่ใดที่หนึ่งเสมอ ไม่ประธานก็กริยา!

4. Unit 2: Adverbs of Frequency

คำกริยาวิเศษณ์บอกความถี่ (always, usually, often, sometimes, never) มีตำแหน่งการจัดวางในประโยคที่ตายตัวและเป็นจุดหลอกในข้อสอบ:

  • วางไว้ หน้ากริยาแท้ (เช่น I always go to school.)
  • วางไว้ หลังกริยาช่วย Verb to be (is, am, are) (เช่น He is never late.)
🧠 Thought Process / Memory Trick:

สูตรลัดตำแหน่งความถี่: "หน้ากริยาหลัก หลังกริยา be" ท่องสั้นๆ "หน้าหลัก หลัง be" หลับตาก็แยกลำดับประโยคได้แม่นยำ

5. Units 1-2: Wh-Questions

คำสรรพนามที่ใช้ตั้งคำถามในชีวิตประจำวัน ต้องตีความข้อมูลที่โจทย์ต้องการคำตอบให้สัมพันธ์กัน:

  • Who: ถามถึง บุคคล | What: ถามถึง สิ่งของ/กิจกรรม | Where: ถามถึง สถานที่
  • When: ถามถึง เวลา | Which: ถามให้เลือกสิ่งใดสิ่งหนึ่ง | Whose: ถามหา เจ้าของ (โครงสร้างมักตามด้วยนาม: Whose pen is this?)
🧠 Thought Process / Memory Trick:

เชื่อมโยงคำตอบ: "ดูคำตอบย้อนกลับหาคำถาม" หากคำตอบเป็นเวลา ตอบ When, หากคำตอบเป็นชื่อคนแสดงความเป็นเจ้าของ ('s) ตอบ Whose ทันที
