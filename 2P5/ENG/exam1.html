ข้อสอบกลางภาคเรียนที่ 1 วิชาภาษาอังกฤษ ชั้นประถมศึกษาปีที่ 5
(ระดับยากเข้มข้น - อ้างอิงแนวทางดรุณศึกษา)

💡 สรุปเนื้อหาสำคัญและเทคนิคคิดเชิงระบบ (Thought Process)

ยินดีต้อนรับสู่น้องๆ ทุกคน ชุดข้อสอบนี้สร้างขึ้นเพื่อทดสอบความพร้อมในระดับวิเคราะห์และประยุกต์ขั้นสูง ประกอบด้วย 4 ส่วนสำคัญหลักที่มักเป็นจุดตัดคะแนนในการสอบแข่งขันและสอบกลางภาค:

1. Present Simple vs Present Continuous

Thought Process: การแยกแยะสอง Tense นี้ให้มองหาคำบ่งบอกเวลา (Time Expressions) เป็นอันดับแรก หากประโยคแสดงถึงสัจธรรม ความเป็นจริงทั่วไป หรือนิสัยที่ทำเป็นประจำ (เช่น always, usually, every day) โครงสร้างคือ Present Simple (S + V.1 โดยประธานเอกพจน์กริยาต้องเติม s/es) แต่หากเป็นเหตุการณ์ที่กำลังดำเนินอยู่ในขณะที่พูด มีความตื่นตัว (เช่น Look!, Listen!, now, at present) จะใช้ Present Continuous (S + is/am/are + V.ing) เสมอ

Memory Trick: "ประจำเติม s/es ถ้านิ่ง — กำลังทำวิ่งซิ่งเติม is/am/are + V.ing"

2. Quantifiers (คำบอกปริมาณ: Some, Any, Much, Many, Few, Little)

Thought Process: ขั้นแรกให้ตรวจสอบคำนามที่ตามหลังว่าเป็นนามนับได้พหูพจน์ (Countable Plural) หรือนามนับไม่ได้ (Uncountable) นามนับไม่ได้ห้ามเติม s และห้ามใช้ Many/Few เด็ดขาด! จากนั้นดูประเภทประโยค ถ้าประโยคบอกเล่า/ขอร้องยื่นข้อเสนอใช้ Some ถ้าปฏิเสธ/คำถามทั่วไปใช้ Any

Memory Trick: "Many พหูพจน์จำไว้, Much นามนับไม่ได้ก้อนใหญ่, Some บอกเล่าทั่วไป, Any ปฏิเสธ-ถามไถ่จำขึ้นใจ"

3. Comparison (การเปรียบเทียบขั้นกว่าและขั้นสุด)

Thought Process: มองหาคำเชื่อมในประโยค ถ้าเจอคำว่า "than" แสดงว่าเป็นขั้นกว่า (Comparative) ต้องใช้โครงสร้าง adjective-er หรือ more + adjective พยางค์ยาว แต่ถ้ามีคำว่า "the" นำหน้าช่องว่างและระบุขอบเขตความเป็นที่สุด (เช่น in the world, of all) แสดงว่าเป็นขั้นสุด (Superlative) ต้องใช้ adjective-est หรือ most + adjective

Memory Trick: "ขั้นกว่ามี than เคียงข้าง (-er/more), ขั้นสุดมี the นำทาง (-est/most)"

4. Health, Symptoms & Advising (อาการเจ็บป่วยและการให้คำแนะนำ)

Thought Process: วิเคราะห์อาการที่โจทย์กำหนด (เช่น toothache, sore throat) เพื่อเลือกคำแนะนำให้ตรงจุดผ่านโครงสร้าง Should (ควรจะ) หรือ Shouldn't (ไม่ควรจะ) ระวังคำกริยาที่ตามหลัง Should/Shouldn't ต้องเป็นกริยาฐาน (V.inf) ไม่เติมสิ่งใดๆ ทั้งสิ้น
