คลังบทเรียนและข้อสอบ Interactive 50 ข้อ: บทที่ 1 เลขยกกำลังและรากที่ n ของจำนวนจริง

ออกแบบโดยระบบผู้เชี่ยวชาญระดับมืออาชีพ มุ่งเน้นการคิดเชิงระบบ (Thought Process)

(1) บทสรุปเนื้อหาหลักและการคิดเชิงระบบ (Thought Process Summary)

เนื้อหาหลักที่ต้องทำความเข้าใจ: บทเรียนนี้แบ่งโครงสร้างหลักออกเป็น 4 ส่วนสำคัญ ได้แก่ เลขยกกำลังที่มีเลขชี้กำลังเป็นจำนวนเต็ม, รากที่ n ของจำนวนจริงและค่าหลัก, การดำเนินการของจำนวนในรูปกรณฑ์ และเลขยกกำลังที่มีเลขชี้กำลังเป็นจำนวนตรรกยะพร้อมสมการ นิยามพื้นฐานของเลขยกกำลังคือการคูณซ้ำกันของฐานเดิม x เป็นจำนวน n ตัว (x^n) โดยมีเงื่อนไขสำคัญที่นักเรียนมักผิดพลาดคือ x^0 = 1 เสมอเมื่อ x ไม่เท่ากับ 0 และ x^(-n) = 1/(x^n) การต่อยอดสู่ระบบรากที่ n (nth Root) ของจำนวนจริง จะอาศัยหลักการทางพีชคณิตที่ว่า y เป็นรากที่ n ของ x ก็ต่อเมื่อ y^n = x ในการพิจารณาจำนวนรากจริง เราพิจารณาค่า n เป็นหลัก: ถ้า n เป็นจำนวนคู่บวก และ x > 0 รากจริงจะมี 2 ค่า (ค่าบวกและค่าลบ) แต่ถ้า x < 0 จะไม่มีรากในระบบจำนวนจริง ในขณะที่ถ้า n เป็นจำนวนคี่บวก จะมีรากจริงเพียงค่าเดียวเสมอไม่ว่า x จะเป็นบวกหรือลบ

ค่าหลักของรากที่ n (Principal nth Root): สัญลักษณ์กรณฑ์ (Radical Sign) แทนค่าหลักของรากที่ n ของ x เขียนแทนด้วย n√x โดยมีข้อกำหนดสำคัญของค่าหลักคือ ต้องมีเครื่องหมายเหมือนกับ x เสมอ (หรือ x และค่าหลักคูณกันต้องมากกว่าหรือเท่ากับ 0) และสมบัติสำคัญที่ต้องระวังในการลดรูปคือ n√(a^n) = a เมื่อ n เป็นจำนวนคี่บวก แต่จะเท่ากับ |a| (ค่าสัมบูรณ์ของ a) เมื่อ n เป็นจำนวนคู่บวก ซึ่งนี่คือจุดหลอกที่ข้อสอบระดับปานกลางถึงยากนิยมนำมาใช้ทดสอบความรอบคอบของผู้เรียน

Memory Tricks (เทคนิคช่วยจำขั้นเทพ):

  • "คู่ถอดครอบคัท" - ถ้ารากอันดับคู่ ถอดกำลังคู่ ต้องใส่เครื่องหมายค่าสัมบูรณ์ (Absolute Value) เสมอ ป้องกันผลลัพธ์ติดลบ
  • "คี่ถอดปล่อยผ่าน" - ถ้ารากอันดับคี่ ถอดออกมาได้ค่าเดิมเสมอ ไม่ว่าจะเป็นบวกหรือลบ
  • "ฐานเหมือนคูณกัน เลขชี้บวกกัน ฐานเหมือนหารกัน เลขชี้ลบกัน" - กฎเหล็กของเลขยกกำลังที่ใช้ได้ทุกระบบจำนวน

Thought Process (กระบวนการคิดอย่างเป็นระบบ): เมื่อเผชิญหน้ากับโจทย์ระดับปานกลางถึงยาก ให้ดำเนินการตามขั้นตอนดังนี้ 1. วิเคราะห์ฐาน: ทำฐานของเลขยกกำลังหรือจำนวนใต้กรณฑ์ให้เป็นจำนวนเฉพาะ (Prime Factorization) เพื่อลดรูปให้ง่ายที่สุด 2. ตรวจสอบอันดับราก: พิจารณาว่าอันดับของรากเป็นเลขคู่หรือเลขคี่เพื่อกำหนดเงื่อนไขของคำตอบและระวังเงื่อนไขห้ามติดลบใต้กรณฑ์อันดับคู่ 3. จัดรูปด้วยสมบัติ: ใช้สมบัติการกระจายเลขชี้กำลัง (a×b)^n = a^n × b^n และ (a^m)^n = a^(m×n) ในการเคลียร์วงเล็บจากในออกนอก 4. แก้สมการ: สำหรับสมการเลขยกกำลัง ทำฐานให้เท่ากันแล้วจับเลขชี้กำลังมาเท่ากัน (ถ้า a^x = a^y แล้ว x = y) หรือกรณีฐานไม่เท่าแต่เลขชี้กำลังเท่ากัน ให้พิจารณาเงื่อนไขที่ทำให้สมการเป็นจริง

(2) ข้อสอบระดับปานกลาง-ยาก Interactive (50 ข้อ)

คำชี้แจง: ให้คลิกเลือกคำตอบที่ถูกต้องที่สุด ระบบจะทำการตรวจและแสดงคำอธิบายทันทีเมื่อคลิกเลือก

ข้อ 1: ค่าของ (2⁵ × 3⁻² × 6⁴) ÷ (2³ × 3² × 12⁻¹) เท่ากับเท่าใด (Clue: เปลี่ยนทุกฐานให้เป็นฐาน 2 และฐาน 3 ก่อนนำเลขชี้กำลังมาดำเนินการกัน)
ข้อ 2: กำหนดให้ a, b, c เป็นจำนวนจริงที่ไม่เท่ากับศูนย์ รูปอย่างง่ายของ [ (a² b⁻³ c⁴) ÷ (a⁻¹ b² c⁻²) ]⁻² ตรงกับข้อใด (Clue: ยุบพจน์ในวงเล็บให้เรียบร้อยแล้วจึงกระจายเลขชี้กำลัง -2 เข้าไปคูณ)
ข้อ 3: ข้อความใดต่อไปนี้ "ถูกต้อง" เกี่ยวกับรากที่ n ของจำนวนจริง (Clue: ทบทวนนิยามของรากอันดับคู่และรากอันดับคี่ รวมถึงความแตกต่างระหว่างรากที่ n กับค่าหลัก)
ข้อ 4: ค่าของ ³√(-125) + ⁴√(-3)⁴ - ⁵√(-32) มีค่าเท่ากับข้อใด (Clue: พิจารณาอันดับรากให้ดี รากอันดับคู่ถอดออกมาห้ามติดลบ รากอันดับคี่ถอดตามเครื่องหมายฐานได้เลย)
ข้อ 5: รูปอย่างง่ายของ √50 + √18 - √128 ตรงกับข้อใด (Clue: ดึงตัวประกอบที่ถอดรากได้ออกมาข้างนอกกรณฑ์ให้หมด แล้วจึงนำสัมประสิทธิ์มาบวกลบกัน)
ข้อ 6: ผลคูณของ ³√3 × √2 มีค่าเท่ากับกรณฑ์ในข้อใด (Clue: การคูณกรณฑ์ที่มีอันดับไม่เท่ากัน ต้องทำอันดับของกรณฑ์ให้เท่ากันก่อนโดยหา ค.ร.น. ของอันดับราก)
ข้อ 7: ทำเศษส่วน 5 ÷ (√7 - √2) ให้ตัวส่วนไม่ติดกรณฑ์ ได้ผลลัพธ์ตรงกับข้อใด (Clue: คูณทั้งเศษและส่วนด้วย สังยุค (Conjugate) ของตัวส่วนเพื่อใช้สูตรผลต่างกำลังสอง)
ข้อ 8: ค่าของ (27)^(2/3) + (16)^(3/4) - (32)^(-1/5) เท่ากับเท่าใด (Clue: แปลงฐานเป็นเลขยกกำลังกำลังต่ำสุด แล้วตัดทอนกับเลขชี้กำลังที่เป็นเศษส่วน)
ข้อ 9: จงหาเซตคำตอบของสมการ 3^(x² - 3x) = 81 (Clue: ทำฐานสองข้างให้เท่ากันคือฐาน 3 แล้วจับเลขชี้กำลังมาตั้งเป็นสมการกำลังสองเพื่อหาค่า x)
ข้อ 10: สมการ 2^(2x + 1) - 9(2^x) + 4 = 0 มีผลบวกของคำตอบทั้งหมดเท่ากับข้อใด (Clue: ใช้การเปลี่ยนตัวแปร สมมติให้ A = 2^x แล้วแยกตัวประกอบสามพจน์)
ข้อ 11: จำนวน (2^200) × (5^195) เมื่อเขียนในรูปสัญกรณ์วิทยาศาสตร์ (A × 10^n) จะมีค่า A ตรงกับข้อใด (Clue: จัดกลุ่มฐาน 2 และฐาน 5 ที่มีเลขชี้กำลังเท่ากันให้ยุบเป็นฐาน 10)
ข้อ 12: ถ้า x เป็นจำนวนจริงลบ ข้อใดคือค่าของ ⁶√x⁶ - ∛x³ (Clue: ตระหนักถึงกฎเครื่องหมายรากคู่และรากคี่ ⁶√x⁶ = |x| ส่วน ∛x³ = x)
ข้อ 13: ค่าของ (√3 + √2)⁶ × (√3 - √2)⁵ มีค่าเท่ากับข้อใด (Clue: แยกตัวกำลัง 6 ออกเป็นกำลัง 5 คูณกับกำลัง 1 เพื่อจับคู่สังยุคคูณกันก่อน)
ข้อ 14: รากที่สองของ 7 + 4√3 คือข้อใดในระบบจำนวนจริง (Clue: จัดรูปให้อยู่ในฟอร์ม √(a+b + 2√ab) เพื่อยุบเป็นกำลังสองสมบูรณ์)
ข้อ 15: กำหนด x^(1/2) + x^(-1/2) = 3 จงหาค่าของ x + x⁻¹ (Clue: ยกกำลังสองทั้งสองข้างของสมการที่กำหนด แล้วย้ายพจน์กลางหน้าหลังตัดกัน)
ข้อ 16: ถ้า (a⁻² b) ÷ (a³ b⁻¹) = a^x b^y แล้ว x - y มีค่าเท่าใด (Clue: นำเลขชี้กำลังฐานเดียวกันมาลบกันตามกฎการหาร แล้วหาค่า x และ y)
ข้อ 17: กำหนดให้ a เป็นจำนวนจริงที่ทำให้ ∛a < √a ข้อใดต่อไปนี้เป็นช่วงที่เป็นไปได้ของ a (Clue: พิจารณาค่าในระบบจำนวนจริงที่รากอันดับสามน้อยกว่ารากที่สอง สังเกตพฤติกรรมช่วง 0 ถึง 1 และมากกว่า 1)
ข้อ 18: รูปอย่างง่ายของ (2√3 - 3√2) ÷ (√6) ตรงกับข้อใด (Clue: กระจายตัวส่วน √6 ไปหารทีละพจน์แล้วตัดทอนเศษส่วนกรณฑ์)
ข้อ 19: จงหาผลเฉลยของสมการ √(2x + 5) = x + 1 (Clue: ยกกำลังสองสองข้าง ย้ายพจน์แยกตัวประกอบ แต่อย่าลืมตรวจคำตอบเงื่อนไขห้ามติดลบในรูท)
ข้อ 20: ค่าของ [ (3^(n+4) - 9 × 3^n) ] ÷ (3^(n+2)) มีค่าเท่ากับข้อใด (Clue: แยกตัวร่วม 3^n ออกมาทั้งตัวเศษและตัวส่วนเพื่อตัดทอนกันให้หมดสิ้น)
ข้อ 21: กำหนดให้ A = ∛9, B = ∜15 และ C = ⁶√80 ข้อใดเป็นการเรียงลำดับจากน้อยไปมากที่ถูกต้อง (Clue: ทำอันดับรากให้เป็น 12 เท่ากันทั้งหมดโดยใช้ ค.ร.น. แล้วเปรียบเทียบค่าตัวเลขภายใน)
ข้อ 22: ค่าของ (1 ÷ (2 + √3)) + (1 ÷ (2 - √3)) มีค่าตรงกับข้อใด (Clue: หา ห.ร.ม. ของส่วนโดยการคูณไขว้รวมเศษส่วน ซึ่งส่วนจะกลายเป็นผลต่างกำลังสองอัตโนมัติ)
ข้อ 23: ถ้า 8^(x-1) = 4^(x+2) แล้ว ค่าของ x² มีค่าเท่าใด (Clue: เปลี่ยนฐานให้เป็น 2 ทั้งสองข้าง ปลดเลขชี้กำลังมาแก้สมการหา x แล้วนำไปยกกำลังสอง)
ข้อ 24: ผลลัพธ์ของ (a^x ÷ a^y)^(x+y) × (a^y ÷ a^z)^(y+z) × (a^z ÷ a^x)^(z+x) ตรงกับข้อใด (Clue: นำเลขชี้กำลังลบกันในวงเล็บ แล้วคูณกระจายเข้าสูตรผลต่างกำลังสอง พจน์ทั้งหมดจะตัดกันหมด)
ข้อ 25: ในการฝากเงินแบบดอกเบี้ยทบต้นทุกปีตามสูตร A = P(1+r)^t ถ้าฝากเงินต้น 10,000 บาท ผ่านไป 2 ปี ได้เงินรวม 12,100 บาท อยากทราบว่าอัตราดอกเบี้ย r ต่อปีเท่ากับเท่าใด (Clue: แทนค่าเพื่อหา (1+r)² ถอดรากที่สองของสัดส่วนเงินแล้วย้ายข้างหา r)
ข้อ 26: ค่าของ [ (4⁻³ × 6⁻² × 3⁴) ÷ (2⁻⁸ × 9⁻¹) ]⁻¹ มีค่าเท่ากับข้อใด (Clue: จัดฐานทุกตัวให้เป็นฐาน 2 และ 3 ยุบเลขชี้กำลังด้านในให้เสร็จสิ้นก่อนกระจายกำลัง -1)
ข้อ 27: ถ้า a และ b เป็นจำนวนจริงที่ไม่เท่ากับศูนย์ รูปอย่างง่ายของ [ (a⁻¹ + b⁻¹) ÷ (a⁻¹ - b⁻¹) ]⁻¹ ตรงกับข้อใด (Clue: เปลี่ยน x⁻¹ เป็น 1/x แล้วทำส่วนของเศษส่วนซ้อนให้เท่ากันก่อนตัดทอน)
ข้อ 28: ค่าของ (2^(n+3) - 6 × 2^(n-1)) ÷ (5 × 2^(n+1)) มีค่าเท่ากับข้อใด (Clue: ดึงตัวร่วม 2^n หรือ 2^(n-1) ออกมาทั้งเศษและส่วนเพื่อตัดทอนตัวแปร n ออกไป)
ข้อ 29: ผลลัพธ์ของ (125⁴ × 4⁶) ÷ (10¹⁰) เมื่อเขียนให้อยู่ในรูปสัญกรณ์วิทยาศาสตร์ A × 10^n จะมีค่า n เท่ากับข้อใด (Clue: เปลี่ยน 125 เป็น 5³ และ 4 เป็น 2² จากนั้นจับคู่ฐาน 2 และ 5 เพื่อสร้างฐาน 10)
ข้อ 30: กำหนดให้ x = 2¹⁰⁰, y = 3⁷⁵ และ z = 5⁵⁰ ข้อใดต่อไปนี้เรียงลำดับจากน้อยไปมากได้ถูกต้อง (Clue: หา ห.ร.ม. ของเลขชี้กำลังทั้งหมด ซึ่งคือ 25 แล้วจัดรูปให้อยู่ในกำลัง 25 เท่ากัน)
ข้อ 31: ถ้า (x⁻² y³ z⁻¹) ÷ (x y⁻² z²) = x^a y^b z^c แล้ว ค่าของ a + b + c ตรงกับข้อใด (Clue: นำเลขชี้กำลังฐานเดียวกันมาลบกันในรูปการหาร (บนลบล่าง) แล้วหาผลรวมสัมประสิทธิ์)
ข้อ 32: พิจารณาข้อความต่อไปนี้: (1) รากที่ 6 ของ -64 ในระบบจำนวนจริงไม่มี (2) ค่าหลักของรากที่ 5 ของ -32 คือ -2 ข้อใดสรุปถูกต้อง (Clue: รากอันดับคู่ของจำนวนลบไม่เป็นจำนวนจริง ส่วนรากอันดับคี่สามารถมีค่าหลักติดลบได้ตามฐาน)
ข้อ 33: ถ้า x เป็นจำนวนจริงใดๆ ค่าของ ⁴√(x - 3)⁴ เท่ากับข้อใดเสมอ (Clue: นิยามของรากอันดับคู่ n√(a^n) = |a| เมื่อ n เป็นจำนวนคู่บวก เพื่อรับประกันผลลัพธ์ไม่ติดลบ)
ข้อ 34: ค่าของ ∛(-216) + ⁶√(-2)⁶ - ⁵√(2⁴ × 2) มีค่าตรงกับข้อใด (Clue: ถอดรากทีละพจน์อย่างรอบคอบ ระวังพจน์รากคู่กำลังคู่ที่ต้องถอดออกมาเป็นค่าบวกเสมอ)
ข้อ 35: ถ้า x < 0 และ y > 0 แล้ว ข้อใดคือรูปอย่างง่ายของ √(x² y⁴) + ∛(x³ y⁶) (Clue: ถอดรากที่สองต้องใส่ absolute ครอบ x ส่วนรากที่สามถอดตามเครื่องหมายเดิมได้เลย)
ข้อ 36: รากที่สองของจำนวนในข้อใด ไม่เป็นจำนวนจริง (Clue: จำนวนที่จะไม่มีรากที่สองในระบบจำนวนจริง คือจำนวนที่มีค่าสุทธิติดลบ)
ข้อ 37: กำหนดให้ ⁸√(x - 2) จะนิยามได้ในระบบจำนวนจริงก็ต่อเมื่อ x มีค่าตรงกับข้อใด (Clue: จำนวนใต้เครื่องหมายกรณฑ์อันดับคู่ จะต้องมีค่ามากกว่าหรือเท่ากับศูนย์เสมอ)
ข้อ 38: ค่าของ 3√20 + 2√45 - √125 มีค่าตรงกับข้อใด (Clue: ถอดตัวประกอบออกจากทุกกรณฑ์ให้อยู่ในรูปตัวเลขคูณกับ √5 แล้วนำสัมประสิทธิ์มาบวกลบกัน)
ข้อ 39: ผลสำเร็จของ √2 × ∛2 มีค่าเท่ากับกรณฑ์อันดับที่ 6 ในข้อใด (Clue: ทำอันดับรากให้เท่ากันเป็น 6 โดยใช้ ค.ร.น. จะได้ฐานในรูทเปลี่ยนไปตามสัดส่วนการขยายอันดับราก)
ข้อ 40: ถ้าทำการคอนจุเกต (Conjugate) เศษส่วน 2 ÷ (√5 + √3) ผลลัพธ์สุดท้ายจะตรงกับข้อใด (Clue: คูณเศษและส่วนด้วย (√5 - √3) ส่วนข้างล่างจะเข้าสูตรผลต่างกำลังสองได้ค่าเป็น 2 พอดี)
ข้อ 41: ค่าของ (2√5 - √2)² มีค่าตรงกับข้อใด (Clue: กระจายพหุนามตามสูตรกำลังสองสมบูรณ์ (หน้า)² - 2(หน้า)(หลัง) + (หลัง)²)
ข้อ 42: รากที่สองของ 11 - 2√30 ในระบบจำนวนจริงคือข้อใด (Clue: ใช้สูตรถอดรากซ้อนราก √(a+b - 2√ab) = √a - √b โดยที่ a ต้องมากกว่า b)
ข้อ 43: ค่าของ (√2 + 1)⁴ × (√2 - 1)⁴ มีค่าเท่ากับเท่าใด (Clue: ยุบเลขชี้กำลังเป็นกำลัง 4 ร่วมกัน แล้วเอาฐานที่เป็นคอนจุเกตคูณกันข้างในก่อน)
ข้อ 44: ค่าของ (8)^(2/3) - (81)^(1/4) + (25)^(-1/2) เท่ากับข้อใด (Clue: เปลี่ยนฐานทุกตัวเป็นฐานเลขยกกำลังต่ำสุดเพื่อทอนเศษส่วนเลขชี้กำลัง)
ข้อ 45: จงหาเซตคำตอบของสมาร 2^(x² - x) = 64 (Clue: เปลี่ยน 64 เป็น 2⁶ ปลดฐานทิ้งแล้วตั้งสมการกำลังสอง x² - x - 6 = 0 เพื่อหาค่า x)
ข้อ 46: เซตคำตอบของสมการ 5^(x+1) + 5^x = 150 คือข้อใด (Clue: แยกสลายพจน์แรกเป็น 5^x × 5 แล้วดึงตัวร่วม 5^x ออกมาคำนวณร่วมกับพจน์หลัง)
ข้อ 47: สมการ 3^(2x) - 10(3^x) + 9 = 0 มีผลรวมของรากสมการทั้งหมดเท่ากับข้อใด (Clue: สมมติให้ A = 3^x จะได้สมการกำลังสอง A² - 10A + 9 = 0 หาคำตอบ A แล้วแปลงกลับเป็นค่า x)
ข้อ 48: จงหาคำตอบของสมการ √(3x + 1) = x - 1 (Clue: ยกกำลังสองสองข้าง แก้สมการพหุนาม แล้วต้อง 'ตรวจคำตอบ' เสมอเพื่อตัดคำตอบที่ขัดกับเงื่อนไขกรณฑ์)
ข้อ 49: กำหนดให้ x^(1/3) + x^(-1/3) = 4 ค่าของ x + x⁻¹ มีค่าตรงกับข้อใด (Clue: ใช้สูตรกำลังสามสมบูรณ์ ยกกำลังสามทั้งสองข้าง พจน์กลางจะตัดกันเหลือโครงสร้างที่ดึงตัวร่วมได้)
ข้อ 50: แบคทีเรียกลุ่มหนึ่งมีการเพิ่มจำนวนตามสูตร N = N₀(2)^(3t) โดย N₀ คือจำนวนเริ่มต้น และ t คือเวลา (ชั่วโมง) ถ้าเริ่มต้นมีแบคทีเรีย 500 ตัว ผ่านไปกี่ชั่วโมงจึงจะมีแบคทีเรีย 32,000 ตัว (Clue: แทนค่า N และ N₀ ย้าย 500 ไปหาร แล้วจัดรูปให้อยู่ในฐาน 2 เพื่อแก้หาค่า t)
