ตะลุยโจทย์ ความสัมพันธ์และฟังก์ชัน (25 ข้อ)

พระจอมเกล้าติวเตอร์ Talk: ข้อสอบชุดนี้เป็นระดับปานกลางถึงยาก เน้นการคิดเชิงระบบและการวิเคราะห์เงื่อนไขทางคณิตศาสตร์อย่างรอบคอบ อย่าเพิ่งรีบตอบ ค่อยๆ คิดตาม Clue สีแดงนะครับ!
ข้อ 1: กำหนด A = {1, 2, 3} และ B = {x ∈ Z | x² - 5x + 6 = 0} จำนวนสมาชิกของ A × B เท่ากับข้อใด (Clue: หาเซต B จากการแก้สมการกำลังสองก่อน แล้วหาจำนวนสมาชิก)

ข้อ 2: กำหนดความสัมพันธ์ r = {(x, y) ∈ R × R | y = 5 / (x² - 4)} โดเมนของ r ตรงกับข้อใด (Clue: รูปเศษส่วน ตัวส่วนต้องห้ามเป็นศูนย์เด็ดขาด!)
ข้อ 3: กำหนด r = {(x, y) ∈ R × R | y = √(16 - x²)} เรนจ์ของ r ตรงกับข้อใด (Clue: พิจารณาค่าต่ำสุดและสูงสุดภายใต้เครื่องหมายรากที่สอง และสมบัติของรากหลักที่สอง)
ข้อ 4: ความสัมพันธ์ในข้อใดต่อไปนี้ "ไม่เป็นฟังก์ชัน" (Clue: มองหาข้อที่ค่า x หนึ่งค่าสามารถจับคู่กับ y ได้มากกว่าหนึ่งค่า หรือสังเกตกำลังของ y)
ข้อ 5: ถ้า f(x) = 3x - 1 และ g(x) = x² + 2 แล้วค่าของ f(g(2)) เท่ากับข้อใด (Clue: คิดจากฟังก์ชันด้านในก่อน หา g(2) แล้วนำผลลัพธ์ไปแทนใน f(x))
ข้อ 6: โดเมนของความสัมพันธ์ r = {(x, y) | y = 1 / √(x - 3)} คือข้อใด (Clue: มีสองเงื่อนไขซ้อนกัน คือ อยู่ในรูทต้อง ≥ 0 และเป็นตัวส่วนต้อง ≠ 0)
ข้อ 7: เรนจ์ของฟังก์ชัน f(x) = x² - 6x + 11 คือข้อใด (Clue: จัดให้อยู่ในรูปกำลังสองสมบูรณ์เพื่อหาจุดต่ำสุดของพาราโบลาหงาย)
ข้อ 8: จุดวกกลับของกราฟฟังก์ชัน y = -2x² + 8x - 5 คือจุดใด (Clue: ใช้สูตร x = -b / 2a เพื่อหาพิกัด x ของจุดยอดก่อน)
ข้อ 9: เรนจ์ของความสัมพันธ์ r = {(x, y) | y = 4 / (x² + 1)} ตรงกับข้อใด (Clue: พิจารณาค่าที่เป็นไปได้ของ x² + 1 ซึ่งมีค่าเริ่มต้นที่ 1 แล้ววิเคราะห์เศษส่วน)
ข้อ 10: เซตของคู่อันดับในข้อใดต่อไปนี้เป็นความสัมพันธ์ที่มีโดเมนเท่ากับเรนจ์ (Clue: หาเซตของสมาชิกตัวหน้าและตัวหลังของแต่ละข้อแล้วนำมาเปรียบเทียบกัน)
ข้อ 11: จำนวนความสัมพันธ์ทั้งหมดจากเซต A = {a, b} ไปยังเซต B = {1, 2, 3} เท่ากับข้อใด (Clue: จำนวนความสัมพันธ์หาได้จากสูตร 2^(n(A × B)))
ข้อ 12: โดเมนของฟังก์ชัน f(x) = √(x² - 9) ตรงกับข้อใด (Clue: เงื่อนไขใต้รากที่สองต้องมากกว่าหรือเท่ากับ 0 แยกตัวประกอบอสมการกำลังสอง)
ข้อ 13: ฟังก์ชันในข้อใดเป็น "ฟังก์ชันลด" บนโดเมนของฟังก์ชัน (Clue: ฟังก์ชันเอกซ์โพเนนเชียล y = a^x จะเป็นฟังก์ชันลดเมื่อ 0 < a < 1)
ข้อ 14: เรนจ์ของความสัมพันธ์ r = {(x, y) | y = |x - 2| + 5} ตรงกับข้อใด (Clue: สมบัติของค่าสัมบูรณ์ต่ำสุดคือ 0 นำไปคิดต่อยอดความสูงขั้นต่ำ)
ข้อ 15: พิจารณากราฟความสัมพันธ์รูปวงกลม x² + y² = 16 ข้อใดกล่าวถูกต้องที่สุด (Clue: ใช้ความรู้เรื่องการตรวจสอบฟังก์ชันและการหาโดเมนเรนจ์ของรูปวงกลม)
ข้อ 16: กำหนด f(x) = ax + b ถ้า f(1) = 5 และ f(3) = 11 แล้วค่าของ a + b เท่ากับข้อใด (Clue: ตั้งระบบสมการสองตัวแปรเพื่อหาค่าคงที่ a และ b ก่อน)
ข้อ 17: ฟังก์ชันขั้นบันไดมักนำมาใช้ประโยชน์ในเรื่องใดในชีวิตประจำวันมากที่สุด (Clue: สังเกตลักษณะค่าบริการที่มีการคิดราคาคงที่ตามช่วงของข้อมูล เช่น น้ำหนัก หรือเวลา)
ข้อ 18: ค่าสูงสุดของฟังก์ชันกำลังสอง f(x) = -x² + 4x + 7 มีค่าตรงกับข้อใด (Clue: สังเกตหน้า x² ติดลบ ได้พาราโบลาคว่ำ ให้ค่าสูงสุด หาจุดยอดพิกัด y)
ข้อ 19: กำหนดความสัมพันธ์ r = {(x, y) ∈ R × R | y = (3x + 1) / (2x - 5)} เรนจ์ของความสัมพันธ์นี้คือข้อใด (Clue: หาเรนจ์โดยการจัดรูปสมการให้ x อยู่ในเทอมของ y แล้วพิจารณาตัวส่วนของ y)
ข้อ 20: ข้อใดเป็นกราฟของฟังก์ชันเอกซ์โพเนนเชียลที่ "ตัดแกน Y ที่จุดเดียวกันเสมอ" หากไม่มีการเลื่อนขนานกราฟ (Clue: ทบทวนสมบัติพื้นฐานว่าเมื่อ x = 0 ค่า y ของฟังก์ชันในรูปมาตรฐานจะมีค่าเท่าใด)
ข้อ 21: กำหนดให้ A = {1, 2} และ B = {a, b} คู่อันดับในข้อใดต่อไปนี้ "ไม่เป็นสมาชิก" ของผลคูณคาร์ทีเซียน A × B (Clue: สมาชิกตัวหน้าต้องมาจากเซต A และสมาชิกตัวหลังต้องมาจากเซต B เท่านั้น)
ข้อ 22: โดเมนของฟังก์ชัน f(x) = 1 / (x² - x - 6) ตรงกับข้อใด (Clue: ตัวส่วนต้องไม่เท่ากับศูนย์ แยกตัวประกอบพหุนามกำลังสองเพื่อหาค่าห้ามเป็น)
ข้อ 23: สมการแกนสมมาตรของพาราโบลา y = 3x² - 12x + 5 คือข้อใด (Clue: แกนสมมาตรของพาราโบลาคือเส้นตรงแนวดิ่งที่ลากผ่านจุดยอด x = -b / 2a)
ข้อ 24: โรงจอดรถแห่งหนึ่งคิดค่าจอดรถดังนี้ 2 ชั่วโมงแรกจอดฟรี ชั่วโมงต่อไปคิดชั่วโมงละ 20 บาท (เศษของชั่วโมงคิดเป็น 1 ชั่วโมง) รูปแบบนี้ตรงกับฟังก์ชันชนิดใด (Clue: อัตราค่าบริการที่ปรับเปลี่ยนแบบก้าวกระโดดเมื่อเวลาผ่านไปเป็นช่วงๆ)
ข้อ 25: โดเมนของฟังก์ชัน f(x) = 1 / √(4 - x²) คือข้อใด (Clue: รวมสองเงื่อนไขคือ ใต้รูทต้องมากกว่าหรือเท่ากับ 0 และส่วนต้องไม่เป็น 0)
ข้อ 26: กำหนด A = {x ∈ Z | -2 ≤ x < 3} และ B = {0, 1, 4} จำนวนสมาชิกของ B × A เท่ากับข้อใด (Clue: นับจำนวนสมาชิกของเซต A ที่เป็นจำนวนเต็มอย่างรอบคอบ แล้วใช้สูตรคูณคาร์ทีเซียน)
ข้อ 27: กำหนดความสัมพันธ์ r = {(x, y) ∈ R × R | y = 7 / √(9 - x²)} โดเมนของ r ตรงกับข้อใด (Clue: อยู่ในรูทและเป็นตัวส่วนพร้อมกัน เงื่อนไขใต้รูทต้องมากกว่าศูนย์อย่างเดียว)
ข้อ 28: เรนจ์ของความสัมพันธ์ r = {(x, y) ∈ R × R | y = 2 - |x + 3|} ตรงกับข้อใด (Clue: ค่าสัมบูรณ์มีค่าต่ำสุดเป็น 0 เมื่อโดนเครื่องหมายลบด้านหน้าพิกัดจะกลับทิศทาง)
ข้อ 29: ความสัมพันธ์ใดต่อไปนี้ "เป็นฟังก์ชัน" (Clue: สังเกตเงื่อนไขที่ y มีกำลังเป็นหนึ่ง และไม่มีเครื่องหมายค่าสัมบูรณ์ครอบตัว y)
ข้อ 30: กำหนด f(x) = 2x + 3 และ g(x) = 4x - 1 ค่าของ g(f(-1)) ตรงกับข้อใด (Clue: คิดในออกนอก หา f(-1) ก่อนแล้วเอาผลลัพธ์ไปใส่ใน g)
ข้อ 31: โดเมนของความสัมพันธ์ r = {(x, y) | y = (2x + 1) / (x² - 5x + 6)} ตรงกับข้อใด (Clue: ตัวส่วนห้ามเป็นศูนย์ แยกตัวประกอบพหุนามของส่วนเพื่อหาค่าวิกฤตที่ต้องตัดออก)
ข้อ 32: เรนจ์ของฟังก์ชัน f(x) = -x² + 8x - 12 คือข้อใด (Clue: หน้า x² เป็นลบเป็นพาราโบลาคว่ำ ให้ค่าสูงสุด หาจุดยอด k ด้วยสูตรลัดหรือจัดกำลังสองสมบูรณ์)
ข้อ 33: จุดยอดของพาราโบลา y = 2(x - 3)² + 5 คือจุดใด (Clue: เทียบกับสมการรูปมาตรฐาน y = a(x - h)² + k สามารถอ่านจุด (h, k) ได้ทันที)
ข้อ 34: เรนจ์ของความสัมพันธ์ r = {(x, y) | y = 6 / (x² + 3)} ตรงกับข้อใด (Clue: หาค่าต่ำสุดของตัวหาร x² + 3 นำไปวิเคราะห์หาค่าสูงสุดและขอบล่างของเศษส่วน)
ข้อ 35: กำหนด r = {(1, 2), (2, b), (a, 4)} ถ้าโดเมนของความสัมพันธ์นี้คือ {1, 2, 3} และเรนจ์คือ {2, 4, 5} แล้วค่าของ a + b เท่ากับข้อใด (Clue: จับคู่สมาชิกโดเมน (ตัวหน้า) และเรนจ์ (ตัวหลัง) ที่ยังขาดหายไปเพื่อหาค่า a และ b)
ข้อ 36: จำนวนความสัมพันธ์ทั้งหมดจากเซต A ไป A เมื่อเซต A = {1, 2, 3, 4} เท่ากับข้อใด (Clue: ความสัมพันธ์จาก A ไป A สมาชิกผลคูณคาร์ทีเซียนคือ n(A) × n(A) แล้วเข้าสูตรสับเซต)
ข้อ 37: โดเมนของฟังก์ชัน f(x) = √(2x - 8) ตรงกับข้อใด (Clue: ตั้งอสมการให้พจน์ใต้รากที่สองมากกว่าหรือเท่ากับศูนย์ แล้วแก้หาค่า x)
ข้อ 38: ฟังก์ชันในข้อใดต่อไปนี้เป็น "ฟังก์ชันเพิ่ม" บนเซตของจำนวนจริง (Clue: สังเกตฟังก์ชันเชิงเส้นที่มีความชัน (สปส. หน้า x) เป็นบวก หรือฟังก์ชันเอกซ์โพเนนเชียลที่มีฐานมากกว่า 1)
ข้อ 39: เรนจ์ของความสัมพันธ์ r = {(x, y) | y = |x + 4| - 3} ตรงกับข้อใด (Clue: นิยามของค่าสัมบูรณ์ต่ำสุดคือศูนย์ แล้วสังเกตตัวเลขคงที่ที่พ่วงท้ายเพื่อขยับจุดยอด)
ข้อ 40: พิจารณาสมการพาราโบลาตะแคงขวา x = y² กราฟนี้เป็นฟังก์ชันหรือไม่ และเพราะเหตุใด (Clue: ลองแทนค่า x = 4 แล้วดูว่าได้ค่า y กี่ค่า หรือนึกภาพการลากเส้นทดสอบแนวดิ่ง)
ข้อ 41: กำหนดฟังก์ชันเชิงเส้น f(x) = mx + 3 ถ้า f(2) = 11 แล้วความชัน m ของกราฟนี้มีค่าเท่าใด (Clue: แทนค่า x = 2 และ f(x) = 11 ลงในสมการเพื่อแก้หาค่าตัวแปร m)
ข้อ 42: สถานการณ์การคำนวณภาษีเงินได้บุคคลธรรมดาของประเทศไทยที่มีอัตราภาษีเพิ่มขึ้นตามขั้นของเงินได้สุทธิ จัดเป็นโมเดลของฟังก์ชันชนิดใด (Clue: อัตราภาษีจะคงที่ในแต่ละช่วงเงินได้ เมื่อเงินได้ทะลุช่วงราคาจะปรับกระโดดเป็นขั้นๆ)
ข้อ 43: ค่าต่ำสุดของฟังก์ชันกำลังสอง f(x) = 2x² - 12x + 10 มีค่าตรงกับข้อใด (Clue: สังเกต สปส. หน้า x² เป็นบวก ได้พาราโบลาหงาย ให้ค่าต่ำสุด หาพิกัด y ของจุดยอด)
ข้อ 44: กำหนดความสัมพันธ์ r = {(x, y) ∈ R × R | y = (4x - 2) / (3x + 6)} เรนจ์ของความสัมพันธ์นี้ตรงกับข้อใด (Clue: ใช้สูตรลัดฟังก์ชันเศษส่วนดีกรี 1 เรนจ์คือ จำนวนจริงทั้งหมดเว้นตัวเลข สปส. หน้า x ตัวบนหารตัวล่าง)
ข้อ 45: กราฟของฟังก์ชันเอกซ์โพเนนเชียลมาตรฐาน y = a^x (เมื่อ a > 0, a ≠ 1) จะ "ไม่มีวัน" ตัดหรือข้ามแกนใดต่อไปนี้ (Clue: ไม่ว่า x จะติดลบมากขนาดไหน ค่า y จะลดลงเข้าใกล้ศูนย์มากๆ แต่ไม่มีวันเท่ากับศูนย์หรือติดลบ)
ข้อ 46: กำหนดให้ A = {3, 4} และ B = {5, 6} คู่อันดับในข้อใดต่อไปนี้เป็นสมาชิกของผลคูณคาร์ทีเซียน B × A (Clue: โจทย์ถามหา B × A นะครับ! ตัวหน้าต้องดึงมาจากเซต B และตัวหลังต้องมาจากเซต A)
ข้อ 47: โดเมนของความสัมพันธ์ r = {(x, y) | y = 1 / (x² - 9)} ตรงกับข้อใด (Clue: ส่วนห้ามเป็นศูนย์เด็ดขาด แยกตัวประกอบกำลังสองสมบูรณ์ผลต่างได้สองค่า)
ข้อ 48: สมการแกนสมมาตรของพาราโบลาคว่ำ y = -x² + 6x - 2 คือข้อใด (Clue: เส้นแบ่งครึ่งสมมาตรหาพิกัดแนวดิ่งจากสูตรลัดจุดยอดแกน X คือ x = -b / 2a)
