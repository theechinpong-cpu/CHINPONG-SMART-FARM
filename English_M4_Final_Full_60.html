📘 สรุปเนื้อหา Unit 4: Innovation & Environment (400+ คำ)

เนื้อหาบทนี้เน้นความก้าวหน้าทางเทคโนโลยีและผลกระทบต่อโลก นักเรียนต้องเข้าใจการสื่อสารเรื่องอนาคตและการคาดการณ์ (Predictions) อย่างเป็นระบบ

1. Future Perfect (will have + V.3): ใช้พูดถึงเหตุการณ์ที่จะ "เสร็จสมบูรณ์" ในช่วงเวลาหนึ่งของอนาคต เช่น By next year, I will have finished this course. สังเกตว่ามักมีคำว่า "By..." นำหน้าเวลาเสมอ

2. Conditional Sentences (If-Clauses): เน้นหนักที่ Type 2 (สมมติในปัจจุบัน/เป็นไปไม่ได้) และ Type 3 (เสียดายในอดีต) รวมถึงการใช้ "Unless" ที่แปลว่า If not

3. Relative Clauses: การใช้ Who, Whom, Whose (คน) / Which, That (สิ่งของ) และ Where (สถานที่) เพื่อขยายความให้ชัดเจน รวมถึงการลดรูปประโยคที่นักเรียน ม.4 มักพลาดบ่อย

4. Vocabulary: คำศัพท์กลุ่มรักษ์โลก เช่น Sustainability (ความยั่งยืน), Biodegradable (ย่อยสลายได้), และกลุ่มเทคโนโลยี เช่น Artificial Intelligence, Cyber-security, Innovation

💡 Memory Tricks & Thought Process:
- สูตรลับ Future Perfect: เห็นคำว่า "By" + เวลาอนาคต = กาข้อที่มี "will have + V.3" ทันที!
- สูตร If-Clause Type 2: If + V.2 คู่กับ would + V.inf (จำไว้ว่า: อดีตคู่กับวู้ด)
- วิธีเช็ค Whose: ถ้าหน้าช่องว่างเป็น "คน" หลังช่องว่างเป็น "นาม" (เช่น The girl ___ dog...) ให้เติม Whose ทันที
