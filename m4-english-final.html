🎯 Slang & Social Expressions

• What’s up?: ทักเพื่อนสนิท (Informal)

• Fancy running into you here!: บังเอิญเจอโดยไม่คาดฝัน

• Coincidence: "What a coincidence!"

• For ages: นานโคตรๆ / Before you know it: แป๊บเดียว

• I’ll bet: พนันได้เลย / (ประชด) ไม่ค่อยเชื่อ

• No way!: ไม่มีทาง / จริงดิ! (Surprised)

💼 Lifestyle & Work

• Trendy: ฮิต / Run of the mill: ธรรมดาๆ

• Tip: เงินพิเศษ หรือ คำแนะนำ

• On the floor: พนักงานขาย / On the job: ใช้แรงงาน

🏗️ Grammar: Used to & Try

  • Used to + V.inf: "เคยทำ" (อดีตทำ ปัจจุบันเลิก)
  • be/get used to + V.ing: "เคยชิน" (ปรับตัวแล้ว)
  • Try on: ลองชุด / Try out: ลองเครื่อง/รถ
  • Try to + V.inf: พยายาม / Try + V.ing: ลองทำเพื่อแก้ปัญหา

🏠 There is vs There are

• There is: + เอกพจน์ (1) / นามนับไม่ได้ (Water, Money)

• There are: + พหูพจน์ (2 ขึ้นไป)
