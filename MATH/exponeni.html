📝 แบบทดสอบเรื่อง เลขยกกำลัง (คณิตศาสตร์ ม.5)

โดย พระจอมเกล้าติวเตอร์

ข้อ 1: ข้อใดต่อไปนี้ลบพจน์รากได้ถูกต้องตามสมบัติคณิตศาสตร์ ของจำนวน \(\sqrt[3]{-64} + \sqrt{144} - \sqrt[4]{81}\)


คำอธิบาย:
- \(\sqrt[3]{-64} = -4\) (เนื่องจาก \((-4)^3 = -64\))
- \(\sqrt{144} = 12\) (ค่าหลักที่เป็นบวก)
- \(\sqrt[4]{81} = 3\) (เนื่องจาก \(3^4 = 81\))
ดังนั้น \((-4) + 12 - 3 = 5\) ตอบ ข้อ ก.

ข้อ 2: จัดรูปพจน์ \(\frac{5}{\sqrt{6}-\sqrt{1}}\) ให้อยู่ในรูปที่ตัวส่วนไม่ติดกรณฑ์ ได้ตรงกับข้อใด


คำอธิบาย:
ใช้เทคนิคการคูณด้วยสังยุค (Conjugate):
\(\frac{5}{\sqrt{6}-1} \times \frac{\sqrt{6}+1}{\sqrt{6}+1} = \frac{5(\sqrt{6}+1)}{6-1} = \frac{5(\sqrt{6}+1)}{5} = \sqrt{6}+1\) ตอบ ข้อ ข.

ข้อ 3: ค่าของ \((25)^{\frac{3}{2}} \times (8)^{-\frac{2}{3}}\) มีค่าเท่ากับเท่าใด


คำอธิบาย:
- \((25)^{\frac{3}{2}} = (5^2)^{\frac{3}{2}} = 5^3 = 125\)
- \((8)^{-\frac{2}{3}} = (2^3)^{-\frac{2}{3}} = 2^{-2} = \frac{1}{4}\)
นำมาคูณกัน: \(125 \times \frac{1}{4} = \frac{125}{4}\) ตอบ ข้อ ก.

ข้อ 4: ฝากเงินแบบคิดดอกเบี้ยทบต้นรายปี อัตราดอกเบี้ย 2% ต่อปี เป็นเวลา 3 ปี เงินต้น 10,000 บาท เมื่อครบกำหนดจะมีเงินรวมตรงกับสมการข้อใด (ตามสูตร \(S_n = P(1+r)^n\))


คำอธิบาย:
จากสูตรการเงิน \(S_n = P(1+r)^n\)
แทนค่า \(P = 10,000\), อัตราดอกเบี้ย \(r = 2\% = 0.02\), และระยะเวลา \(n = 3\)
จะได้ \(S_3 = 10,000(1 + 0.02)^3 = 10,000(1.02)^3\) ตอบ ข้อ ข.
