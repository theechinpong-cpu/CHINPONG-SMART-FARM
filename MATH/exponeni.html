พระจอมเกล้าติวเตอร์ คณิตศาสตร์ ม.5
เลขยกกำลังและรากที่ n

สรุปเนื้อหา: เลขยกกำลัง (Exponents)

รวบรวมครบทุกหัวข้อสำหรับสอบ — พร้อมทริคจำ

1. รากที่ n ในระบบจำนวนจริง และค่าหลัก (Principal Root)

b เป็นรากที่ n ของ a ก็ต่อเมื่อ bⁿ = a โดยที่ a, b เป็นจำนวนจริง และ n เป็นจำนวนเต็มที่มากกว่า 1

⚠️ n เป็นเลขคู่: ถ้า a > 0 มีราก 2 ค่า คือ +ⁿ√a และ −ⁿ√a / ถ้า a < 0 ไม่มีรากจริง (เป็นจำนวนจินตภาพ)
n เป็นเลขคี่: มีรากค่าเดียวเสมอ (ราก + เมื่อ a บวก / ราก − เมื่อ a ลบ)
ค่าหลัก ⁿ√a: มีเครื่องหมายเดียวกับ a เช่น ⁴√16 = 2 (ไม่ใช่ −2) และ ³√(−8) = −2
2. เลขยกกำลังที่มีเลขชี้กำลังเป็นจำนวนตรรกยะ

ถ้า p, q เป็นจำนวนเต็ม (p/q อย่างต่ำ) และ q > 1 โดยที่ ⁿ√a เป็นจำนวนจริง จะได้:

a^(1/q) = ⁿ√a      a^(p/q) = (ⁿ√a)ᵖ = ⁿ√(aᵖ)

ตัวอย่าง: 8^(2/3) = (³√8)² = 2² = 4

3. ทฤษฎีบทและสมบัติที่ต้องใช้สอบ

ให้ a, b เป็นจำนวนจริงบวก และ r, s เป็นจำนวนตรรกยะ:

  • aʳ · aˢ = aʳ⁺ˢ ฐานเหมือนกันคูณกัน → นำเลขชี้กำลังมาบวกกัน
  • aʳ ÷ aˢ = aʳ⁻ˢ ฐานเหมือนกันหารกัน → นำเลขชี้กำลังมาลบกัน
  • (aʳ)ˢ = aʳˢ เลขชี้กำลังซ้อนกัน → คูณกัน
  • (ab)ʳ = aʳ · bʳ   และ   (a/b)ʳ = aʳ/bʳ กระจายเลขชี้กำลังเข้าในผลคูณและผลหารได้
  • ⁿ√xⁿ = |x| เมื่อ n คู่    ⁿ√xⁿ = x เมื่อ n คี่ สมบัติสำคัญ! รากคู่ต้องใส่ค่าสัมบูรณ์เสมอ

💡 ทริคการจำจากพี่พระจอมเกล้าติวเตอร์

"รากคู่...ระวังคุก (ต้องใส่ Absolute!)" จำไว้เลย! เจอ √(x²) หรือ ⁴√(x⁴) ห้ามตอบ x เฉยๆ ต้องใส่ |x| เสมอ เพราะค่าหลักรากคู่ไม่มีทางติดลบ
"บนคือใน ล่างคือนอก" a^(p/q) → ⁿ√(aᵖ)   ตัวเศษ p อยู่บน = เลขยกกำลังที่อยู่ใน / ตัวส่วน q อยู่ล่าง = อันดับราก (นอก)
"คอนจูเกต (สังยุค) คู่แท้แก้รูทส่วน" เจอ 1/(A−√B) ให้คูณ (A+√B)/(A+√B) ทันที ใช้สูตร A²−B² สลัดรูทที่ตัวส่วนให้หายไปเลย

🧠 ส่วนที่ 2: กระบวนการคิดและ Step การมองโจทย์ (Thought Processes)

เวลาที่เจอกราฟิกหรือโจทย์เลขยกกำลังในห้องสอบ พี่อยากให้น้องๆ ท่อง 3 Step นี้ให้ขึ้นใจเพื่อสับสวิตช์สมองให้มองหาทางออกได้อย่างรวดเร็วครับ:

1 Step 1: ชำแหละฐานให้เล็กที่สุด (Prime Factorization)

เมื่อเจอโจทย์เลขยกกำลังที่มีตัวเลขเยอะๆ เช่น $8, 9, 16, 25, 27$ อย่าเพิ่งตกใจ ให้แปลงฐานเหล่านั้นให้อยู่ในรูปจำนวนเฉพาะก่อนเสมอ เช่น เปลี่ยน $27^x$ เป็น $(3^3)^x = 3^{3x}$ เพื่อเคลียร์ฐานให้หน้าตาเหมือนกันและตัดทอนหรือยุบรวมกันได้ง่ายขึ้น

2 Step 2: จัดระเบียบเครื่องหมายลบ

หากเลขชี้กำลังติดลบ ให้พลิกเศษเป็นส่วนตามกฎ $a^{-n} = \frac{1}{a^n}$ ก่อน เพื่อจัดรูปพจน์คณิตศาสตร์ให้ดูง่าย และป้องกันความสับสนในระหว่างการคำนวณบวกลบเลขชี้กำลัง

3 Step 3: เช็คเงื่อนไขโจทย์

ตรวจสอบเสมอว่าโจทย์กำหนดให้ตัวแปรเป็นจำนวนจริงบวกหรือไม่ หากโจทย์ไม่ได้กำหนด เมื่อต้องถอดรากเลขคู่ (เช่น รากที่สอง รากที่สี่) น้องๆ ต้องระวังพจน์ที่เป็นค่าสัมบูรณ์ (Absolute) ไว้ด้วยเสมอเพื่อไม่ให้หลงกลข้อสอบ

แบบฝึกหัด 1.2.1: รากที่ $n$ และการจัดรูปพจน์ติดกรณฑ์อย่างง่าย

จัดทำโดย พระจอมเกล้าติวเตอร์ | เนื้อหาคณิตศาสตร์ ม.5

ข้อ 1. จงหาค่าหลักของรากที่ $n$ ของจำนวนจริงต่อไปนี้ (มี 8 ข้อย่อย)
1.1) จงหาค่าหลักของรากที่สองของ $121$
$$\sqrt{121} = \sqrt{11^2}$$ (เปลี่ยน 121 ให้เป็นจำนวนเฉพาะยกกำลัง เพื่อเตรียมถอดรากที่สอง)
$= 11$ (ถอดรากที่สองออกได้สำเร็จ ค่าหลักของรากคู่ต้องเป็นบวกเสมอ) ตอบ 11
1.2) จงหาค่าหลักของรากที่สองของ $0$
$$\sqrt{0} = \sqrt{0^2}$$ (พิจารณาโครงสร้างเชิงนิยามของศูนย์)
$= 0$ (ค่าหลักของรากที่ $n$ ของ 0 มีค่าเท่ากับ 0 เสมอ) ตอบ 0
1.3) จงหาค่าหลักของรากที่สามของ $27$
$$\sqrt[3]{27} = \sqrt[3]{3^3}$$ (เปลี่ยนฐาน 27 เป็นเลขฐานจำนวนเฉพาะยกกำลังสาม)
$= 3$ (รากที่สามเจอเลขยกกำลังสาม จะถอดกรณฑ์ออกมาได้พอดี) ตอบ 3
1.4) จงหาค่าหลักของรากที่สามของ $-64$
$$\sqrt[3]{-64} = \sqrt[3]{(-4)^3}$$ (เปลี่ยน -64 เป็น (-4) ยกกำลังสาม รากคี่ข้างในติดลบได้ไม่มีปัญหา)
$= -4$ (ถอดกรณฑ์รากที่สามออกมา ได้ผลลัพธ์เป็นค่าลบตามเครื่องหมายของฐานเดิม) ตอบ -4
1.5) จงหาค่าหลักของรากที่สี่ของ $81$
$$\sqrt[4]{81} = \sqrt[4]{3^4}$$ (แปลง 81 ให้อยู่ในรูปของจำนวนเฉพาะยกกำลังสี่)
$= 3$ (ถอดรากอันดับที่สี่ได้ลงตัว และต้องได้คำตอบเป็นค่าบวกเท่านั้นเพราะเป็นรากคู่) ตอบ 3
1.6) จงหาค่าหลักของรากที่ห้าของ $-32$
$$\sqrt[5]{-32} = \sqrt[5]{(-2)^5}$$ (แปลง -32 เป็น (-2) ยกกำลังห้า เพื่อสอดคล้องกับอันดับรากที่เป็นจำนวนคี่)
$= -2$ (ถอดรากอันดับที่ห้าออกมา ได้ค่าลบตามเครื่องหมายของตัวเลขภายใน) ตอบ -2
1.7) จงหาค่าหลักของรากที่สองของ $x^2$ เมื่อ $x \ge 0$
$$\sqrt{x^2} = |x|$$ (ใช้สมบัติพื้นฐานของรากที่สองถอดออกมาได้ค่าสัมบูรณ์ของ x)
$= x$ (เนื่องจากโจทย์กำหนดเงื่อนไขระบุชัดเจนว่า $x \ge 0$ จึงปลด Absolute ได้ทันที) ตอบ x
1.8) จงหาค่าหลักของรากที่สามของ $x^3$ เมื่อ $x$ เป็นจำนวนจริงใดๆ
$$\sqrt[3]{x^3} = x$$ (เนื่องจากเป็นรากอันดับคี่ เครื่องหมายผลลัพธ์จะล้อตาม x เสมอ จึงสามารถถอดได้โดยไม่ต้องใส่ Absolute) ตอบ x
ข้อ 2. จงเขียนแสดงพจน์ต่อไปนี้ให้อยู่ในรูปอย่างง่าย (มี 8 ข้อย่อย)
2.1) จงจัดรูปอย่างง่ายของ $\sqrt{50}$
$$\sqrt{50} = \sqrt{25 \times 2} = \sqrt{5^2 \times 2}$$ (แยกตัวประกอบหาเลขยกกำลังสองที่ซ่อนอยู่ภายใน 50)
$= 5\sqrt{2}$ (ดึงเลข 5 ที่ยกกำลังสองอยู่ออกมานอกกรณฑ์รากที่สอง ส่วน 2 ไม่มีคู่ให้ติดไว้ข้างใน) ตอบ $5\sqrt{2}$
2.2) จงจัดรูปอย่างง่ายของ $\sqrt[3]{54}$
$$\sqrt[3]{54} = \sqrt[3]{27 \times 2} = \sqrt[3]{3^3 \times 2}$$ (เนื่องจากเป็นรากที่สาม จึงหาตัวประกอบที่ยกกำลังสาม ซึ่งก็คือ 27 = 3³)
$= 3\sqrt[3]{2}$ (ดึง 3 ออกมานอกรากที่สาม ส่วนเลข 2 ยังติดค้างอยู่ภายใต้รากอันดับที่สามเหมือนเดิม) ตอบ $3\sqrt[3]{2}$
2.3) จงจัดรูปอย่างง่ายของ $\sqrt{8} + \sqrt{18}$
$= \sqrt{4 \times 2} + \sqrt{9 \times 2}$ (แยกตัวประกอบของตัวเลขในแต่ละพจน์เพื่อหาพจน์กำลังสองสมบูรณ์)
$= 2\sqrt{2} + 3\sqrt{2}$ (ถอดรากแต่ละพจน์ออกมาให้อยู่ในรูปกรณฑ์อย่างง่ายที่มีตัวร่วมเหมือนกัน)
$= (2+3)\sqrt{2} = 5\sqrt{2}$ (ดึงตัวร่วม $\sqrt{2}$ ออกแล้วนำสัมประสิทธิ์ข้างหน้ามาบวกกันตามปกติ) ตอบ $5\sqrt{2}$
2.4) จงจัดรูปอย่างง่ายของ $5\sqrt{12} - \sqrt{27}$
$= 5\sqrt{4 \times 3} - \sqrt{9 \times 3}$ (จัดรูปเลขใต้กรณฑ์ แยกกลุ่มที่มีเลขยกกำลังสองซ่อนอยู่)
$= 5(2\sqrt{3}) - 3\sqrt{3} = 10\sqrt{3} - 3\sqrt{3}$ (ถอดรากของ 4 ได้ 2 นำไปคูณกับ 5 ข้างหน้า ได้เป็น 10 และถอดรากของ 9 ได้ 3)
$= 7\sqrt{3}$ (นำสัมประสิทธิ์หน้ากรณฑ์ชนิดเดียวกันมาลบกันให้เรียบร้อย) ตอบ $7\sqrt{3}$
2.5) จงจัดรูปอย่างง่ายของ $\sqrt{2} \times \sqrt{8}$
$= \sqrt{2 \times 8} = \sqrt{16}$ (ใช้สมบัติการคูณกรณฑ์อันดับเดียวกัน ยุบรวมตัวเลขเข้ามาคูณกันข้างใน)
$= 4$ (ถอดรากที่สองของ 16 ออกมาได้ลงตัวเป็นจำนวนเต็มพอดี) ตอบ 4
2.6) จงจัดรูปอย่างง่ายของ $\frac{\sqrt{75}}{\sqrt{3}}$
$= \sqrt{\frac{75}{3}} = \sqrt{25}$ (ใช้สมบัติการหารกรณฑ์อันดับเดียวกัน รวมร่างเศษส่วนแล้วตัดทอนตัวเลขข้างใน)
$= 5$ (ถอดรากที่สองของ 25 ออกมาได้ผลลัพธ์สุดท้ายที่สวยงาม) ตอบ 5
2.7) จงจัดรูปอย่างง่ายของ $\sqrt{x^3 y^4}$ เมื่อ $x \ge 0, y \ge 0$
$= \sqrt{x^2 \cdot x \cdot (y^2)^2}$ (กระจายตัวแปรออกเป็นกลุ่มๆ ที่ยกกำลังคู่เพื่อให้สอดรับกับรากที่สอง)
$= x y^2 \sqrt{x}$ (ดึงตัวแปรกำลังสองออกนอกกรณฑ์ ส่วน x ที่เหลือตัวเดียวยังคงติดอยู่ข้างใน) ตอบ $x y^2 \sqrt{x}$
2.8) จงจัดรูปอย่างง่ายของ $\sqrt[3]{a^4 b^6}$ เมื่อ $a, b$ เป็นจำนวนจริงใดๆ
$= \sqrt[3]{a^3 \cdot a \cdot (b^2)^3}$ (จัดกลุ่มให้ตัวแปรยกกำลังสามเพื่อเตรียมถอดรากอันดับที่สาม)
$= a b^2 \sqrt[3]{a}$ (ดึงกลุ่มกำลังสามออกมาข้างนอกกรณฑ์ ตัวแปร a ที่เหลือยังติดอยู่ในรากที่สามเหมือนเดิม) ตอบ $a b^2 \sqrt[3]{a}$
ข้อ 3. จงจัดรูปพจน์ต่อไปนี้ให้ตัวส่วนไม่ติดกรณฑ์ (มี 10 ข้อย่อย)
3.1) จงจัดรูป $\frac{1}{\sqrt{2}}$ ไม่ให้ส่วนติดกรณฑ์
$= \frac{1}{\sqrt{2}} \times \frac{\sqrt{2}}{\sqrt{2}}$ (คูณเศษและส่วนด้วย $\sqrt{2}$ เพื่อทำให้รากที่สองตรงส่วนยกกำลังสองและหลุดออกไป)
$= \frac{\sqrt{2}}{2}$ (เศษคูณเศษได้ $\sqrt{2}$ ส่วนคูณส่วนจะได้ $\sqrt{2} \times \sqrt{2} = 2$) ตอบ $\frac{\sqrt{2}}{2}$
3.2) จงจัดรูป $\frac{6}{\sqrt{3}}$ ไม่ให้ส่วนติดกรณฑ์
$= \frac{6}{\sqrt{3}} \times \frac{\sqrt{3}}{\sqrt{3}}$ (นำรากที่สองของสามคูณทั้งส่วนและเศษเพื่อเคลียร์กรณฑ์ด้านล่าง)
$= \frac{6\sqrt{3}}{3}$ (คำนวณผลคูณแยกชั้น เศษได้ $6\sqrt{3}$ ส่วนได้ค่าเป็น 3)
$= 2\sqrt{3}$ (นำ 3 ด้านล่างไปหาร 6 ตัวเศษ ตัดทอนจนได้รูปอย่างง่ายที่สุด) ตอบ $2\sqrt{3}$
3.3) จงจัดรูป $\frac{\sqrt{2}}{\sqrt{5}}$ ไม่ให้ส่วนติดกรณฑ์
$= \frac{\sqrt{2}}{\sqrt{5}} \times \frac{\sqrt{5}}{\sqrt{5}}$ (เลือกคูณด้วย $\sqrt{5}$ ทั้งบนและล่างโดยอิงจากสมบัติของตัวส่วนเป็นหลัก)
$= \frac{\sqrt{10}}{5}$ (เศษคูณกันยุบรวมเป็น $\sqrt{10}$ และตัวส่วนกลายเป็นจำนวนเต็ม 5) ตอบ $\frac{\sqrt{10}}{5}$
3.4) จงจัดรูป $\frac{1}{\sqrt[3]{2}}$ ไม่ให้ส่วนติดกรณฑ์
$= \frac{1}{\sqrt[3]{2}} \times \frac{\sqrt[3]{2^2}}{\sqrt[3]{2^2}}$ (ระวัง! เป็นรากที่สาม จึงต้องคูณเติมเต็มด้วย $2^2$ เพื่อรวมให้ข้างล่างครบกำลังสามคือ $2^3$)
$= \frac{\sqrt[3]{4}}{2}$ (ตัวเศษคำนวณกำลังได้เป็น $\sqrt[3]{4}$ ส่วนตัวส่วนหลุดออกมาเป็นเลข 2 เคลียร์ส่วนไม่ติดรูท) ตอบ $\frac{\sqrt[3]{4}}{2}$
3.5) จงจัดรูป $\frac{5}{\sqrt[3]{5}}$ ไม่ให้ส่วนติดกรณฑ์
$= \frac{5}{\sqrt[3]{5}} \times \frac{\sqrt[3]{5^2}}{\sqrt[3]{5^2}}$ (คูณเศษส่วนด้วยรากที่สามของ $5^2$ เพื่อทำให้กำลังของตัวส่วนสะสมรวมกันได้ครบสามพจน์)
$= \frac{5\sqrt[3]{25}}{5}$ (เศษจะกลายเป็น $5\sqrt[3]{25}$ ส่วนพจน์ข้างล่างหลุดกรณฑ์ออกเป็นเลข 5 เต็มตัว)
$= \sqrt[3]{25}$ (นำเลข 5 ที่ส่วนตัดทอนกับเลข 5 ด้านบนพจน์เศษจนหมดไป) ตอบ $\sqrt[3]{25}$
3.6) จงจัดรูป $\frac{1}{2 + \sqrt{3}}$ ไม่ให้ส่วนติดกรณฑ์
$= \frac{1}{2 + \sqrt{3}} \times \frac{2 - \sqrt{3}}{2 - \sqrt{3}}$ (ตัวส่วนมีสองพจน์ ต้องคูณด้วยสังยุค หรือ Conjugate เปลี่ยนจากบวกเป็นลบ)
$= \frac{2 - \sqrt{3}}{2^2 - (\sqrt{3})^2} = \frac{2 - \sqrt{3}}{4 - 3}$ (ด้านล่างเข้าสูตรผลต่างกำลังสอง $(A+B)(A-B) = A^2 - B^2$ รูทจะหลุดหายไปทันที)
$= \frac{2 - \sqrt{3}}{1} = 2 - \sqrt{3}$ (เมื่อตัวส่วนคำนวณหักลบกันแล้วได้ 1 จึงยุบตอบเหลือเฉพาะส่วนเศษ) ตอบ $2 - \sqrt{3}$
3.7) จงจัดรูป $\frac{1}{\sqrt{5} - \sqrt{2}}$ ไม่ให้ส่วนติดกรณฑ์
$= \frac{1}{\sqrt{5} - \sqrt{2}} \times \frac{\sqrt{5} + \sqrt{2}}{\sqrt{5} + \sqrt{2}}$ (ส่วนเป็นลบ ให้คูณด้วยคอนจูเกตที่เป็นเครื่องหมายบวกเพื่อเข้าโครงสร้างผลต่างกำลังสอง)
$= \frac{\sqrt{5} + \sqrt{2}}{(\sqrt{5})^2 - (\sqrt{2})^2} = \frac{\sqrt{5} + \sqrt{2}}{5 - 2}$ (กระจายส่วนเป็นหน้ากำลังสองลบหลังกำลังสอง ทำให้กำลังสองตัดรากขาดกระจาย)
$= \frac{\sqrt{5} + \sqrt{2}}{3}$ (คำนวณเลขส่วนเป็น 3 ตัวแปรเศษติดรูทต่างชนิดบวกกันต่อไม่ได้แล้ว) ตอบ $\frac{\sqrt{5} + \sqrt{2}}{3}$
3.8) จงจัดรูป $\frac{\sqrt{3} + 1}{\sqrt{3} - 1}$ ไม่ให้ส่วนติดกรณฑ์
$= \frac{\sqrt{3} + 1}{\sqrt{3} - 1} \times \frac{\sqrt{3} + 1}{\sqrt{3} + 1}$ (ยึดตามตัวส่วนที่ติดลบ นำสังยุคเครื่องหมายบวกคูณเข้าเติมไปทั้งระบบ)
$= \frac{(\sqrt{3} + 1)^2}{(\sqrt{3})^2 - 1^2}$ (เศษกลายเป็นกำลังสองสมบูรณ์ ส่วนด้านล่างยุบตัวเป็นหน้ากำลังสองลบหลังกำลังสอง)
$= \frac{3 + 2\sqrt{3} + 1}{3 - 1} = \frac{4 + 2\sqrt{3}}{2}$ (กระจายกำลังสองสมบูรณ์ข้างบน หน้า²+2หน้าหลัง+หลัง² นำจำนวนเต็มบวกกันรวมกองได้ 4)
$= \frac{2(2 + \sqrt{3})}{2} = 2 + \sqrt{3}$ (ดึงเลข 2 ที่เป็นตัวร่วมของเศษออกมาเพื่อตัดทอนกับตัวส่วนด้านล่างให้สิ้นซาก) ตอบ $2 + \sqrt{3}$
3.9) จงจัดรูป $\frac{x}{\sqrt{x}}$ เมื่อ $x > 0$ ไม่ให้ส่วนติดกรณฑ์
$= \frac{x}{\sqrt{x}} \times \frac{\sqrt{x}}{\sqrt{x}}$ (คูณปรับพจน์ด้วย $\sqrt{x}$ ทั้งส่วนและเศษเพื่อกวาดล้างรูทส่วนล่าง)
$= \frac{x\sqrt{x}}{x}$ (ตัวส่วนคูณกันได้ x เต็มตามเงื่อนไขที่โจทย์ปูพรมมาว่ามันเป็นบวกแน่ๆ)
$= \sqrt{x}$ (ตัดทอนตัวแปร x ของพจน์เศษและพจน์ส่วนทิ้ง ได้คำตอบสำเร็จรูปขั้นสูงสุด) ตอบ $\sqrt{x}$
3.10) จงจัดรูป $\frac{1}{\sqrt{x+1} - \sqrt{x}}$ เมื่อ $x \ge 0$ ไม่ให้ส่วนติดกรณฑ์
$= \frac{1}{\sqrt{x+1} - \sqrt{x}} \times \frac{\sqrt{x+1} + \sqrt{x}}{\sqrt{x+1} + \sqrt{x}}$ (ส่วนเป็นกลุ่มลบ ให้ระดมคูณด้วย Conjugate รูปแบบบวกของชุดเดิมเข้าไป)
$= \frac{\sqrt{x+1} + \sqrt{x}}{(\sqrt{x+1})^2 - (\sqrt{x})^2} = \frac{\sqrt{x+1} + \sqrt{x}}{(x+1) - x}$ (ประยุกต์สูตรผลต่างกำลังสองเข้าสลายตัวกรณฑ์ชุดล่างทั้งหมด)
$= \frac{\sqrt{x+1} + \sqrt{x}}{1} = \sqrt{x+1} + \sqrt{x}$ (ส่วนตัดลบเหลือ 1 ย้ายคำตอบเศษขึ้นมาจัดพิมพ์สรุปได้เรียบร้อย) ตอบ $\sqrt{x+1} + \sqrt{x}$

แบบทดสอบ: เลขยกกำลัง ม.5

ข้อ 1: จงหาค่าของ ³√(−64) + √144 − ⁴√81


³√(−64) = −4   เพราะ (−4)³ = −64
√144 = 12   (ค่าหลักที่เป็นบวก)
⁴√81 = 3   เพราะ 3⁴ = 81
∴ (−4) + 12 − 3 = 5 → ข้อ ก.

ข้อ 2: จัดรูป 5/(√6 − 1) ให้ตัวส่วนไม่ติดกรณฑ์ ได้ตรงกับข้อใด


คูณด้วยสังยุค: [5/(√6−1)] × [(√6+1)/(√6+1)]
= 5(√6+1) / (6−1)   [ใช้สูตร (A−B)(A+B) = A²−B²]
= 5(√6+1) / 5 = √6+1 → ข้อ ข.

ข้อ 3: (25)^(3/2) × (8)^(−2/3) มีค่าเท่ากับเท่าใด


(25)^(3/2) = (5²)^(3/2) = 5^(2×3/2) = 5³ = 125
(8)^(−2/3) = (2³)^(−2/3) = 2^(3×−2/3) = 2^(−2) = 1/4
∴ 125 × 1/4 = 125/4 → ข้อ ก.

ข้อ 4: ฝากเงิน 10,000 บาท ดอกเบี้ยทบต้น 2% ต่อปี นาน 3 ปี จะมีเงินตามสมการใด (สูตร Sₙ = P(1+r)ⁿ)


P = 10,000, r = 2% = 0.02, n = 3
S₃ = 10,000 × (1 + 0.02)³
∴ 10,000(1.02)³ → ข้อ ข.
