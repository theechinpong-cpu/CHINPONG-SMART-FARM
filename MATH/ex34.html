ตัวอย่างที่ 3: จงเขียนจำนวนให้อยู่ในรูปอย่างง่าย และเลขชี้กำลังเป็นจำนวนเต็มบวก

(1) จงทำให้อยู่ในรูปอย่างง่าย: $$\frac{3^0 + 2^0}{2^{-3}}$$
วิธีทำ: (เริ่มต้นการคำนวณจากโจทย์ที่กำหนดให้)
$$= \frac{1 + 1}{2^{-3}} \quad \text{[เปลี่ยน } 3^0 = 1 \text{ และ } 2^0 = 1 \text{ ตามนิยาม } a^0 = 1 \text{ เมื่อ } a \neq 0\text{]}$$
$$= \frac{2}{2^{-3}} \quad \text{[รวมผลบวกที่ตัวเศษด้านบน } 1 + 1 = 2\text{]}$$
$$= 2^1 \times 2^3 \quad \text{[ย้าย } 2^{-3} \text{ จากตัวส่วนขึ้นมาเป็นตัวคูณด้านบน ทำให้เลขชี้กำลังเปลี่ยนจากลบเป็นบวกตามกฎ } \frac{1}{a^{-n}} = a^n\text{]}$$
$$= 2^{1+3} \quad \text{[นำเลขชี้กำลังมาบวกกันเมื่อฐานเหมือนกันคูณกันตามกฎ } a^m \times a^n = a^{m+n}\text{]}$$
$$= 2^4 \quad \text{[คำนวณผลรวมเลขชี้กำลังได้ } 1 + 3 = 4\text{]}$$
ตอบ: ผลลัพธ์คือ $16$ $\quad \text{[คำนวณค่าของ } 2^4 = 2 \times 2 \times 2 \times 2 = 16\text{]}$
(2) จงทำให้อยู่ในรูปอย่างง่าย: $$\frac{3^{-1} + 2^0}{2^{-1} + 2^2}$$
วิธีทำ: (เริ่มต้นการคำนวณจากโจทย์ที่กำหนดให้)
$$= \frac{\frac{1}{3} + 1}{\frac{1}{2} + 4} \quad \text{[ปรับ } 3^{-1} = \frac{1}{3} \text{, } 2^{-1} = \frac{1}{2} \text{ ตามกฎ } a^{-n} = \frac{1}{a^n} \text{ และคิดค่า } 2^0 = 1 \text{, } 2^2 = 4\text{]}$$
$$= \frac{\frac{1+3}{3}}{\frac{1+8}{2}} \quad \text{[หา ค.ร.น. เพื่อรวมเศษส่วน โดยเศษด้านบนคูณด้วย } \frac{3}{3} \text{ และส่วนด้านล่างคูณด้วย } \frac{2}{2}\text{]}$$
$$= \frac{\frac{4}{3}}{\frac{9}{2}} \quad \text{[คำนวณผลบวกของตัวเลขในเศษและส่วน จะได้เศษคือ } \frac{4}{3} \text{ และส่วนคือ } \frac{9}{2}\text{]}$$
$$= \frac{4}{3} \times \frac{2}{9} \quad \text{[เปลี่ยนเครื่องหมายหารยาวเป็นคูณ แล้วกลับเศษเป็นส่วนของตัวหารด้านล่าง จาก } \frac{9}{2} \text{ เป็น } \frac{2}{9}\text{]}$$
$$= \frac{4 \times 2}{3 \times 9} \quad \text{[จับเศษคูณเศษ และส่วนคูณส่วนตามหลักการคูณเศษส่วนทั่วไป]}$$
ตอบ: ผลลัพธ์คือ $$\frac{8}{27}$$ $\quad \text{[คำนวณผลคูณสุทธิสุท้าย } 4 \times 2 = 8 \text{ และ } 3 \times 9 = 27\text{]}$
(3) จงทำให้อยู่ในรูปอย่างง่าย: $$\left( \frac{x^{2n}}{x^{2n-1}} \right)^{-3}$$
วิธีทำ: (เริ่มต้นการคำนวณจากโจทย์ที่กำหนดให้)
$$= \left( x^{2n - (2n - 1)} \right)^{-3} \quad \text{[ยุบฐานตัวหารด้านในวงเล็บ โดยนำเลขชี้กำลังด้านล่างขึ้นไปลบตามกฎ } \frac{a^m}{a^n} = a^{m-n}\text{]}$$
$$= \left( x^{2n - 2n + 1} \right)^{-3} \quad \text{[กระจายเครื่องหมายลบเข้าไปในวงเล็บย่อย ทำให้จาก } -(2n-1) \text{ กลายเป็น } -2n+1\text{]}$$
$$= \left( x^1 \right)^{-3} \quad \text{[หักล้างตัวแปรในเลขชี้กำลัง } 2n - 2n = 0 \text{ เหลือเพียงเลขชี้กำลังก้อนในเป็น } 1\text{]}$$
$$= x^{-3} \quad \text{[นำเลขชี้กำลังซ้อนกันมาคูณกันด้านนอกวงเล็บตามกฎ } (a^m)^n = a^{m \times n} \text{ จะได้ } 1 \times (-3) = -3\text{]}$$
ตอบ: รูปอย่างง่ายคือ $$\frac{1}{x^3}$$ $\quad \text{[กลับเลขชี้กำลังที่ติดลบลงไปเป็นตัวส่วนด้านล่างเพื่อให้เป็นค่าบวกตามนิยาม } a^{-n} = \frac{1}{a^n}\text{]}$
(4) จงทำให้อยู่ในรูปอย่างง่าย: $$\frac{x^{-1} + y^{-1}}{x^{-1} - y^{-1}}$$
วิธีทำ: (เริ่มต้นการคำนวณจากโจทย์ที่กำหนดให้)
$$= \frac{\frac{1}{x} + \frac{1}{y}}{\frac{1}{x} - \frac{1}{y}} \quad \text{[แปลงตัวแปรยกกำลังลบทั้งหมดให้อยู่ในรูปเศษส่วนบวกตามกฎ } x^{-1}=\frac{1}{x} \text{ และ } y^{-1}=\frac{1}{y}\text{]}$$
$$= \frac{\frac{y + x}{xy}}{\frac{y - x}{xy}} \quad \text{[หา ค.ร.น. ของเศษส่วนทั้งชุดบนและชุดล่างคือ } xy \text{ แล้วทำส่วนให้เท่ากันด้วยการคูณไขว้]}$$
$$= \frac{y + x}{xy} \times \frac{xy}{y - x} \quad \text{[เปลี่ยนเครื่องหมายหารยาวตรงกลางเป็นคูณ แล้วกลับเศษเป็นส่วนของพจน์ล่างจาก } \frac{y-x}{xy} \text{ เป็น } \frac{xy}{y-x}\text{]}$$
$$= \frac{(y + x) \cdot xy}{xy \cdot (y - x)} \quad \text{[รวมรูปคูณตัดทอน โดยพบว่ามีตัวแปร } xy \text{ อยู่ทั้งเศษและส่วนจึงสามารถหารตัดกันได้ลดรูปเหลือ } 1\text{]}$$
ตอบ: รูปอย่างง่ายคือ $$\frac{x + y}{y - x}$$ $\quad \text{[จัดสลับที่ระบบตัวแปรการบวกพจน์บนเศษให้สวยงามจาก } y+x \text{ เป็น } x+y\text{]}$
(5) จงทำให้อยู่ในรูปอย่างง่าย: $$\frac{x^{-1} + y^{-1}}{x^{-1}}$$
วิธีทำ: (เริ่มต้นการคำนวณจากโจทย์ที่กำหนดให้)
$$= \frac{x^{-1}}{x^{-1}} + \frac{y^{-1}}{x^{-1}} \quad \text{[ใช้สมบัติการกระจายเศษส่วนแยกตัวส่วนร่วมออกเป็นสองพจน์ตามสูตร } \frac{a+b}{c} = \frac{a}{c} + \frac{b}{c}\text{]}$$
$$= 1 + \frac{x^1}{y^1} \quad \text{[พจน์แรกตัดกันเหลือ } 1 \text{ พจน์หลังสลับฝั่งยกกำลังลบตามสูตร } \frac{y^{-1}}{x^{-1}} = \frac{x^1}{y^1}\text{]}$$
$$= 1 + \frac{x}{y} \quad \text{[ถอดรูปเลขยกกำลังหนึ่งออก เนื่องจากเลขชี้กำลัง } 1 \text{ ไม่จำเป็นต้องเขียนแสดง]}$$
$$= \frac{y}{y} + \frac{x}{y} \quad \text{[ปรับปรุงจำนวนเต็ม } 1 \text{ ให้เป็นเศษส่วนโดยคูณด้วย } \frac{y}{y} \text{ เพื่อให้ฐานส่วนเท่ากันเตรียมบวกพจน์]}$$
ตอบ: รูปอย่างง่ายคือ $$\frac{y + x}{y}$$ $\quad \text{[รวมเศษเข้าด้วยกันโดยมีส่วนร่วมคือ } y \text{ ได้ผลลัพธ์สำเร็จรูปเรียบร้อยแล้า]}$
(6) จงทำให้อยู่ในรูปอย่างง่าย: $$\frac{(x+y)^{-1}}{x^{-1} + y^{-1}}$$
วิธีทำ: (เริ่มต้นการคำนวณจากโจทย์ที่กำหนดให้)
$$= \frac{\frac{1}{x+y}}{\frac{1}{x} + \frac{1}{y}} \quad \text{[เปลี่ยนกำลังลบเศษและส่วนแยกพจน์ตามสมบัติ } a^{-1} = \frac{1}{a}\text{]}$$
$$= \frac{\frac{1}{x+y}}{\frac{y+x}{xy}} \quad \text{[รวมพจน์ส่วนด้านล่างให้เป็นกลุ่มเดียวกันโดยการหา ค.ร.น. แบบคูณไขว้ได้พจน์เศษเป็น } y+x \text{ มีส่วนเป็น } xy\text{]}$$
$$= \frac{1}{x+y} \times \frac{xy}{x+y} \quad \text{[เปลี่ยนเครื่องหมายหารยาวหลักเป็นคูณ แล้วตลบเศษส่วนของตัวล่างขึ้นมา และปรับรูปจาก } y+x \text{ เป็น } x+y\text{]}$$
$$= \frac{1 \times xy}{(x+y) \times (x+y)} \quad \text{[ดำเนินการคูณเศษส่วนโดยจับกลุ่มเศษชนเศษและกลุ่มส่วนคูณส่วนเข้าด้วยกัน]}$$
ตอบ: รูปอย่างง่ายคือ $$\frac{xy}{(x+y)^2}$$ $\quad \text{[ยุบรวมกลุ่มฐานส่วนที่เหมือนกันคูณกันให้อยู่ในรูปยกกำลังสองตามกฎ } a \times a = a^2\text{]}$

ตัวอย่างที่ 4: จงหาเซตคำตอบของสมการต่อไปนี้ (การแก้สมการเลขยกกำลัง)

(1) แก้สมการ: $$27^{x-2} = 81$$
วิธีทำ: (เริ่มต้นกระบวนการแก้สมการโดยการจัดรูปฐานตัวเลข)
$$(3^3)^{x-2} = 3^4 \quad \text{[แปลงฐานทั้งสองฝั่งให้เป็นเลขฐานเฉพาะ 3 โดยเปลี่ยน } 27 = 3^3 \text{ และ } 81 = 3^4\text{]}$$
$$3^{3(x-2)} = 3^4 \quad \text{[นำเลขชี้กำลังซ้อนมาคูณกระจายเข้าด้วยกันตามสมบัติของเลขยกกำลัง } (a^m)^n = a^{mn}\text{]}$$
$$3(x-2) = 4 \quad \text{[สมบัติฟังก์ชันเอกซ์โพเนนเชียล: เมื่อฐานเลขเท่ากันทั้งสองฝั่ง ให้จับส่วนเลขชี้กำลังมาตั้งสมการเท่ากันได้เลย]}$$
$$3x - 6 = 4 \quad \text{[กระจายเลข } 3 \text{ เข้าไปในวงเล็บทางด้านซ้ายของสมการ จะได้ } 3x - 6\text{]}$$
$$3x = 10 \quad \text{[ย้ายค่าคงที่ } -6 \text{ จากฝั่งซ้ายข้ามไปบวกทางฝั่งขวา จะได้ } 4 + 6 = 10\text{]}$$
$$x = \frac{10}{3} \quad \text{[ย้ายเลข } 3 \text{ ซึ่งทำหน้าที่คูณอยู่กับตัวแปร } x \text{ ข้ามฝั่งลงไปเป็นตัวหารด้านล่าง]}$$
เซตคำตอบคือ: $$\left\{ \frac{10}{3} \right\}$$ $\quad \text{[นำคำตอบตัวแปร } x \text{ มาเขียนสรุปจัดรูปแบบให้อยู่ในรูปของเซตคำตอบตามโครงสร้างคณิตศาสตร์]}$
(2) แก้สมการ: $$8 \times \left(\frac{4}{9}\right)^{x+3} = 27$$
วิธีทำ: (เริ่มต้นกระบวนการแก้สมการโดยการจัดรูปแยกกลุ่ม)
$$\left(\frac{4}{9}\right)^{x+3} = \frac{27}{8} \quad \text{[ย้ายเลข } 8 \text{ จากฝั่งซ้ายที่เป็นตัวคูณ ข้ามฝั่งมาเป็นตัวหารด้านล่างของฝั่งขวาเพื่อแยกกลุ่มตัวแปร]}$$
$$\left( \left(\frac{2}{3}\right)^2 \right)^{x+3} = \frac{3^3}{2^3} \quad \text{[เปลี่ยนเศษส่วนให้เป็นเลขยกกำลังฐานย่อย โดยเปลี่ยน } \frac{4}{9} = (\frac{2}{3})^2 \text{ และเปลี่ยน } \frac{27}{8} = \frac{3^3}{2^3}\text{]}$$
$$\left(\frac{2}{3}\right)^{2x+6} = \left(\frac{3}{2}\right)^3 \quad \text{[ฝั่งซ้ายคูณกระจายเลขชี้กำลังซ้อนกันเข้าไป ฝั่งขวารวบดึงเลขยกกำลังร่วมออกมานอกวงเล็บ]}$$
$$\left(\frac{2}{3}\right)^{2x+6} = \left(\frac{2}{3}\right)^{-3} \quad \text{[ปรับฐานฝั่งขวาโดยการสลับเศษเป็นส่วนเพื่อให้ฐานเป็น } \frac{2}{3} \text{ เท่ากัน ส่งผลให้เลขชี้กำลังเปลี่ยนจากบวกเป็นลบตามสูตร]}$$
$$2x + 6 = -3 \quad \text{[ฐานของสมการเลขยกกำลังเท่ากันทั้งสองฝั่งแล้ว ให้ทำการปลดฐานออกแล้วจับเลขชี้กำลังมาเท่ากัน]}$$
$$2x = -9 \quad \text{[ย้ายค่าคงที่ } +6 \text{ จากฝั่งซ้ายข้ามไปลบทางฝั่งขวา ทำให้ได้ค่าฝั่งขวาเท่ากับ } -3 - 6 = -9\text{]}$$
$$x = -\frac{9}{2} \quad \text{[ย้ายเลข } 2 \text{ ที่เป็นสัมประสิทธิ์คูณอยู่กับ } x \text{ ลงมาเป็นตัวหารทางฝั่งขวาเพื่อหาค่าตัวแปรสุดท้าย]}$$
เซตคำตอบคือ: $$\left\{ -\frac{9}{2} \right\}$$ $\quad \text{[นำค่าตัวแปร } x \text{ ที่ได้มาบรรจุลงในวงเล็บปีกกาเพื่อสรุปเป็นรูปของเซตคำตอบคณิตศาสตร์ที่สมบูรณ์]}$
(3) แก้สมการ: $$4^x = \frac{1}{8}$$
วิธีทำ: (เริ่มต้นกระบวนการแก้สมการโดยจัดรูปเลขฐานเพื่อเตรียมลดทอนสัญลักษณ์)
$$(2^2)^x = \frac{1}{2^3} \quad \text{[แปลงพจน์ของเลขฐานในสมการให้เป็นฐาน 2 ทั้งหมด โดยแปลงรูป } 4 = 2^2 \text{ และเปลี่ยน } 8 = 2^3\text{]}$$
$$2^{2x} = 2^{-3} \quad \text{[ฝั่งซ้ายคูณกำลังซ้อนกันได้ } 2x \text{ ฝั่งขวาดันตัวยกกำลังจากส่วนด้านล่างขึ้นมาด้านบนทำให้เลขกำลังสลับเครื่องหมายเป็นลบ]}$$
$$2x = -3 \quad \text{[ตัดฐานเลข 2 ออกทั้งสองข้างเนื่องจากฐานสมดุลเท่ากันแล้ว จากนั้นจับส่วนเลขชี้กำลังมาเทียบค่าเท่ากัน]}$$
$$x = -\frac{3}{2} \quad \text{[ย้ายสัมประสิทธิ์ตัวเลข } 2 \text{ จากฝั่งซ้ายที่ทำหน้าที่คูณอยู่ ข้ามฝั่งลงไปหารใต้เลข } -3 \text{ ทางฝั่งขวา]}$$
เซตคำตอบคือ: $$\left\{ -\frac{3}{2} \right\}$$ $\quad \text{[สรุปรูปข้อตกลงทางคณิตศาสตร์โดยเขียนผลลัพธ์คำตอบให้อยู่ภายในเครื่องหมายเซตปีกกา]}$
(4) แก้สมการ: $$\left(\frac{4}{9}\right)^{3x} = \frac{32}{243}$$
วิธีทำ: (เริ่มต้นกระบวนการแก้สมการโดยการแยกวิเคราะห์ตัวประกอบของฐานทั้งสองฝั่ง)
$$\left( \left(\frac{2}{3}\right)^2 \right)^{3x} = \frac{2^5}{3^5} \quad \text{[วิเคราะห์แตกตัวเลขเป็นฐานจำนวนเฉพาะ โดยเปลี่ยน } \frac{4}{9} = (\frac{2}{3})^2 \text{ และเปลี่ยน } \frac{32}{243} = \frac{2^5}{3^5}\text{]}$$
$$\left(\frac{2}{3}\right)^{6x} = \left(\frac{2}{3}\right)^5 \quad \text{[ฝั่งซ้ายคูณกำลังซ้อนกันได้ } 2 \times 3x = 6x \text{ ฝั่งขวารวบดึงยกกำลัง } 5 \text{ เป็นกำลังร่วมด้านนอกวงเล็บ]}$$
$$6x = 5 \quad \text{[ปลดเศษส่วนฐาน } \frac{2}{3} \text{ ทิ้งทั้งสองฝั่ง เนื่องจากฐานสมบูรณ์เท่ากันแล้ว จึงนำเลขชี้กำลังมาตั้งเทียบกัน]}$$
$$x = \frac{5}{6} \quad \text{[ย้ายเลข } 6 \text{ ที่อยู่ติดกับตัวแปรตัวอักษรทางฝั่งซ้ายลงไปเป็นตัวส่วนหารทางฝั่งขวาเพื่อหาคำตอบสำเร็จ]}$$
เซตคำตอบคือ: $$\left\{ \frac{5}{6} \right\}$$ $\quad \text{[นำค่าตัวแปรเศษส่วนคำตอบของสมการมาประกอบร่างเขียนสรุปปิดท้ายให้อยู่ในรูปรูปแบบสัญลักษณ์เซต]}$$
(5) แก้สมการ: $$3^{-x+1} = \frac{9}{25}(5^{1-x})$$
วิธีทำ: (เริ่มต้นกระบวนการโดยแยกวิเคราะห์กลุ่มเลขฐานต่าง ๆ ออกจากกัน)
$$3^{-x+1} = \frac{3^2}{5^2} \cdot 5^{1-x} \quad \text{[จัดรูปเลขเศษส่วนหลักด้านขวาคือเปลี่ยนเลข } 9 = 3^2 \text{ และเปลี่ยนเลข } 25 = 5^2 \text{ เพื่อแยกส่วนกลุ่มฐาน]}$$
$$\frac{3^{-x+1}}{3^2} = \frac{5^{1-x}}{5^2} \quad \text{[ใช้กฎการย้ายข้างข้ามฝั่ง โดยย้ายตัวคูณและตัวหารให้ฐานเดียวกันไปรวมกลุ่มอยู่ฝั่งเดียวกันซ้ายและขวา]}$$
$$3^{-x+1-2} = 5^{1-x-2} \quad \text{[ใช้สมบัติการหารเลขยกกำลังฐานเดียวกัน โดยนำค่าชี้กำลังจากส่วนด้านล่างขึ้นไปลบพจน์บนตามสูตร]}$$
$$3^{-x-1} = 5^{-x-1} \quad \text{[คำนวณผลลัพธ์เลขชี้กำลังของทั้งสองฝั่งสมการ โดยคิดคำนวณในส่วนค่าคงที่ } 1 - 2 = -1\text{]}$$
$$\frac{3^{-x-1}}{5^{-x-1}} = 1 \quad \text{[ย้ายกลุ่มเลขยกกำลังฝั่งขวาทั้งหมดลงมาเป็นตัวหารฝั่งซ้าย เพื่อทำให้โครงสร้างฝั่งขวาเหลือค่าเท่ากับ } 1\text{]}$$
$$\left(\frac{3}{5}\right)^{-x-1} = \left(\frac{3}{5}\right)^0 \quad \text{[รวบดึงกำลังร่วมฝั่งซ้ายออกนอกวงเล็บ และปรับค่าเลข } 1 \text{ ฝั่งขวาให้กลายเป็นฐานเหมือนกันยกกำลังศูนย์]}$$
$$-x - 1 = 0 \quad \text{[เมื่อเลขฐานของทั้งสองข้างเท่ากันเรียบร้อยแล้ว ให้ทำการปลดฐานออกแล้วนำค่าชี้กำลังมาตั้งสมการเทียบค่ากัน]}$$
$$-x = 1 \quad \text{[ย้ายค่าคงตัวสัญลักษณ์เลข } -1 \text{ จากฝั่งซ้ายข้ามฟากไปทางขวา ทำให้เครื่องหมายเปลี่ยนผันสลับจากลบกลายเป็นบวก]}$$
$$x = -1 \quad \text{[นำเครื่องหมายลบคูณตลอดทั้งสมการทั้งสองข้าง เพื่อสลับเปลี่ยนให้ตัวแปร } x \text{ เป็นบวกตามรูปแบบมาตรฐาน]}$$
เซตคำตอบคือ: $$\{ -1 \}$$ $\quad \text{[สรุปรูปข้อตกลงขั้นตอนสุดท้ายโดยเขียนตัวเลขคำตอบที่ถูกต้องบรรจุลงในระบบวงเล็บปีกกาเซต]}$$
(6) แก้สมการ: $$(8^{x-1})(0.4)^{-2} = 5^{3x-3}$$
วิธีทำ: (เริ่มต้นกระบวนการแก้สมการโดยการแปลงเลขทศนิยมให้เป็นเศษส่วนอย่างต่ำ)
$$(2^3)^{x-1} \times \left(\frac{2}{5}\right)^{-2} = 5^{3(x-1)} \quad \text{[แปลงรูปเลขฐาน } 8 = 2^3 \text{ ส่วนเลขทศนิยม } 0.4 = \frac{4}{10} = \frac{2}{5} \text{ และดึงตัวร่วมพจน์บนฝั่งขวาออก]}$$
$$2^{3x-3} \times \left(\frac{5}{2}\right)^2 = 5^{3x-3} \quad \text{[ฝั่งซ้ายคูณกำลังซ้อนพจน์แรก และพจน์สองกลับเศษส่วนด้านในวงเล็บเพื่อเปลี่ยนกำลังติดลบเป็นกำลังบวก]}$$
$$2^{3x-3} \times \frac{5^2}{2^2} = 5^{3x-3} \quad \text{[กระจายเลขชี้กำลังสองด้านนอกเข้าถล่มทั้งตัวเศษและตัวส่วนภายในวงเล็บตัวกลางตามหลักพีชคณิต]}$$
$$\frac{2^{3x-3}}{2^2} = \frac{5^{3x-3}}{5^2} \quad \text{[ย้ายสลับข้างกลุ่มตัวเลขฐานเดียวกันมารวมกัน โดยย้ายฐาน 2 มาหารฝั่งซ้าย และย้ายฐาน 5 ไปหารฝั่งขวา]}$$
$$2^{3x-3-2} = 5^{3x-3-2} \quad \text{[คำนวณลดรูปเลขยกกำลังของการหารเลขฐานเดียวกันตามสูตรกฎเลขยกกำลังโดยนำกำลังตัวส่วนขึ้นไปลบ]}$$
$$2^{3x-5} = 5^{3x-5} \quad \text{[คำนวณผลรวมค่าคงที่ในเลขชี้กำลังของทั้งเศษส่วนสองฟากข้าง โดยจะได้ผลลัพธ์คือ } -3 - 2 = -5\text{]}$$
$$\frac{2^{3x-5}}{5^{3x-5}} = 1 \quad \text{[ย้ายข้างโดยนำกลุ่มพจน์เลขยกกำลังฝั่งขวาทั้งหมดลงมาเป็นตัวหารฝั่งซ้ายเพื่อให้ฝั่งขวาเหลือสมดุลค่าเท่ากับ } 1\text{]}$$
$$\left(\frac{2}{5}\right)^{3x-5} = \left(\frac{2}{5}\right)^0 \quad \text{[รวบเลขชี้กำลังฝั่งซ้ายออกด้านนอก และแปลงเลข } 1 \text{ ฝั่งขวาให้อยู่ในรูปเลขฐานเศษส่วนเดียวกันยกกำลังศูนย์]}$$
$$3x - 5 = 0 \quad \text{[เมื่อฐานคณิตศาสตร์ของฟังก์ชันเท่ากันดีแล้ว ให้ปลดฐานออกแล้วจับสมการของเลขชี้กำลังด้านบนมาเทียบกัน]}$$
$$3x = 5 \quad \text{[ย้ายสัญลักษณ์ตัวเลขค่าคงที่ } -5 \text{ จากทางฝั่งซ้ายของสมการย้ายสลับข้างข้ามฟากมาเป็นบวกทางฝั่งขวา]}$$
$$x = \frac{5}{3} \quad \text{[ย้ายเลขสัมประสิทธิ์ } 3 \text{ ที่อยู่หน้าตัวแปรฝั่งซ้ายข้ามฝั่งสลับหน้าที่ลงมาเป็นตัวส่วนของเศษส่วนทางฝั่งขวา]}$$
เซตคำตอบคือ: $$\left\{ \frac{5}{3} \right\}$$ $\quad \text{[นำเอาค่าคำตอบเศษส่วนตัวแปรสุดท้ายมาเขียนสรุปเป็นโครงสร้างเซตคำตอบที่สมบูรณ์ตามระบบระเบียบเป็นอันเสร็จ]}$$

แบบฝึกหัด 1.1 (ข้อ 1): จงหาค่าของแต่ละข้อต่อไปนี้

(1) หาค่าของ: $$1^{100} + (-1)^{100}$$
วิธีทำ: (เริ่มต้นกระบวนการคิดวิเคราะห์โดยการแยกแยะพจน์เพื่อหาค่าทีละส่วน)
$$1^{100} = 1 \quad \text{[เนื่องจากค่าตัวเลขเลข 1 เมื่อนำมายกกำลังด้วยจำนวนเต็มบวกใดๆ ก็ตาม จะได้ผลลัพธ์เป็น 1 เสมอ]}$$
$$(-1)^{100} = 1 \quad \text{[เนื่องจากจำนวนจริงลบหนึ่งเมื่อนำมายกกำลังด้วยตัวเลขคู่ ผลลัพธ์จะผกผันเปลี่ยนเป็นบวกหนึ่งเสมอ]}$$
$$= 1 + 1 \quad \text{[นำค่าตัวเลขผลลัพธ์สำเร็จรูปที่วิเคราะห์ได้จากทั้งสองส่วนพจน์ด้านบนมาเขียนเชื่อมตั้งการบวก]}$$
ตอบ: ค่าของอนุกรมพจน์นี้คือ $2$ $\quad \text{[บวกคำนวณตัวรวมค่าตัวเลขขั้นสุดท้ายทั่วไปได้ผลลัพธ์สุทธิสำเร็จเท่ากับ } 2\text{]}$$
(2) หาค่าของ: $$(-1)^{2019} + 2019^0$$
วิธีทำ: (เริ่มต้นกระบวนการโดยแกะค่าคุณสมบัติเฉพาะของเลขยกกำลังแต่ละพจน์ตามสูตรคณิตศาสตร์)
$$(-1)^{2019} = -1 \quad \text{[เนื่องจากเลขชี้กำลัง } 2019 \text; \text{เป็นจำนวนคี่ จึงทำให้ตัวฐานติดลบหนึ่งคงค่าติดลบเช่นเดิมไม่เปลี่ยน]}$$
$$2019^0 = 1 \quad \text{[จำนวนจริงใดๆ ที่ไม่เท่ากับศูนย์ เมื่อถูกยกลำด้วยศูนย์จะมีค่าเท่ากับหนึ่งตามกฎนิยาม } a^0 = 1\text{]}$$
$$= (-1) + 1 \quad \text{[นำค่าตัวเลขสรุปที่ถอดได้จากสมบัติทั้งสองส่วนข้างต้นมาวางแสดงตำแหน่งตามลำดับโจทย์]}$$
ตอบ: ค่าของพจน์นี้คือ $0$ $\quad \text{[รวมค่าผลบวกหักล้างทางพีชคณิตจำนวนเต็มระหว่างติดลบหนึ่งกับบวกหนึ่ง หักล้างกันหมดพอดีได้ค่าเป็น } 0\text{]}$$
(3) หาค่าของ: $$1^{2563} + (-1)^{2020}$$
วิธีทำ: (เริ่มต้นกระบวนการคิดโดยกระจายแยกคิดคำนวณเพื่อป้องกันความสับสนเรื่องเครื่องหมาย)
$$1^{2563} = 1 \quad \text{[เนื่องจากฐานเป็นเลข 1 ไม่ว่าจะคูณตัวเองซ้ำกี่หมื่นกี่แสนครั้ง ค่าของมันก็จะคงเดิมเท่ากับ 1 เสมอ]}$$
$$(-1)^{2020} = 1 \quad \text{[พิจารณาตัวเลขชี้กำลัง } 2020 \text{ เป็นเลขคู่ก้อนใหญ่ ส่งผลให้ฐานลบคูณลบสลับกันจนกลายเป็นบวกหนึ่ง]}$$
$$= 1 + 1 \quad \text{[หยิบตัวเลขผลลัพธ์องค์ประกอบย่อยที่แกะได้ทั้งสองซีกมาตั้งสมการรวมยอดการคำนวณตามเครื่องหมาย]}$$
ตอบ: ผลลัพธ์รวมคือ $2$ $\quad \text{[ทำเรื่องง่ายให้ง่ายโดยรวมผลบวกตัวเลขธรรมดาสุดท้ายจนได้คำตอบที่สมบูรณ์เท่ากับ } 2\text{]}$$
(4) หาค่าของ: $$(-3.14)^0 + (-1)^{13}$$
วิธีทำ: (เริ่มต้นแยกส่วนวิเคราะห์โจทย์ข้อสอบซึ่งประกอบด้วยเลขทศนิยมและจำนวนเต็มติดลบ)
$$(-3.14)^0 = 1 \quad \text{[ฐานทศนิยมติดลบก้อนนี้ถูกคลุมด้วยกำลังศูนย์ จึงลัดค่าลดรูปเป็นหนึ่งทันทีตามข้อตกลงพีชคณิต]}$$
$$(-1)^{13} = -1 \quad \text{[เนื่องจากเลขชี้กำลังสิบสามเป็นกลุ่มเลขคี่ ผลลัพธ์การคูณจึงยังคงรักษาลักษณะความเป็นลบไว้เช่นเดิม]}$$
$$= 1 + (-1) \quad \text{[นำส่วนโครงสร้างตัวเลขแทนค่ารวมที่ได้ในสองขั้นตอนก่อนหน้ามาประกอบร่างคำนวณตามคำสั่งโจทย์]}$$
ตอบ: ผลรวมสุทธิคือ $0$ $\quad \text{[หักล้างยอดคณิตศาสตร์มูลค่าจำนวนบวกหนึ่งกับจำนวนลบหนึ่ง ยอดรวมหักล้างกันสิ้นซากได้ผลลัพธ์เป็น } 0\text{]}$$

(1) รูปอย่างง่ายของ: $$\left( 3^{-2} \times 17^8 \right)^0$$

(2) รูปอย่างง่ายของ: $$2^5 \times 4^{-3} \times \left( 32^{-2} \times 16 \right)^{-1}$$

$$= 2^5 \times (2^2)^{-3} \times \left( (2^5)^{-2} \times 2^4 \right)^{-1} \quad \text{[ปรับเปลี่ยนระบบตัวเลขฐานให้เป็นฐาน 2 เหมือนกันทั้งหมด โดยแปลง } 4=2^2 \text{, } 32=2^5 \text{ และ } 16=2^4\text{]}$$
$$= 2^5 \times 2^{-6} \times \left( 2^{-10} \times 2^4 \right)^{-1} \quad \text{[ระเบิดกำลังซ้อนกันโดยคูณเลขกำลังเข้าหากันตามกฎคณิตศาสตร์ โดยได้ผลคูณเท่ากับ } 2 \times (-3) = -6 \text{ และ } 5 \times (-2) = -10\text{]}$$
$$= 2^{5 - 6} \times \left( 2^{-6} \right)^{-1} \quad \text{[รวมฐานสองก้อนแรกโดยเอาเลขกำลังมาบวกกัน ส่วนกลุ่มก้อนในวงเล็บหลังนำชี้กำลังมารวมกันได้ } -10 + 4 = -6\text{]}$$
$$= 2^{-1} \times 2^6 \quad \text{[คำนวณหักล้างเลขชี้กำลังพจน์หน้าได้ } -1 \text{ พจน์หลังคูณกำลังซ้อนนอกวงเล็บกระจายเข้าไปได้ผลคูณ } -6 \times (-1) = 6\text{]}$$
$$= 2^{-1 + 6} \quad \text{[ยุบฐานเลขสองตัวสุดท้ายที่ตั้งคูณกันอยู่เข้าหากันโดยรวบรวมนำเอาเลขชี้กำลังมาตั้งสมการคำนวณบวกกันตามกฎ]}$$
$$= 2^5 \quad \text{[หักล้างตัวเลขดัชนีชี้กำลังขั้นสุดท้ายให้เรียบร้อยสมบูรณ์จนได้คำตอบเลขชี้กำลังออกมาเป็นจำนวนเต็มบวกคือ } 5\text{]}$$
ตอบ: รูปอย่างง่ายคือ $2^5$ (หรือ $32$) $\quad \text{[เขียนแสดงผลลัพธ์คณิตศาสตร์มาตรฐานสุดท้าย หรือสามารถแจงค่าเต็มได้เท่ากับ } 2 \times 2 \times 2 \times 2 \times 2 = 32\text{]}$$
(3) รูปอย่างง่ายของ: $$\frac{2^2 \times 10^3}{5^4 \times 2^5}$$

$$= \frac{2^2 \times (2 \times 5)^3}{5^4 \times 2^5} \quad \text{[จัดการแยกวิเคราะห์เลขฐานสิบในส่วนเศษด้านบน โดยแปลงแตกตัวประกอบให้กระจายร่างเป็นฐาน } 2 \times 5\text{]}$$
$$= \frac{2^2 \times 2^3 \times 5^3}{5^4 \times 2^5} \quad \text{[กระจายสัญลักษณ์เลขยกกำลังสามที่อยู่ภายนอกเข้าไปคุมตัวประกอบย่อยทุกพจน์ด้านในวงเล็บตามสมบัติ]}$$
$$= \frac{2^{2+3} \times 5^3}{2^5 \times 5^4} \quad \text{[ดำเนินการจัดกลุ่มยุบรวมตัวเลขฐาน 2 ของกลุ่มตัวเศษด้านบนเข้าด้วยกันโดยจับเลขกำลังมาบวกกัน]}$$
$$= \frac{2^5 \times 5^3}{2^5 \times 5^4} \quad \text{[คำนวณรวบผลยอดเลขชี้กำลังกลุ่มตัวเลขฐาน 2 ของเศษด้านบน ได้ผลรวมยอดคือ } 2 + 3 = 5\text{]}$$
$$= 2^{5-5} \times 5^{3-4} \quad \text{[คำนวณหักล้างเลขฐานเศษส่วนส่วนบนกับส่วนล่าง โดยนำเลขกำลังของส่วนล่างขยับดันขึ้นมาลบตามสูตรหลักการหาร]}$$
$$= 2^0 \times 5^{-1} \quad \text{[คำนวณเคลียร์ยอดเลขกำลังพจน์หน้าได้กำลังศูนย์ และคำนวณคิดเลขกำลังพจน์ฐานห้าได้ผลลัพธ์เป็นกำลังติดลบหนึ่ง]}$$
$$= 1 \times 5^{-1} \quad \text{[ลดรูปเปลี่ยนคุณสมบัติของค่าพจน์คณิตศาสตร์ โดยถอดรูปเปลี่ยนสัญลักษณ์โครงสร้าง } 2^0 \text{ ปรับรูปให้กลายเป็นเลข } 1\text{]}$$
$$= 5^{-1} \quad \text{[คุณสมบัติการคูณด้วยเลขเอกลักษณ์หนึ่งส่งผลให้พจน์คณิตศาสตร์คงเหลือระบบตัวเลขเท่าเดิมไม่เปลี่ยนค่า]}$$
ตอบ: รูปอย่างง่ายคือ $$\frac{1}{5}$$ $\quad \text{[แปลงเปลี่ยนเลขชี้กำลังที่ติดลบอยู่ ดันโยกลงไปเป็นส่วนตัวหารด้านล่างเพื่อผันเครื่องหมายให้เป็นจำนวนเต็มบวกตามกฎ]}$$
(4) รูปอย่างง่ายของ: $$\frac{3^{-7} \times 2^{-3}}{8^0 \times 3^{-7}}$$
วิธีทำ: (เริ่มต้นการสังเกตความสัมพันธ์และพจน์ร่วมที่สามารถลดทอนรูปแบบได้อย่างรวดเร็ว)
$$\text{ตรวจสอบพจน์ร่วมของระบบเศษส่วนทั้งส่วนบนและส่วนล่าง พบว่ามีโครงสร้างสัญลักษณ์ } 3^{-7} \text{ หน้าตาเหมือนกันทุกประการ}$$
$$= \frac{2^{-3}}{8^0} \quad \text{[ดำเนินการตัดทอนตัดเลขพจน์ร่วมที่หน้าตาซ้ำกันทิ้งไปทั้งเศษส่วนด้วยกระบวนการหารร่วมลดรูปเหลือพจน์เดี่ยว]}$$
$$= \frac{2^{-3}}{1} \quad \text{[คำนวณวิเคราะห์ค่าคุณสมบัติเฉพาะของตัวเลขส่วนฐานล่างด้านใต้เศษส่วน โดยถอดสัญลักษณ์เปลี่ยนจากค่า } 8^0 = 1\text{]}$$
$$= 2^{-3} \quad \text{[สิ่งใดก็ตามที่ถูกหารด้วยเลขหนึ่ง ระบบตัวเลขจะมีค่าคงที่เดิมไม่ส่งผลเปลี่ยนแปลงโครงสร้างเชิงมูลค่าตามหลักคณิตศาสตร์]}$$
ตอบ: ปรับกำลังลบลงมาเป็นเศษส่วนได้ผลลัพธ์คือ $$\frac{1}{2^3} \quad \text{(หรือ } \frac{1}{8}\text{)}$$ $\quad \text{[ดันโยกเลขยกกำลังลบลงมาทำหน้าที่เป็นตัวส่วนด้านล่างเพื่อแก้ปัญหาเลขกำลังติดลบให้เป็นจำนวนเต็มบวกตามนิยามและแจงค่าถอดได้แปด]}$$
(5) รูปอย่างง่ายของ: $$\frac{(-18)^{-3} \times 4^5}{(-27)^2 \times 6^{-3}}$$
วิธีทำ: (เริ่มต้นโดยการวิเคราะห์จัดการเครื่องหมายลบแยกออกจากก้อนเลขฐานเชิงซ้อน)
$$= \frac{-(2 \times 3^2)^{-3} \times (2^2)^5}{(3^3)^2 \times (2 \times 3)^{-3}} \quad \text{[กระจายฐานแยกเป็นเลขฐานจำนวนเฉพาะย่อยคือ 2 และ 3 โดยแปลง } 18=2 \times 3^2 \text{, } 4=2^2 \text{, } -27 \text{ ยกกำลังเลขคู่กลายเป็นบวก } 3^3 \text{ และ } 6=2 \times 3\text{]}$$
$$= \frac{-(2^{-3} \times 3^{-6}) \times 2^{10}}{3^6 \times 2^{-3} \times 3^{-3}} \quad \text{[คูณกระจายแจกแจงพจน์เลขยกกำลังซ้อนนอกวงเล็บทั้งหมดทะลวงเข้าไปบดขยี้เลขกำลังด้านในทุกๆ พจน์ตามระเบียบสมบัติ]}$$
$$= \frac{-2^{-3+10} \times 3^{-6}}{2^{-3} \times 3^{6-3}} \quad \text{[รวมฐานตัวแปรที่หน้าตาซ้ำกันในฝั่งเดียวกัน โดยด้านบนรวมกำลังฐานสอง และด้านล่างรวมกลุ่มกำลังของฐานสามเข้าหากัน]}$$
$$= \frac{-2^7 \times 3^{-6}}{2^{-3} \times 3^3} \quad \text{[คำนวณสรุปผลบวกเลขชี้กำลังกลุ่มตัวแปรย่อย โดยพจน์เศษด้านบนคำนวณได้บวกเจ็ด ส่วนพจน์ส่วนด้านล่างคำนวณผลรวมได้บวกสาม]}$$
$$= -2^{7 - (-3)} \times 3^{-6 - 3} \quad \text{[ใช้ระบบการลบพจน์เลขยกกำลังของการหารฐานตัวเลขร่วมกัน โดยนำเอาเลขชี้กำลังของตัวล่างด้านใต้ขยับดีดขึ้นไปตั้งกระบวนการลบพจน์บน]}$$
$$= -2^{10} \times 3^{-9} \quad \text{[คำนวณยอดเงินเลขกำลังสุทธิ โดยฝั่งฐานสองเกิดลบเจอลบลดรูปเป็นบวกคำนวณได้สิบ ส่วนฝั่งฐานสามลบซ้อนลบรวมได้ลบเก้า]}$$
ตอบ: จัดรูปเลขชี้กำลังลบให้เป็นบวกตามกฎ $$-\frac{2^{10}}{3^9}$$ $\quad \text{[ดึงสัญลักษณ์ตัวแปรฐานสามซึ่งแบกเลขชี้กำลังติดลบอยู่ โยกตัวลงไปทำหน้าที่เป็นส่วนด้านใต้เศษส่วนเพื่อปรับเปลี่ยนค่าตัวยกกำลังให้เป็นบวกตามหลักกฎคณิตศาสตร์]}$
(6) รูปอย่างง่ายของ: $$\frac{15^3 \times (-9)^{-3}}{(-5)^5 \times 27^4}$$
วิธีทำ: (เริ่มต้นกระบวนการแยกชำแหละเลขฐานผสมเชิงซ้อนและเคลียร์ปัญหาเรื่องพจน์สัญลักษณ์ติดลบ)
$$= \frac{(3 \times 5)^3 \times (-1 \times 3^2)^{-3}}{(-1 \times 5)^5 \times (3^3)^4} \quad \text{[แยกกระจายพจน์จำนวนเฉพาะเพื่อสร้างกลุ่มฐานเดียวกัน โดยแปลงตัวเลข } 15=3 \times 5 \text{, } -9=-1 \times 3^2 \text{, } -5=-1 \times 5 \text{ และ } 27=3^3\text{]}$$
$$= \frac{3^3 \times 5^3 \times (-1)^{-3} \times 3^{-6}}{(-1)^5 \times 5^5 \times 3^{12}} \quad \text{[คูณแจกแจงเลขกำลังรวมวงเล็บด้านนอกเข้าสู่ด้านในวงเล็บของทุกพจน์ตามคุณสมบัติการกระจายกำลังพีชคณิต]}$$
$$= \frac{3^3 \times 5^3 \times (-1) \times 3^{-6}}{(-1) \times 5^5 \times 3^{12}} \quad \text{[ถอดวิเคราะห์ค่าคงที่ลบหนึ่ง โดยค่าลบหนึ่งยกกำลังคี่จะคงเหลือค่าเท่ากับลบหนึ่งเท่าเดิมทั้งชุดตัวเศษด้านบนและชุดตัวส่วนด้านล่าง]}$$
$$= \frac{3^{3-6} \times 5^3}{3^{12} \times 5^5} \quad \text{[เครื่องหมายค่าลบหนึ่งของเศษส่วนทั้งบนและล่างตั้งหารกันอยู่จึงหักล้างตัดทอนเป็นบวกหายไปหมดสิ้น และรวบฐานสามด้านบนเข้าหากัน]}$$
$$= \frac{3^{-3} \times 5^3}{3^{12} \times 5^5} \quad \text{[คำนวณเคลียร์ผลบวกคณิตศาสตร์พจน์เลขชี้กำลังกลุ่มฐานสามของตัวเศษด้านบน โดยคำนวณหักล้างยอดได้พจน์ยกกำลังเท่ากับลบสาม]}$$
$$= 3^{-3-12} \times 5^{3-5} \quad \text{[ยุบรวมจัดการฐานตัวเลขเศษส่วนแนวตั้งข้ามฟาก โดยนำเลขชี้กำลังของตัวส่วนด้านล่างยกขึ้นไปตั้งกระบวนการลบพจน์กำลังด้านบน]}$$
$$= 3^{-15} \times 5^{-2} \quad \text{[คำนวณหักล้างสรุปผลรวมตัวเลขดัชนีชี้กำลัง โดยส่วนฐานสามได้ลบสิบห้า และตัวเลขส่วนฐานห้าคำนวณผลลัพธ์ยอดรวมได้ลบสอง]}$$
ตอบ: ยุบกำลังที่ติดลบลงไปเป็นตัวส่วนเพื่อทำกำลังให้เป็นบวกได้ $$\frac{1}{3^{15} \cdot 5^2}$$ $\quad \text{[ดันโยกเลขยกกำลังทั้งหมดที่กำลังติดลบอยู่ โยกลงมาจับกลุ่มคูณกันทำหน้าที่เป็นตัวส่วนด้านใต้เศษส่วนเพื่อปรับผันเครื่องหมายกำลังให้เป็นบวกตามกฎมาตรฐานคณิตศาสตร์]}$
