เอกสารประกอบการเรียน เรื่องเลขยกกำลัง ชั้นมัธยมศึกษาปีที่ 5 หน้า 10

1.2 รากที่ n ของจำนวนจริง

บทเรียนเกี่ยวกับคุณสมบัติ บทนิยาม และทฤษฎีบทของรากที่ n ในระบบจำนวนจริง

รากที่ n ของจำนวนจริง เป็นหนึ่งในบทเรียนคณิตศาสตร์ระดับมัธยมปลาย (ม.5) ที่สร้างความปวดหัวให้กับนักเรียนหลายคน เพราะดูเป็นนามธรรมและเต็มไปด้วยสัญลักษณ์แปลกๆ เช่น √nx

เรามาดูกันแบบเข้าใจง่ายๆ ว่ามันคืออะไร และทำไมระบบการศึกษาถึงกำหนดให้เด็กๆ ต้องเรียนเรื่องนี้ครับ

1. รากที่ n ของจำนวนจริง คืออะไร?

ถ้าจะอธิบายให้เห็นภาพง่ายที่สุด "การหารากที่ n คือกระบวนการย้อนกลับของการยกกำลัง n"

  • การยกกำลัง คือการถามว่า: "เอาตัวเลขนี้มาคูณตัวเอง n ครั้ง แล้วจะได้ผลลัพธ์เท่าไหร่?"
    เช่น 3 ยกกำลัง 2 (32) คือ 3 × 3 = 9
  • การหาราก (Root) คือการถามกลับกันว่า: "เลขอะไรที่คูณตัวเอง n ครั้ง แล้วได้ผลลัพธ์เท่ากับเลขที่กำหนดให้?"
    เช่น รากที่ 2 ของ 9 คือ เลขอะไรที่คูณกัน 2 ครั้งแล้วได้ 9? คำตอบคือ 3 (และ -3 เพราะ (-3) × (-3) = 9 เช่นกัน)
นิยามตามหลักคณิตศาสตร์:

ให้ x และ y เป็นจำนวนจริง y จะเป็นรากที่ n ของ x ก็ต่อเมื่อ yn = x

พอขยายไปเป็น "รากที่ n" (รากที่ 3, 4, 5, ...) วิธีคิดก็ยังเหมือนเดิม:

  • รากที่ 3 ของ 8 คือ 2 (เพราะ 2 × 2 × 2 = 8)
  • รากที่ 4 ของ 81 คือ 3 และ -3 (เพราะ 34 = 81 และ (-3)4 = 81)

2. ทำไมเด็กนักเรียนต้องเรียนเรื่องนี้? (ความสำคัญและประโยชน์)

คำถามที่ว่า "เรียนไปทำไม ไม่เห็นได้ใช้ในชีวิตประจำวันเลย" จะหมดไป ถ้าเราเข้าใจว่ารากที่ n เป็นรากฐานของนวัตกรรมและเทคโนโลยีรอบตัวเรา ดังนี้ครับ:

  1. มันคือเครื่องมือแก้ปัญหาในมิติที่ซับซ้อนขึ้น (Geometry & Physics)
    ในโลกความจริง เราไม่ได้เจอแค่เส้นตรง (1 มิติ) แต่เราเจอพื้นที่ (2 มิติ) และปริมาตร (3 มิติ) ถ้าต้องการสร้างกล่องทรงลูกบาศก์ ให้มีความจุ (ปริมาตร) 1,000 ลูกบาศก์เซนติเมตรพอดี คำถามคือต้องตัดแผ่นไม้ให้มีความกว้าง ยาว สูง ด้านละกี่เซนติเมตร? เราต้องใช้ รากที่ 3 ในการหาคำตอบ: √31000 = 10 เซนติเมตร
  2. เป็นรากฐานของสูตรคำนวณใน "สถาปัตยกรรม" และ "วิศวกรรม"
    วิศวกรโครงสร้างใช้รากที่สองและรากที่ n ในการคำนวณแรงบิด แรงต้านทานของคอนกรีต และการกระจายน้ำหนักของสะพานหรือตึกสูง หากไม่มีคณิตศาสตร์เรื่องนี้ การคำนวณเพื่อความปลอดภัยของสิ่งก่อสร้างจะทำไม่ได้เลย
  3. ใช้ในระบบ "ดอกเบี้ยทบต้น" และ "การเงิน" (Finance)
    สูตรการคิดดอกเบี้ยเงินฝากหรือการเติบโตของประชากรเป็นฟังก์ชันเอกซ์โพเนนเชียล (การยกกำลัง) ดังนั้น หากนักวิเคราะห์การเงินต้องการคำนวณย้อนกลับว่า "ถ้าอยากให้เงินก้อนนี้โตขึ้นเป็น 2 เท่าในอีก 5 ปี ข้างหน้า จะต้องหาการลงทุนที่ได้ผลตอบแทนปีละกี่เปอร์เซ็นต์?" เครื่องมือที่จะช่วยถอดสมการนี้ออกมาได้ก็คือ รากที่ 5 นั่นเอง
  4. เทคโนโลยีคอมพิวเตอร์และการประมวลผลภาพ (Computer Science & CGI)
    ภาพกราฟิก 3D ในเกม, เอฟเฟกต์ในภาพยนตร์ (CGI), หรือแม้กระทั่งอัลกอริทึมในการปรับความสว่างของหน้าจอโทรศัพท์ (Gamma Correction) ล้วนใช้สมการที่มีเรื่อง "รากที่ n" และ "เลขยกกำลัง" ทำงานอยู่เบื้องหลังในระดับโค้ด เพื่อคำนวณการหักเหของแสงและสีพิกเซล
  5. ฝึกทักษะการคิดเชิงตรรกะแบบย้อนกลับ (Reverse Engineering)
    ในแง่ของพัฒนาการทางสมอง การฝึกทำโจทย์เรื่องนี้ช่วยให้เด็กนักเรียนได้ฝึก "คิดย้อนกลับ" จากผลลัพธ์ไปหาต้นเหตุ ซึ่งเป็นทักษะสำคัญมากในการแก้ปัญหาขั้นสูง (Problem Solving) ไม่ว่าจะเป็นการสืบสวน การวางแผนธุรกิจ หรือการดีบั๊ก (Debug) ซอฟต์แวร์

สรุป

รากที่ n ไม่ใช่แค่ตัวเลขในกระดาษสอบ แต่เป็น "ภาษาและเครื่องมือ" ที่มนุษย์สร้างขึ้นเพื่ออธิบายและควบคุมสิ่งต่างๆ ในธรรมชาติที่มีความสัมพันธ์แบบทวีคูณ การที่เด็กๆ ได้เรียนเรื่องนี้ จึงเป็นการเปิดประตูสู่การเข้าใจศาสตร์ชั้นสูง ทั้งวิทยาศาสตร์ วิศวกรรมศาสตร์ และการเงินในอนาคตครับ

1.2.1 รากที่สองของจำนวนจริง

บทนิยาม ให้ x และ y เป็นจำนวนจริง
y เป็นรากที่สองของ x ก็ต่อเมื่อ y2 = x

หมายเหตุ

  • เนื่องจากกำลังของจำนวนจริงใดๆ ต้องมากกว่าหรือเท่ากับ 0 ดังนั้นจะมีรากที่สองของจำนวนจริงบวกหรือศูนย์เท่านั้น นั่นคือ x ในนิยามข้างต้นจะมากกว่าหรือเท่ากับศูนย์เท่านั้น
    • เช่น 3 เป็นรากที่สองของ 9 เพราะ 32 = 9
    • -½ เป็นรากที่สองของ ¼ เพราะ (-½)2 = ¼
    • 0 เป็นรากที่สองของ 0 เพราะ 02 = 0
  • ถ้า x ≥ 0 แล้ว x จะมีรากที่สองที่มากกว่าหรือเท่ากับศูนย์เสมอ เรียกรากนี้ว่า รากที่สองที่ไม่เป็นจำนวนจริงลบ ของ x และแทนด้วยสัญลักษณ์ √x โดยเครื่องหมาย √ เรียกว่า เครื่องหมายกรณฑ์ (radical sign)
  • สำหรับ x > 0 จะมีรากที่สองของ x สองราก คือ √x และ -√x โดยที่ √x เป็นจำนวนจริงบวก และ -√x เป็นจำนวนจริงลบ
    ถ้า x = 0 แล้วจะมีจำนวนจริงจำนวนเดียว คือ 0 เป็นรากที่สองของ x นั่นคือ √0 = 0
    ถ้า x < 0 แล้วจะไม่มีรากที่สองของ x ที่เป็นจำนวนจริง
  • √x = y เมื่อ x ≥ 0 หมายความว่า y2 = x และ y ≥ 0
ทฤษฎีบท ให้ x ≥ 0 และ y ≥ 0 จะได้
√x · √y = √xy
√x / √y = √(x/y) เมื่อ y ≠ 0
