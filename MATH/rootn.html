เอกสารประกอบการเรียน เรื่องเลขยกกำลัง ชั้นมัธยมศึกษาปีที่ 5 หน้า 10

1.2 รากที่ n ของจำนวนจริง

บทเรียนเกี่ยวกับคุณสมบัติ บทนิยาม และทฤษฎีบทของรากที่ n ในระบบจำนวนจริง

รากที่ n ของจำนวนจริง เป็นหนึ่งในบทเรียนคณิตศาสตร์ระดับมัธยมปลาย (ม.5) ที่สร้างความปวดหัวให้กับนักเรียนหลายคน เพราะดูเป็นนามธรรมและเต็มไปด้วยสัญลักษณ์แปลกๆ เช่น √nx

เรามาดูกันแบบเข้าใจง่ายๆ ว่ามันคืออะไร และทำไมระบบการศึกษาถึงกำหนดให้เด็กๆ ต้องเรียนเรื่องนี้ครับ

1. รากที่ n ของจำนวนจริง คืออะไร?

ถ้าจะอธิบายให้เห็นภาพง่ายที่สุด "การหารากที่ n คือกระบวนการย้อนกลับของการยกกำลัง n"

  • การยกกำลัง คือการถามว่า: "เอาตัวเลขนี้มาคูณตัวเอง n ครั้ง แล้วจะได้ผลลัพธ์เท่าไหร่?"
    เช่น 3 ยกกำลัง 2 (32) คือ 3 × 3 = 9
  • การหาราก (Root) คือการถามกลับกันว่า: "เลขอะไรที่คูณตัวเอง n ครั้ง แล้วได้ผลลัพธ์เท่ากับเลขที่กำหนดให้?"
    เช่น รากที่ 2 ของ 9 คือ เลขอะไรที่คูณกัน 2 ครั้งแล้วได้ 9? คำตอบคือ 3 (และ -3 เพราะ (-3) × (-3) = 9 เช่นกัน)
นิยามตามหลักคณิตศาสตร์:

ให้ x และ y เป็นจำนวนจริง y จะเป็นรากที่ n ของ x ก็ต่อเมื่อ yn = x

พอขยายไปเป็น "รากที่ n" (รากที่ 3, 4, 5, ...) วิธีคิดก็ยังเหมือนเดิม:

  • รากที่ 3 ของ 8 คือ 2 (เพราะ 2 × 2 × 2 = 8)
  • รากที่ 4 ของ 81 คือ 3 และ -3 (เพราะ 34 = 81 และ (-3)4 = 81)

2. ทำไมเด็กนักเรียนต้องเรียนเรื่องนี้? (ความสำคัญและประโยชน์)

คำถามที่ว่า "เรียนไปทำไม ไม่เห็นได้ใช้ในชีวิตประจำวันเลย" จะหมดไป ถ้าเราเข้าใจว่ารากที่ n เป็นรากฐานของนวัตกรรมและเทคโนโลยีรอบตัวเรา ดังนี้ครับ:

  1. มันคือเครื่องมือแก้ปัญหาในมิติที่ซับซ้อนขึ้น (Geometry & Physics)
    ในโลกความจริง เราไม่ได้เจอแค่เส้นตรง (1 มิติ) แต่เราเจอพื้นที่ (2 มิติ) และปริมาตร (3 มิติ) ถ้าต้องการสร้างกล่องทรงลูกบาศก์ ให้มีความจุ (ปริมาตร) 1,000 ลูกบาศก์เซนติเมตรพอดี คำถามคือต้องตัดแผ่นไม้ให้มีความกว้าง ยาว สูง ด้านละกี่เซนติเมตร? เราต้องใช้ รากที่ 3 ในการหาคำตอบ: √31000 = 10 เซนติเมตร
  2. เป็นรากฐานของสูตรคำนวณใน "สถาปัตยกรรม" และ "วิศวกรรม"
    วิศวกรโครงสร้างใช้รากที่สองและรากที่ n ในการคำนวณแรงบิด แรงต้านทานของคอนกรีต และการกระจายน้ำหนักของสะพานหรือตึกสูง หากไม่มีคณิตศาสตร์เรื่องนี้ การคำนวณเพื่อความปลอดภัยของสิ่งก่อสร้างจะทำไม่ได้เลย
  3. ใช้ในระบบ "ดอกเบี้ยทบต้น" และ "การเงิน" (Finance)
    สูตรการคิดดอกเบี้ยเงินฝากหรือการเติบโตของประชากรเป็นฟังก์ชันเอกซ์โพเนนเชียล (การยกกำลัง) ดังนั้น หากนักวิเคราะห์การเงินต้องการคำนวณย้อนกลับว่า "ถ้าอยากให้เงินก้อนนี้โตขึ้นเป็น 2 เท่าในอีก 5 ปี ข้างหน้า จะต้องหาการลงทุนที่ได้ผลตอบแทนปีละกี่เปอร์เซ็นต์?" เครื่องมือที่จะช่วยถอดสมการนี้ออกมาได้ก็คือ รากที่ 5 นั่นเอง
  4. เทคโนโลยีคอมพิวเตอร์และการประมวลผลภาพ (Computer Science & CGI)
    ภาพกราฟิก 3D ในเกม, เอฟเฟกต์ในภาพยนตร์ (CGI), หรือแม้กระทั่งอัลกอริทึมในการปรับความสว่างของหน้าจอโทรศัพท์ (Gamma Correction) ล้วนใช้สมการที่มีเรื่อง "รากที่ n" และ "เลขยกกำลัง" ทำงานอยู่เบื้องหลังในระดับโค้ด เพื่อคำนวณการหักเหของแสงและสีพิกเซล
  5. ฝึกทักษะการคิดเชิงตรรกะแบบย้อนกลับ (Reverse Engineering)
    ในแง่ของพัฒนาการทางสมอง การฝึกทำโจทย์เรื่องนี้ช่วยให้เด็กนักเรียนได้ฝึก "คิดย้อนกลับ" จากผลลัพธ์ไปหาต้นเหตุ ซึ่งเป็นทักษะสำคัญมากในการแก้ปัญหาขั้นสูง (Problem Solving) ไม่ว่าจะเป็นการสืบสวน การวางแผนธุรกิจ หรือการดีบั๊ก (Debug) ซอฟต์แวร์

สรุป

รากที่ n ไม่ใช่แค่ตัวเลขในกระดาษสอบ แต่เป็น "ภาษาและเครื่องมือ" ที่มนุษย์สร้างขึ้นเพื่ออธิบายและควบคุมสิ่งต่างๆ ในธรรมชาติที่มีความสัมพันธ์แบบทวีคูณ การที่เด็กๆ ได้เรียนเรื่องนี้ จึงเป็นการเปิดประตูสู่การเข้าใจศาสตร์ชั้นสูง ทั้งวิทยาศาสตร์ วิศวกรรมศาสตร์ และการเงินในอนาคตครับ

1.2.1 รากที่สองของจำนวนจริง

บทนิยาม ให้ x และ y เป็นจำนวนจริง
y เป็นรากที่สองของ x ก็ต่อเมื่อ y2 = x

หมายเหตุ

  • เนื่องจากกำลังของจำนวนจริงใดๆ ต้องมากกว่าหรือเท่ากับ 0 ดังนั้นจะมีรากที่สองของจำนวนจริงบวกหรือศูนย์เท่านั้น นั่นคือ x ในนิยามข้างต้นจะมากกว่าหรือเท่ากับศูนย์เท่านั้น
    • เช่น 3 เป็นรากที่สองของ 9 เพราะ 32 = 9
    • -½ เป็นรากที่สองของ ¼ เพราะ (-½)2 = ¼
    • 0 เป็นรากที่สองของ 0 เพราะ 02 = 0
  • ถ้า x ≥ 0 แล้ว x จะมีรากที่สองที่มากกว่าหรือเท่ากับศูนย์เสมอ เรียกรากนี้ว่า รากที่สองที่ไม่เป็นจำนวนจริงลบ ของ x และแทนด้วยสัญลักษณ์ √x โดยเครื่องหมาย √ เรียกว่า เครื่องหมายกรณฑ์ (radical sign)
  • สำหรับ x > 0 จะมีรากที่สองของ x สองราก คือ √x และ -√x โดยที่ √x เป็นจำนวนจริงบวก และ -√x เป็นจำนวนจริงลบ
    ถ้า x = 0 แล้วจะมีจำนวนจริงจำนวนเดียว คือ 0 เป็นรากที่สองของ x นั่นคือ √0 = 0
    ถ้า x < 0 แล้วจะไม่มีรากที่สองของ x ที่เป็นจำนวนจริง
  • √x = y เมื่อ x ≥ 0 หมายความว่า y2 = x และ y ≥ 0
ทฤษฎีบท ให้ x ≥ 0 และ y ≥ 0 จะได้
√x · √y = √xy
√x / √y = √(x/y) เมื่อ y ≠ 0
ครูจำนงค์ ซ้ายหุย และคณะ กลุ่มสาระการเรียนรู้คณิตศาสตร์ โรงเรียนกัลยาณีศรีธรรมราช

💡 พี่ "พระจอมเกล้าติวเตอร์" สรุปให้: เรื่องเลขยกกำลังและรากที่ n (ม.5)

สวัสดีครับน้อง ๆ ม.5 ทุกคน! วันนี้พี่พระจอมเกล้าติวเตอร์จะพามาลุยเนื้อหาสำคัญที่เป็นฐานรากของบทเลขยกกำลัง นั่นคือ "เรื่องรากที่ n และจำนวนที่อยู่ในรูปกรณฑ์" ครับ เรื่องนี้ถ้าน้องจับหลักการมองภาพรวมได้ จะทำข้อสอบได้ไวมาก ไม่ว่าจะเป็นข้อสอบในโรงเรียน หรือสนามใหญ่อย่าง A-Level ครับ

💡 ทริคการจำ (Memory Tricks)

  • "รากที่สอง" VS "เครื่องหมายกรณฑ์ (Root)" : คำว่า "รากที่สองของ a" จะต้องมีคำตอบทั้งค่าบวกและค่าลบเสมอ (ยกเว้นรากของ 0) เพราะจำนวนลบยกกำลังสองก็ย่อมได้ค่าบวก เช่น รากที่สองของ 25 คือ $\pm 5$ แต่ถ้าเป็นเครื่องหมาย $\sqrt{25}$ จะหมายถึงรากที่สองที่เป็นบวกเท่านั้น ซึ่งก็คือ $5$ ครับ จำไว้เลยว่า "รากที่สองมีสองค่า แต่สัญลักษณ์รูทมีค่าเดียว"
  • "ถอดรูทคู่ต้องสวม Absolute" : เมื่อเราถอดรากที่สองของตัวแปร $\sqrt{x^2}$ ผลลัพธ์ที่ได้ต้องเป็น $|x|$ เสมอ เพื่อป้องกันไม่ให้ค่าติดลบ เพราะผลลัพธ์จากเครื่องหมายรูทหลักต้องเป็นบวกหรือศูนย์เท่านั้น
  • "การสังยุค (Conjugate) เปลี่ยนลบเป็นบวก เปลี่ยนบวกเป็นลบ" : ถ้าส่วนติดรูทในรูป $\sqrt{b} + \sqrt{c}$ ให้คูณด้วยคู่คอนจุเกตคือ $\sqrt{b} - \sqrt{c}$ ทั้งเศษและส่วน เพื่อใช้สูตรผลต่างกำลังสอง $(A+B)(A-B) = A^2 - B^2$ รูทที่ส่วนจะหายวับไปทันทีครับ!

🧠 กระบวนการคิด/Step การมองโจทย์ (Thought Processes)

  1. Step 1: เช็คเงื่อนไขโจทย์ก่อนลุย ดูว่าโจทย์ถามคำว่า "รากที่สอง" (ตอบ 2 ค่า บวกลบ) หรือถามค่าของ "เครื่องหมายกรณฑ์" (ตอบค่าตามเครื่องหมาย)
  2. Step 2: จัดรูปตัวเลขฐาน ถ้าเจอเลขเยอะ ๆ ให้อยู่ในรูปตัวเลขยกกำลังต่ำสุด (เช่น $200 = 100 \times 2 = 10^2 \times 2$) เพื่อจะได้ดึงตัวร่วมออกจากรูทได้ง่าย
  3. Step 3: ทำกรณฑ์ให้เป็นอันดับเดียวกัน ในการบวกลบคูณหารกรณฑ์ สิ่งสำคัญคือ "อันดับของกรณฑ์" ต้องเท่ากัน และเลขข้างในต้องเหมือนกันจึงจะบวกลบกันได้เหมือนเอกนามทั่วไป

เอกสารประกอบการเรียน เรื่องเลขยกกำลัง ชั้นมัธยมศึกษาปีที่ 5 หน้า 11

ตัวอย่าง 1 จงหาคำตอบแต่ละข้อต่อไปนี้

(1) รากที่สองของ $25$ คือ ............................................................................

วิธีทำ: หาจำนวน $x$ ที่ยกกำลังสองแล้วได้ $25$ นั่นคือ $x^2 = 25$ จะได้ $x = \sqrt{25}$ และ $x = -\sqrt{25}$ ดังนั้นคำตอบคือ $5$ และ $-5$ (หรือ $\pm 5$)
[ทริค/กฎเหล็ก]: อ้างอิงบทนิยาม "รากที่สองของ $a$ เมื่อ $a \ge 0$ คือจำนวนที่ยกกำลังสองแล้วได้ $a$" ดังนั้นต้องมีทั้งค่าบวกและค่าลบ จำง่าย ๆ โจทย์ถามเป็นภาษาไทยว่า "รากที่สอง" ตอบ 2 ค่าเสมอ!

(2) รากที่สองของ $\frac{1}{16}$ คือ ............................................................................

วิธีทำ: ตั้งสมการ $x^2 = \frac{1}{16}$ ถอดรากได้ $x = \pm\sqrt{\frac{1}{16}} = \pm\frac{\sqrt{1}}{\sqrt{16}}$ จะได้คำตอบคือ $\frac{1}{4}$ และ $-\frac{1}{4}$ (หรือ $\pm\frac{1}{4}$)
[ทริค/กฎเหล็ก]: ใช้สมบัติกระจายรูทเศษส่วน $\sqrt{\frac{a}{b}} = \frac{\sqrt{a}}{\sqrt{b}}$ แยกคิดบนล่าง มองหาเลขยกกำลังสอง $\rightarrow 1^2=1$ และ $4^2=16$ และห้ามลืมใส่เครื่องหมายลบด้วยเพราะเป็นคำว่ารากที่สอง

(3) รากที่สองของ $3$ คือ ............................................................................

วิธีทำ: เนื่องจากไม่มีจำนวนเต็มใดที่ยกกำลังสองแล้วได้ $3$ ลงตัว จึงเขียนคำตอบให้อยู่ในรูปกรณฑ์ได้โดยตรง คือ $\sqrt{3}$ และ $-\sqrt{3}$ (หรือ $\pm\sqrt{3}$)
[ทริค/กฎเหล็ก]: ถ้าน้องเจอตัวเลขที่เป็นจำนวนเฉพาะหรือถอดรูทไม่ลงตัว ให้ติดเครื่องหมายกรณฑ์ (Root) ครอบเอาไว้ได้เลย โดยคงหลักการเดิมคือต้องตอบทั้งค่าบวกและค่าลบเสมอ

(4) รากที่สองของ $\frac{1}{2}$ คือ ............................................................................

วิธีทำ: เขียนคำตอบเบื้องต้นคือ $\pm\sqrt{\frac{1}{2}} = \pm\frac{1}{\sqrt{2}}$ จัดรูปคูณด้วย $\frac{\sqrt{2}}{\sqrt{2}}$ ทั้งเศษและส่วน จะได้คำตอบในรูปอย่างง่ายคือ $\frac{\sqrt{2}}{2}$ และ $-\frac{\sqrt{2}}{2}$ (หรือ $\pm\frac{\sqrt{2}}{2}$)
[ทริค/กฎเหล็ก]: กฎเหล็กทางคณิตศาสตร์ "ส่วนต้องไม่ติดรูท" ดังนั้นต้องเอาตัวมันเองคูณทั้งบนและล่าง เพื่อทำให้ส่วนเป็นจำนวนเต็มตามหลักของตัวหารที่มีเหตุผล (Rationalization)

(5) รากที่สองของ $0.04$ คือ ............................................................................

วิธีทำ: แปลงทศนิยมเป็นเศษส่วน $\rightarrow 0.04 = \frac{4}{100}$ ถอดรากที่สองจะได้ $\pm\sqrt{\frac{4}{100}} = \pm\frac{2}{10} = \pm 0.2$ ดังนั้นคำตอบคือ $0.2$ และ $-0.2$
[ทริค/กฎเหล็ก]: ทริคคิดลัดสำหรับทศนิยม: ถอดรากที่สองของตัวเลขหลักก่อน ($\sqrt{4} = 2$) จากนั้นจำนวนตำแหน่งทศนิยมของคำตอบจะลดลงครึ่งหนึ่ง เดิมมี 2 ตำแหน่ง คำตอบจึงเหลือ 1 ตำแหน่ง ได้เป็น $0.2$ ทันทีครับ

(6) รากที่สองของ $0.0025$ คือ ............................................................................

วิธีทำ: แปลงทศนิยมเป็นเศษส่วน $\rightarrow 0.0025 = \frac{25}{10000}$ ถอดรากที่สองจะได้ $\pm\sqrt{\frac{25}{10000}} = \pm\frac{5}{100} = \pm 0.05$ ดังนั้นคำตอบคือ $0.05$ และ $-0.05$
[ทริค/กฎเหล็ก]: ใช้ทริคลัดทศนิยม: มองเลข $25$ ถอดรูทได้ $5$ ทศนิยมต้นฉบับมี 4 ตำแหน่ง หารครึ่งจะเหลือ 2 ตำแหน่ง จึงเติมศูนย์ข้างหน้าได้เป็น $0.05$ สะดวกรวดเร็วและแม่นยำที่สุด

ตัวอย่าง 2 จงเขียนจำนวนต่อไปนี้ให้อยู่ในรูปอย่างง่าย

(1) $\sqrt{200}$ = ............................................................

วิธีทำ: แยกตัวประกอบภายในรูทให้มีกำลังสองสมบูรณ์ $\rightarrow \sqrt{200} = \sqrt{100 \times 2} = \sqrt{10^2 \times 2} = 10\sqrt{2}$
[ทริค/กฎเหล็ก]: ใช้สมบัติการแยกคูณของกรณฑ์ $\sqrt{a \cdot b} = \sqrt{a} \cdot \sqrt{b}$ พยายามแยกเอาเลขที่ถอดรูทลงตัวที่มากที่สุดออกมา (ในที่นี้คือ 100) จะทำให้จัดรูปเสร็จในบรรทัดเดียว

(2) $\sqrt{75}$ = ............................................................

วิธีทำ: แยกตัวประกอบของ $75$ ให้มีเลขยกกำลังสอง $\rightarrow \sqrt{75} = \sqrt{25 \times 3} = \sqrt{5^2 \times 3} = 5\sqrt{3}$
[ทริค/กฎเหล็ก]: มองหาแม่สูตรคูณหรือจำนวนกำลังสองสมบูรณ์ที่หาร $75$ ลงตัว นั่นคือ $25$ จากนั้นดึง $5$ ออกมานอกเครื่องหมายกรณฑ์ตามสมบัติ $\sqrt{A^2 \cdot B} = A\sqrt{B}$ (เมื่อ $A \ge 0$)

(3) $\frac{\sqrt{850}}{\sqrt{32}}$ = ............................................................

วิธีทำ: รวมรูทเข้าด้วยกันก่อนเพื่อตัดทอนเศษส่วน $\rightarrow \sqrt{\frac{850}{32}} = \sqrt{\frac{425}{16}} = \frac{\sqrt{425}}{\sqrt{16}} = \frac{\sqrt{25 \times 17}}{4} = \frac{5\sqrt{17}}{4}$
[ทริค/กฎเหล็ก]: ทริคสำคัญ: อย่าเพิ่งแยกถอดรูทแยกจากกันตั้งแต่แรก ให้ใช้สมบัติ $\frac{\sqrt{a}}{\sqrt{b}} = \sqrt{\frac{a}{b}}$ รวบเข้าหากันแล้วเอาแม่ 2 ตัดทอนตัวเลขให้เล็กลงก่อน จะคำนวณง่ายขึ้นมหาศาลครับ!

(4) $\sqrt{3} \times \sqrt{15}$ = ............................................................

วิธีทำ: ใช้การรวมกรณฑ์เพื่อหาคู่จับกลุ่ม $\rightarrow \sqrt{3} \times \sqrt{15} = \sqrt{3 \times 15} = \sqrt{3 \times (3 \times 5)} = \sqrt{3^2 \times 5} = 3\sqrt{5}$
[ทริค/กฎเหล็ก]: ทริคคิดลัด: ไม่จำเป็นต้องเอา $3 \times 15 = 45$ แล้วค่อยมาตั้งหารสั้นนะครับ ให้แตกตัวประกอบย่อยของ 15 ออกเป็น $3 \times 5$ ทันที จะเห็นคู่เลข 3 สองตัวจับคู่หลุดออกจากรูททันใจ!

1.2.2 การหาผลบวก ผลต่างและผลคูณของจำนวนที่อยู่ในรูปกรณฑ์

เราสามารถหาผลบวก ผลต่างและผลคูณของจำนวนที่มีเครื่องหมายกรณฑ์อันดับเดียวกันและมีจำนวนภายในเครื่องหมายกรณฑ์เป็นจำนวนเดียวกันดังนี้

$a\sqrt{b} + c\sqrt{b} - d\sqrt{b} = (a + c - d)\sqrt{b}$
$(a\sqrt{b})(c\sqrt{d}) = ac\sqrt{bd}$
$\frac{a}{\sqrt{b} + \sqrt{c}} = \frac{a(\sqrt{b} - \sqrt{c})}{(\sqrt{b} + \sqrt{c})(\sqrt{b} - \sqrt{c})} = \frac{a(\sqrt{b} - \sqrt{c})}{b - c}$
