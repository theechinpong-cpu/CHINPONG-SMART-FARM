📐 พระจอมเกล้าติวเตอร์ เลขยกกำลัง ม.5 (Exponents)
💡 คลังข้อสอบอัตนัย + ไวท์บอร์ด

สรุปเนื้อหา: เลขยกกำลัง & รากที่ n

กฎเกณฑ์สำคัญ + ทริคลัด เตรียมสอบ A-Level

📌 รากที่ n ในระบบจำนวนจริง

b เป็นรากที่ n ของ a ↔ bⁿ = a (n > 1)
• n คู่: a > 0 มีรากจริง 2 ค่า (+ⁿ√a, -ⁿ√a) , a < 0 ไม่มีรากจริง
• n คี่: มีรากจริง 1 ค่า (เครื่องหมายเดียวกับ a)

⚠️ ค่าหลัก ⁿ√a : ถ้า n คู่ → ผลลัพธ์ ≥ 0 เสมอ

✏️ เลขชี้กำลังเป็นจำนวนตรรกยะ

a^(p/q) = (ⁿ√a)ᵖ = ⁿ√(aᵖ) , โดยที่ ⁿ√a เป็นจำนวนจริง

ตัวอย่าง 8^(2/3) = (³√8)² = 2² = 4

📚 สมบัติเด็ด

  • aʳ·aˢ = aʳ⁺ˢ
  • aʳ÷aˢ = aʳ⁻ˢ
  • (aʳ)ˢ = aʳˢ
  • (ab)ʳ = aʳ·bʳ
  • ⁿ√xⁿ = |x| (n คู่), ⁿ√xⁿ = x (n คี่)

🧠 กระบวนการคิด 3 ขั้น (Step การมองโจทย์)

Step 1: แยกฐานให้เล็กที่สุด (Prime Factorization)
Step 2: จัดระเบียบเลขชี้กำลังลบ (a⁻ⁿ = 1/aⁿ)
Step 3: ระวังเงื่อนไขค่าสัมบูรณ์เมื่อถอดรากคู่

📌 คลังข้อสอบอัตนัย + แบบทดสอบโต้ตอบ (พระจอมเกล้าติวเตอร์)

✅ ตัวอย่าง: จงหารากที่ n (1-8) และ จงเขียนในรูปอย่างง่าย 8 ข้อ (ครบตามต้นฉบับ)
(1) √3 · √5 → วิธีทำ: ใช้สมบัติ √a·√b = √(ab) = √15
(2) ³√10 · ³√100 → ³√1000 = 10
(3) ³√-81 / ³√3 → ³√(-27) = -3
(4) √(4x⁵)/√(x³) , x>0 → 2x
(5) ³√3² · ⁶√3 → 3^(5/6)
(6) [³√x²·√x]/⁶√x , x>0 → x
(7) √(a-b)·³√(a-b)² → (a-b)^(7/6)
(8) √x³y·³√xy² / ⁶√x⁵y⁴ → x√y

📢 แบบทดสอบระบบอัจฉริยะ (Interactive Quiz) – ข้อสอบเลขยกกำลัง ตั้งแต่ราก, เลขชี้กำลังตรรกยะ, การจัดรูป และประยุกต์ (ครบทุกหัวข้อ) อยู่ในแท็บ “แบบฝึกหัด & แบบทดสอบ” ด้านบน! คลิกเปลี่ยนแท็บเพื่อทำข้อสอบแบบเต็ม พร้อมวิเคราะห์จุดอ่อนทันที ✅

ข้อสอบวัดผล เลขยกกำลัง ม.5 (ครอบคลุมรากที่ n, เลขชี้กำลังเศษส่วน, สังยุค, สมการ, โจทย์ปัญหาประยุกต์)

🔹 แบบทดสอบย่อย : รากที่ n และค่าหลัก (ข้อ 1-4)

❶ ³√−64 + √144 − ⁴√81 = ?
ตอบ 5 (³√−64 = -4, √144=12, ⁴√81=3 → -4+12-3=5)
❷ 5/(√6−1) ทำส่วนไม่ติดกรณฑ์ → √6+1
❸ (25)^(3/2)×(8)^(−2/3) = 125/4
❹ ดอกเบี้ยทบต้น 2% 3 ปี เงินต้น 10,000 → 10,000(1.02)³

📊 แบบทดสอบเลขยกกำลังตรรกยะ

1) 8^(1/3) = 2   2) 27^(2/3)=9   3) (0.25)^(3/2)=0.125
4) 16^(-1/4)=1/2   5) (1/32)^(-1/5)=2   6) (-32)^(3/5) = -8
7) (-64)^(2/3)=16   8) (-16)^(3/4) = ไม่มีในจำนวนจริง
9) (4/9)^(3/2)=8/27   10) (0.001)^(-2/3)=100

🧮 จัดรูปพจน์ตัวแปร (Exponent laws)

✔ (x²y³)(x⁻³y⁴) = y⁷/x
✔ (x⁻¹y²)⁻² = x² / y⁴
✔ (x^(1/2)y²)(x^(5/2)y⁻¹)= x³y
✔ (x^(-1/3)y^(2/3))³ = y²/x

📈 สมการและอสมการเอกซ์โพเนนเชียล

🔹 x^(1/2)=3 → {9}   🔹 x^(-2)=1/9 → {-3,3}
🔹 3^(2x+1)-10·3^x+3=0 → {-1,1}
🔹 (1/2)^(x²-x) > (1/4)^(x+1) → ช่วง (-1,2)
