📗 สรุปเนื้อหาเน้นสอบ (1,000 คำ)

1. กาลเวลา (Tenses):
- Present Simple: $S + V.1$ ใช้กับความจริงทั่วไปหรือนิสัย (Every day, always). **หัวใจสำคัญ:** ประธาน He, She, It, Peter กริยาต้องเติม s/es.
- Present Continuous: $S + is/am/are + V.ing$ ใช้กับสิ่งที่กำลังเกิดตอนนี้ (Now, Look!, Listen!).
- Past Simple: $S + V.2$ ใช้กับอดีต (Yesterday, Last week, 2 days ago). กริยามักเติม -ed หรือเปลี่ยนรูป เช่น go -> went, eat -> ate.

2. การเปรียบเทียบ (Comparison):
- ขั้นกว่า: $Adj-er + than$ หรือ $More + Adj + than$
- ขั้นสุด: $The + Adj-est$ หรือ $The + Most + Adj$

3. คำบอกปริมาณ:
- Some: บอกเล่า / Any: คำถามและปฏิเสธ
- How much: นามนับไม่ได้ (น้ำ, ข้าว, เงิน) หรือราคา / How many: นามนับได้พหูพจน์

4. บุพบท (Prepositions):
- In: เดือน, ปี, ฤดู, ประเทศ / On: วัน, วันที่, ถนน / At: เวลาเฉพาะเจาะจง, บ้านเลขที่

📘 สรุปเข้มเนื้อหาภาษาอังกฤษ ป.5 (ฉบับเตรียมสอบ 100 ข้อ)

1. กาลเวลา (Tenses)

  • Present Simple: $S + V.1$ (จริง/นิสัย)
    *ประธานคนเดียว (He/She/It) กริยาเติม s/es*
  • Present Continuous: $S + is/am/are + V.ing$
    *กำลังทำอยู่ (Now, Look!, Listen!)*
  • Past Simple: $S + V.2$ (อดีต)
    *Yesterday, Last night, Ago*

2. การเปรียบเทียบ (Comparison)

  • ขั้นกว่า (Comparative): $Adj+er + than$ หรือ $More + Adj + than$
  • ขั้นสุด (Superlative): $The + Adj+est$ หรือ $The + Most + Adj$
  • *จดจำ:* good ➔ better ➔ best / bad ➔ worse ➔ worst

3. การบอกจำนวน (Quantifiers)

  • Some: ประโยคบอกเล่า (I have some milk.)
  • Any: ประโยคปฏิเสธ/คำถาม (Do you have any milk?)
  • How many: ถามจำนวน (นับได้ เช่น apples, cars)
  • How much: ถามปริมาณ (นับไม่ได้ เช่น water) / ถามราคา

4. คำบุพบท (Prepositions)

  • In: เดือน, ปี, ฤดู, ช่วงเวลาเช้า/บ่าย (In May, In 2024)
  • On: วัน (Monday), วันที่ (Oct 31st), บนพื้นผิว
  • At: เวลา (At 7 o'clock), สถานที่เฉพาะ (At school)
