ข้อสอบจำลองกลางภาคเรียนที่ 1 วิชาคณิตศาสตร์ ม.6

จัดทำโดย พระจอมเกล้าติวเตอร์ - คลังข้อสอบคัดสรรตามหลักสูตรแกนกลางเพื่อเตรียมความพร้อมสู่ห้องสอบ

ข้อ 1. พจน์ที่ 20 ของลำดับเลขคณิต 3, 7, 11, 15, ... มีค่าเท่ากับข้อใด? (Clue: ใช้สูตรพจน์ทั่วไปของลำดับเลขคณิต a_n = a_1 + (n - 1)d)
  • A. 76
  • B. 79
  • C. 83
  • D. 80
คำตอบที่ถูกต้อง: B
คำอธิบายอย่างละเอียด:
จากลำดับ 3, 7, 11, 15, ... จะได้ พจน์แรก a_1 = 3 และผลต่างร่วม d = 7 - 3 = 4
ต้องการหาพจน์ที่ 20 (n = 20):
a_20 = a_1 + (20 - 1)d
a_20 = 3 + 19(4) = 3 + 76 = 79
Tutor's Secret Tip: ทริคคิดเลขเร็ว ให้มองหาค่าเพิ่มขึ้นต่อสเต็ป ในที่นี้คือแม่ 4 ดังนั้นพจน์ทั่วไปคือ 4n - 1 อยากได้พจน์ที่ 20 ก็เอา 4(20) - 1 = 79 ได้คำตอบไวสุดๆ ครับ!
ข้อ 2. พจน์ทั่วไป (a_n) ของลำดับเรขาคณิต 2, 6, 18, 54, ... ตรงกับข้อใด? (Clue: หาอัตราส่วนร่วม r = a_2 / a_1 แล้วแทนในสูตร a_n = a_1 * r^(n-1))
  • A. a_n = 2 * 3^n
  • B. a_n = 2 * 3^(n-1)
  • C. a_n = 3 * 2^(n-1)
  • D. a_n = 6^(n-1)
คำตอบที่ถูกต้อง: B
คำอธิบายอย่างละเอียด:
พจน์แรก a_1 = 2
หาอัตราส่วนร่วม r = 6 / 2 = 3
จากสูตรพจน์ทั่วไปของลำดับเรขาคณิต: a_n = a_1 * r^(n-1)
แทนค่าได้: a_n = 2 * 3^(n-1)
Tutor's Secret Tip: ถ้าจำสูตรไม่ได้ ให้ใช้วิธีลองแทนค่า n = 1 ลงในช้อยส์ ช้อยส์ไหนได้คำตอบเท่ากับ 2 (พจน์แรก) และ n = 2 ได้ 6 ช้อยส์นั้นแหละครับคือคำตอบ!
ข้อ 3. ผลบวก 15 พจน์แรกของลำดับเลขคณิต 5, 9, 13, 17, ... เท่ากับข้อใด? (Clue: ใช้สูตรอนุกรมเลขคณิต S_n = (n/2) * [2a_1 + (n - 1)d])
  • A. 495
  • B. 510
  • C. 525
  • D. 600
คำตอบที่ถูกต้อง: A
คำอธิบายอย่างละเอียด:
จากโจทย์ a_1 = 5, d = 4, n = 15
ใช้สูตร: S_15 = (15/2) * [2(5) + (15 - 1)4]
S_15 = (15/2) * [10 + 14(4)]
S_15 = (15/2) * [10 + 56] = (15/2) * 66
S_15 = 15 * 33 = 495
Tutor's Secret Tip: สังเกตตัวเลขให้ดีๆ ถ้านำ 2 ไปตัดกับ 66 ในวงเล็บก่อนจะเหลือเลข 33 ซึ่งทำให้เราคูณกับ 15 ได้ง่ายและแม่นยำขึ้นมาก ลดโอกาสคิดเลขผิดในห้องสอบครับ
ข้อ 4. อนุกรมเรขาคณิตอนันต์ 4 + 2 + 1 + 0.5 + ... มีผลบวกตรงกับข้อใด? (Clue: เป็นอนุกรมอนันต์ที่ลู่เข้า เนื่องจาก |r| < 1 ให้ใช้สูตร S_∞ = a_1 / (1 - r))
  • A. 7
  • B. 8
  • C. 12
  • D. หาผลบวกไม่ได้เพราะเป็นอนุกรมอนันต์
คำตอบที่ถูกต้อง: B
คำอธิบายอย่างละเอียด:
จากอนุกรม: a_1 = 4
หาอัตราส่วนร่วม r = 2 / 4 = 1/2 (ซึ่ง |1/2| < 1 อนุกรมลู่เข้า)
ใช้สูตรอนุกรมอนันต์: S_∞ = a_1 / (1 - r)
S_∞ = 4 / (1 - 1/2) = 4 / (1/2) = 4 * 2 = 8
Tutor's Secret Tip: จำกฎเหล็กไว้เลย! สูตร S_∞ = a_1 / (1-r) จะใช้ได้ก็ต่อเมื่อค่า r ต้องอยู่ระหว่าง -1 กับ 1 เท่านั้น ถ้า r มากกว่าหรือเท่ากับ 1 จะตอบทันทีว่า "หาค่าไม่ได้ (Divergent)" ครับ
ข้อ 5. ค่าของลิมิต lim (n→∞) ของลำดับ a_n = (3n^2 - 5n) / (2n^2 + 7) เท่ากับข้อใด? (Clue: พิจารณาดีกรีสูงสุดของพจน์เศษและส่วน ถดีกรีเท่ากัน ลิมิตคือสัญญลักษณ์สัมประสิทธิ์หน้าดีกรีนั้น)
  • A. 0
  • B. 3/2
  • C. -5/7
  • D. หาค่าไม่ได้ (∞)
คำตอบที่ถูกต้อง: B
คำอธิบายอย่างละเอียด:
พิจารณาลำดับเศษส่วนพหุนามเมื่อ n เข้าใกล้ ∞:
ดีกรีสูงสุดของเศษคือ n^2 (สัมประสิทธิ์คือ 3)
ดีกรีสูงสุดของส่วนคือ n^2 (สัมประสิทธิ์คือ 2)
เนื่องจากกำลังสูงสุดเท่ากัน (ดีกรี 2 เท่ากัน) สามารถตัดพจน์ย่อยทิ้งแล้วนำสัมประสิทธิ์มาตอบได้ทันที
ดังนั้น ลิมิตเท่ากับ 3/2
Tutor's Secret Tip: สูตรลัดลิมิตอนันต์: "กำลังบน = กำลังล่าง ตอบ สปส. ตัวบนหารตัวล่าง", "กำลังบน < กำลังล่าง ตอบ 0", "กำลังบน > กำลังล่าง ตอบ หาค่าไม่ได้" จำไปใช้สอบเสร็จไวแน่นอน!
ข้อ 6. ลำดับในข้อใดต่อไปนี้เป็น "ลำดับลู่เข้า" (Convergent Sequence)? (Clue: ลำดับลู่เข้าคือลำดับที่เมื่อหาลิมิตเมื่อ n→∞ แล้วได้ผลลัพธ์เป็นจำนวนจริงคงที่)
  • A. a_n = 2^n
  • B. a_n = (-1)^n
  • C. a_n = (n + 1) / n
  • D. a_n = n^2 - 3
คำตอบที่ถูกต้อง: C
คำอธิบายอย่างละเอียด:
- ข้อ A: lim 2^n = ∞ (ลู่ออก)
- ข้อ B: lim (-1)^n แกว่งกวัดระหว่าง -1, 1 ไม่มีค่าแน่นอน (ลู่ออก)
- ข้อ C: lim (n + 1) / n = 1 พุ่งเข้าหาเลข 1 แน่นอน (ลู่เข้า)
- ข้อ D: lim (n^2 - 3) = ∞ (ลู่ออก)
Tutor's Secret Tip: ลำดับแบบข้อ B มีชื่อเฉพาะเรียกว่า ลำดับแกว่งกวัด (Oscillating sequence) ถือเป็นกลุ่มหนึ่งของลำดับลู่ออกเพราะเราไม่สามารถฟันธงปลายทางของมันได้ครับ
ข้อ 7. ผลบวกของอนุกรม ซิกมา ∑ (จาก i=1 ถึง 10) ของ (2i + 3) มีค่าเท่ากับข้อใด? (Clue: กระจายซิกมาเข้าไปตามสมบัติ ∑(2i) + ∑3 แล้วใช้สูตร ∑i = n(n+1)/2 และ ∑c = c*n)
  • A. 110
  • B. 140
  • C. 143
  • D. 150
คำตอบที่ถูกต้อง: B
คำอธิบายอย่างละเอียด:
∑(2i + 3) = 2∑i + ∑3
= 2 * [10(10 + 1) / 2] + (3 * 10)
= 2 * [110 / 2] + 30
= 110 + 30 = 140
Tutor's Secret Tip: ระวังจุดพลาดบ่อย! ∑3 จาก 1 ถึง 10 ไม่ใช่ได้ 3 นะครับ แต่มันคือเลข 3 บวกกัน 10 ตัว จึงต้องจับเอา 3 คูณกับ 10 ได้เป็น 30 ครับผม
ข้อ 8. ค่าของ lim (x→3) ของ (x^2 - 9) / (x - 3) เท่ากับข้อใด? (Clue: แทนค่าตรงๆ จะได้รูป 0/0 ให้ใช้วิธีแยกตัวประกอบส่วนต่างกำลังสองเพื่อตัดทอนพจน์เจ้าปัญหา)
  • A. 0
  • B. 3
  • C. 6
  • D. หาค่าไม่ได้
คำตอบที่ถูกต้อง: C
คำอธิบายอย่างละเอียด:
แทนค่า x = 3 ทันที จะได้ (3^2 - 9)/(3 - 3) = 0/0 ซึ่งสรุปไม่ได้
จัดรูปใหม่โดยแยกตัวประกอบเศษ: x^2 - 9 = (x - 3)(x + 3)
จะได้ lim (x→3) [(x - 3)(x + 3) / (x - 3)]
ตัดพจน์ (x - 3) ออก จะเหลือแค่ lim (x→3) (x + 3)
แทนค่า x = 3 จะได้ 3 + 3 = 6
Tutor's Secret Tip: ถ้าน้องเรียนดิฟฟ์มาแล้ว สามารถใช้ "กฎของโลปีตาล (L'Hôpital's Rule)" โดยการดิฟเศษและดิฟส่วนแยกกันได้เลย ดิฟเศษได้ 2x ดิฟส่วนได้ 1 แทนค่า x=3 ตอบ 6 เท่ากัน เร็วกว่าเดิมเยอะ!
ข้อ 9. กำหนดให้ f(x) = 2x^2 - 5x + 3 อัตราการเปลี่ยนแปลงเฉลี่ยของ f(x) เทียบกับ x เมื่อ x เปลี่ยนจาก 1 ไปเป็น 3 เท่ากับข้อใด? (Clue: อัตราการเปลี่ยนแปลงเฉลี่ยใช้สูตร [f(x_2) - f(x_1)] / (x_2 - x_1))
  • A. 3
  • B. 4
  • C. 5
  • D. 6
คำตอบที่ถูกต้อง: A
คำอธิบายอย่างละเอียด:
สูตร: อัตราการเปลี่ยนแปลงเฉลี่ย = [f(3) - f(1)] / (3 - 1)
- หา f(3) = 2(3)^2 - 5(3) + 3 = 18 - 15 + 3 = 6
- หา f(1) = 2(1)^2 - 5(1) + 3 = 2 - 5 + 3 = 0
แทนค่ากลับในสูตร: [6 - 0] / [3 - 1] = 6 / 2 = 3
Tutor's Secret Tip: คำว่า "อัตราการเปลี่ยนแปลงเฉลี่ย" ให้คิดเสมือนว่าเรากำลังหาความชัน (m) ของเส้นตรงที่ลากผ่านจุดสองจุดนั่นเองครับ อย่าสับสนกับอัตราการเปลี่ยนแปลง ณ ขณะใดขณะหนึ่ง (อันนั้นต้องดิฟนะ)
ข้อ 10. ฟังก์ชัน f(x) จะมีความต่อเนื่อง (Continuity) ที่จุด x = a ก็ต่อเมื่อเงื่อนไขใดเป็นจริง? (Clue: ความต่อเนื่องต้องใช้ 3 เงื่อนไข: มีค่าฟังก์ชัน, มีลิมิตซ้ายขวาที่เท่ากัน และทั้งสองข้อต้องมีค่าเท่ากัน)
  • A. f(a) หาค่าได้ และ lim(x→a) f(x) หาค่าได้
  • B. lim(x→a⁻) f(x) = lim(x→a⁺) f(x)
  • C. f(a) หาค่าได้ และ lim(x→a) f(x) = f(a)
  • D. f'(a) หาค่าได้เท่านั้น
คำตอบที่ถูกต้อง: C
คำอธิบายอย่างละเอียด:
นิยามความต่อเนื่องที่จุด x = a ประกอบด้วยเงื่อนไขสมบูรณ์คือ:
1. f(a) หาค่าได้เป็นจำนวนจริง
2. lim(x→a) f(x) ต้องหาค่าได้ (ลิมิตซ้ายเท่ากับลิมิตขวา)
3. ค่าที่ได้จากข้อ 1 และข้อ 2 ต้องเท่ากันเป๊ะ นั่นคือ lim(x→a) f(x) = f(a)
Tutor's Secret Tip: สรุปสั้นๆ ให้จำง่ายๆ ไปห้องสอบคือกราฟไม่ขาดตอน จุดทึบ (f(a)) และเส้นวิ่งเข้าหาจากสองข้าง (Limit) ต้องมาสบและถมเต็มที่จุดเดียวกันพอดีครับ
ข้อ 11. อนุพันธ์ของฟังก์ชัน f(x) = 4x^3 - 5x^2 + 7x - 9 คือข้อใด? (Clue: ใช้สูตรพื้นฐานของการดิฟพหุนามตบกำลังลงมาคูณแล้วกำลังลดลงไป 1)
  • A. f'(x) = 12x^2 - 10x + 7
  • B. f'(x) = 12x^3 - 10x^2 + 7
  • C. f'(x) = 4x^2 - 5x + 7
  • D. f'(x) = 12x^2 - 10x
คำตอบที่ถูกต้อง: A
คำอธิบายอย่างละเอียด:
ทำการหาอนุพันธ์ทีละพจน์ (Differentiate):
- d/dx(4x^3) = 4 * 3x^2 = 12x^2
- d/dx(-5x^2) = -5 * 2x^1 = -10x
- d/dx(7x) = 7
- d/dx(-9) = 0 (ดิฟค่าคงที่ได้ 0)
รวมคำตอบได้: f'(x) = 12x^2 - 10x + 7
Tutor's Secret Tip: จำกฎการดิฟฟ์พหุนามเบื้องต้น: "ตบกำลังลงมาคูณข้างหน้า แล้วลดกำลังของเดิมลงไปหนึ่ง" ส่วนตัวเลขเดี่ยวๆ ที่ไม่มี x ติดอยู่ ดิฟแล้วหายวับกลายเป็น 0 เสมอครับ!
ข้อ 12. ความชันของเส้นสัมผัสเส้นโค้ง y = x^2 - 3x ที่จุด (2, -2) มีค่าเท่ากับเท่าใด? (Clue: ความชันของเส้นสัมผัสเส้นโค้งที่จุดใดๆ คือค่าของอนุพันธ์ f'(x) ณ จุด x นั้นๆ)
  • A. -2
  • B. 1
  • C. 4
  • D. 2
คำตอบที่ถูกต้อง: B
คำอธิบายอย่างละเอียด:
จากสมการเส้นโค้ง f(x) = x^2 - 3x
หาอนุพันธ์เพื่อสร้างสมการความชัน: f'(x) = 2x - 3
โจทย์ระบุจุดสัมผัสคือ (2, -2) นำพิกัด x = 2 ไปแทนค่าใน f'(x):
ความชัน (m) = f'(2) = 2(2) - 3 = 4 - 3 = 1
Tutor's Secret Tip: โจทย์เรขาคณิตแคลคูลัสชอบให้พิกัดมาเป็นคู่อันดับ (x, y) หลอกตา จำไว้แม่นๆ ว่าตอนนำไปแทนค่าในสมการความชัน f'(x) เราใช้แค่ตัวแปร x เท่านั้นครับ ตัว y ไม่ต้องเอามาเกี่ยว
ข้อ 13. ถ้า f(x) = (2x - 1)(x + 4) แล้วค่าของ f'(x) เท่ากับข้อใด? (Clue: สามารถใช้สูตรดิฟผลคูณ (หน้าดิฟหลัง + หลังดิฟหน้า) หรือจะคูณกระจายวงเล็บก่อนแล้วค่อยดิฟปกติก็ได้)
  • A. 4x + 7
  • B. 2x + 7
  • C. 4x - 7
  • D. 2
คำตอบที่ถูกต้อง: A
คำอธิบายอย่างละเอียด:
วิธีที่ง่ายที่สุดคูณกระจายฟังก์ชันก่อนดิฟ:
f(x) = (2x - 1)(x + 4) = 2x^2 + 8x - x - 4 = 2x^2 + 7x - 4
หาอนุพันธ์ f'(x) = d/dx(2x^2 + 7x - 4) = 4x + 7
Tutor's Secret Tip: ถ้าเป็นพหุนามดีกรีต่ำๆ แนะนำให้คูณกระจายให้อยู่ในรูปการบวกลบธรรมดาก่อนค่อยดิฟ จะไวกว่าและลดโอกาสพลาดสูตรดิฟผลคูณ "หน้าดิฟหลัง บวก หลังดิฟหน้า" ได้ดีมากเลยครับ
ข้อ 14. กำหนด f(x) = (3x^2 + 1)^5 จงหาค่าของ f'(x) โดยใช้กฎลูกโซ่ (Chain Rule) (Clue: มองภาพรวมเป็นดิฟไส้นอกก่อน ยกลูกกำลังลงมา แล้วคูณด้วยดิฟฟังก์ชันข้างในวงเล็บซ้ำอีกรอบ)
  • A. 5(3x^2 + 1)^4
  • B. 30x(3x^2 + 1)^4
  • C. 6x(3x^2 + 1)^4
  • D. 15x^2(3x^2 + 1)^4
คำตอบที่ถูกต้อง: B
คำอธิบายอย่างละเอียด:
จากกฎลูกโซ่: ดิฟนอกคูณดิฟใน
f'(x) = 5(3x^2 + 1)^(5-1) * d/dx(3x^2 + 1)
f'(x) = 5(3x^2 + 1)^4 * (6x)

ข้อ 19. ผลบวกของอนุกรม 1/2 + 1/4 + 1/8 + 1/16 + ... + 1/1024 มีค่าเท่ากับข้อใด? (Clue: เป็นอนุกรมเรขาคณิตจำกัด ใช้สูตร S_n = a_1(1 - r^n) / (1 - r) โดยต้องนับจำนวนพจน์ n ออกมาให้ถูกก่อน)
  • A. 1023 / 1024
  • B. 511 / 512
  • C. 1
  • D. 2047 / 2048
คำตอบที่ถูกต้อง: A
คำอธิบายอย่างละเอียด:
อนุกรมชุดนี้มี a_1 = 1/2 , r = 1/2
พจน์สุดท้ายคือ 1/1024 ซึ่งคือ (1/2)^10 แสดงว่าจำนวนพจน์ n = 10
ใช้สูตร: S_10 = (1/2) * [1 - (1/2)^10] / [1 - 1/2]
เนื่องจากตัวส่วนคือ (1 - 1/2) = 1/2 จะตัดกับ 1/2 ตัวข้างหน้าเศษหมดพอดี

ข้อ 20. จงหาค่าของลิมิต lim (n→∞) ของลำดับ a_n = [5^n + 3^n] / [5^(n+1) - 2^n] (Clue: ลำดับเรขาคณิตดัชนีอนันต์ ให้เลือกนำฐานที่มีค่ามากที่สุดหารทั้งเศษและส่วน หรือพิจารณาเฉพาะฐานที่ใหญ่ที่สุด)

ข้อ 21. กำหนดให้ f(x) = x^3 - 3x^2 - 9x + 5 จุดวิกฤต (Critical Point) ในส่วนค่าของ x มีค่าเท่ากับข้อใดต่อไปนี้? (Clue: จุดวิกฤตหาได้จากการหาอนุพันธ์อันดับที่หนึ่งแล้วจับเท่ากับศูนย์ f'(x) = 0)
  • A. x = 1, -3
  • B. x = -1, 3
  • C. x = 0, 3
  • D. x = -1, -3
คำตอบที่ถูกต้อง: B
คำอธิบายอย่างละเอียด:
หา f'(x) = 3x^2 - 6x - 9
ตั้งสมการหาจุดวิกฤตโดยกำหนดให้ f'(x) = 0
3x^2 - 6x - 9 = 0
นำ 3 หารตลอดสมการ: x^2 - 2x - 3 = 0
แยกตัวประกอบพหุนาม: (x - 3)(x + 1) = 0
ดังนั้น ค่าวิกฤต x = 3 หรือ x = -1
Tutor's Secret Tip: จุดวิกฤตคือตำแหน่งบนโค้งที่กราฟเริ่มวกกลับ (ความชันราบเป็นศูนย์) ซึ่งจุดเหล่านี้จะเป็นตัวเลือกหลักที่นำไปสู่การเป็นจุดสูงสุดหรือต่ำสุดสัมพัทธ์ของฟังก์ชันต่อไปครับ
ข้อ 22. จากฟังก์ชันในข้อ 21 (f(x) = x^3 - 3x^2 - 9x + 5) จุดวิกฤตใดที่ให้ "ค่าสูงสุดสัมพัทธ์" (Relative Maximum)? (Clue: ทดสอบด้วยอนุพันธ์อันดับสอง f''(x) ถ้าค่าออกมาน้อยกว่า 0 (< 0) จุดนั้นจะเป็นจุดสูงสุดสัมพัทธ์)
  • A. x = 3
  • B. x = -1
  • C. x = 5
  • D. ให้ค่าสูงสุดเท่ากันทั้งสองจุด
คำตอบที่ถูกต้อง: B
คำอธิบายอย่างละเอียด:
จาก f'(x) = 3x^2 - 6x - 9 จะได้อนุพันธ์อันดับสองคือ f''(x) = 6x - 6
นำค่าวิกฤตจากข้อก่อนหน้า (-1 และ 3) มาตรวจสอบ:
- แทน x = 3: f''(3) = 6(3) - 6 = 12 ซึ่ง > 0 (ให้จุดต่ำสุดสัมพัทธ์)
- แทน x = -1: f''(1) = 6(-1) - 6 = -12 ซึ่ง < 0 (ให้จุดสูงสุดสัมพัทธ์)
ดังนั้น x = -1 คือค่าวิกฤตที่ให้ค่าสูงสุดสัมพัทธ์
Tutor's Secret Tip: ทริคจำสวนทาง: "ดิฟสองครั้งแล้วได้บวก คือหงาย(ต่ำสุด), ดิฟสองครั้งแล้วได้ลบ คือคว่ำ(สูงสุด)" ท่องไว้ให้ดี เครื่องหมายจะสลับความรู้สึกกับชื่อจุดครับ!
ข้อ 23. ถ้ากำหนดฟังก์ชันอนุพันธ์ f'(x) = 2x + 3 และบอกเงื่อนไขเพิ่มว่า f(1) = 5 แล้วตัวฟังก์ชันดั้งเดิม f(x) คือข้อใด? (Clue: อินทิเกรตคืนกลับ f'(x) เพื่อหาพจน์ติดค่า C แล้วใช้ f(1) = 5 ในการแก้หาค่าคงที่ C ตัวนั้นออกมา)
  • A. f(x) = x^2 + 3x
  • B. f(x) = x^2 + 3x + 1
  • C. f(x) = x^2 + 3x + 5
  • D. f(x) = 2x^2 + 3x
คำตอบที่ถูกต้อง: B
คำอธิบายอย่างละเอียด:
อินทิเกรต f'(x) เพื่อหา f(x): f(x) = ∫(2x + 3) dx = x^2 + 3x + C
